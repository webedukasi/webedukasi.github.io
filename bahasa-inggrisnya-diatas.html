13. apa bahasa inggris di atas


SIMPLE PRESENT

rumus :

(+) Subject + V1 (s/es) + Object

(-) Subject + Do/Does + Not + V1 + object

(?) Do/Does + Subject + V1 + Object


catatan :

Verb 1 dengan tambahan s/es hanya digunakan untuk subject (she/he/it)


Translate into english

1. Riko selalu sarapan setiap pagi.

=> Riko always has breakfast every morning.

2. Kami biasanya pulang sekolah  jam 12.00

=> We usually go home at twelve o'clock

3. Saya jarang bermain kelereng disore hari.

=> I rarely play marble in the afternoon

4. Budi kadang-kadang bangun jam 4.30

=> Budi sometimes wakes up at a half past four.

5. Rudi biasanya tidur siang pada jam 01.30

=> Rudi usually takes a nap at a half past one

Jawaban:

1.) Dont enter the classroom !

Dont = Jangan

Enter = Masuk / Memasuki

Classroom = Ruang kelas / kelas

Dont Enter The Classroom ! = Jangan masuk kelas !

2.) Dont Fight With Your Friends !

Dont = Jangan

Fight = Bertengkar / Bertarung

With = Dengan

Your = Milikmu

Friends = Teman

Dont Fight With Your Friends ! = Jangan Bertengkar dengan temanmu !

3.) Dont Play Game Too Much !

Dont = Jangan

Play = Bermain

Game = Permainan

Too = Juga

Much = Banyak

Dont Play Game Too Much ! = Jangan Terlalu banyak bermain Permainan !

4.) Dont Watch TV too Close !

Dont = Jangan

Watch = menonton

TV = Televisi

Too = Juga

Close = Menutup

Dont Watch TV too Close ! = Jangan Menonton TV terlalu Dekat !

5.) Dont Talk Loudly, This is a Library !

Dont = Jangan

Talk = Berbicara

Loudly = Dengan keras

This = Ini

Is = Adalah

Library = Perpustakaan

Dont Talk Loudly, This is a Library ! = Jangan bicara keras keras, Ini adalah  - perpustakaan !

6.) Dont Disturb Your Friend !

Dont = Jangan

Disturb = Mengganggu

Your = Milikmu

Friend = Teman

Dont Disturb Your Friend ! = Jangan Mengganggu temanmu !

7.) Dont Litter, Put The Trash On The Trash Can !

Dont = Jangan

Litter = Sampah

Put = Taruh

Trash = Sampah

On = Pada

Can = Bisa

Dont Litter, Put the Trash On The Trash Can ! = Jangan Buang Sampah, Letakkan Sampah Di Tempat Sampah !

8.) Dont Run In The Classroom !

Dont = Jangan

Run = Lari / Berlari

In = Di

Classroom = Ruangan Kelas / Kelas

Dont Run In The Classroom ! = Jangan Lari di kelas !

9.) Dont Laugh !

Dont = Jangan

Laugh = Tertawa

Dont Laugh ! = Jangan Tertawa !

10.) Dont Go Anywhere, stay in Your Room !

Dont = Jangan

Go = Pergi

Anywhere = Di mana pun / Di mana Saja

Stay = Tetap

In = Di

Your = Milikmu

Room = Ruangan

Dont Go Anywhere, Stay In Your Room ! = Jangan Ke mana mana, tetap di kamarmu !

Vocabulary :

Don't = Jangan

Run = Lari

Your = Milikmu

In = Di
