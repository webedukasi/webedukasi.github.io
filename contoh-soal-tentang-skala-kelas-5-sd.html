25. contoh gerakan lokomotor antara lain adalah ..soal kelas 5 sd ​


Contoh gerakan lokomotor antara lain adalah :

BerjalanBerlariMelompatMeloncatMemanjat

Pembahasan

         Gerakan lokomotor adalah gerakan yang dilakukan oleh tubuh sehingga membuat tubuh berpindah tempat. Seperti yang terlah disebutkan diatas, gerakan-gerakan diatas adalah gerakan yang membuat tubuh bergerak berpindah tempat. Gerak lokomotor termasuk ke dalam salah satu gerakan tubuh manusia. Gerakan dasar tubuh manusia dibagi menjadi 3 macam, yaitu gerakan lokomotor, gerakan nonlokomotor, dan gerakan manipulatif.

         Berikut pembahasannya untuk masing-masing gerakan :

Gerakan Lokomotor

Gerakan nonlokomotor ialah, aktivitas yang dilakukan tubuh sehingga tubuh bergerak dan berpindah tempat. Contoh gerakan lokomotor antara lain :

BerjalanBerlariMelompatMeloncatMemanjat

     2. Gerakan Nonlokomotor

Gerakan nonlokomotor adalah gerakan yang dilakukan tubuh sehingga tubuh bergerak namun tidak berpindah tempat/tetap di tempat. Contoh gerakan nonlokomotor antara di antaranya :

JongkokMembungkukMenolehMengayun

     3. Gerakan Manipulatif

Gerakan manipulatif yaitu, gerakan yang dilakukan oleh manusia dengan melibatkan/menggunakan alat tertentu. Contohnya :

MelemparMenendangMenerima/Menangkap

Penjelasan dengan langkah-langkah:

1. Maka :

Skala = 1: 2.500.000

Jarak pada peta = 12 cm

Diminta jarak yang sebenarnya?

Jb = Jp: S

Jb = 12: 1 / 250.000

Jb = 12 x 250.000

Jb = 3.000.000 cm = 30 km

Dengan demikian jarak sebenarnya adalah 30 km

Jawabannya: B

2. Maka :

Jarak pada peta = 10 cm

Skala = 1: 2.500.000

Diminta jarak yang sebenarnya?

Jb = Jp: S

Jb = 10: 1 / 2.500.000

Jb = 10 × 2.500.000

Jarak aktual = 25.000.000 cm = 250 km

Jawabanya: C

3. Maka :

Skala = 1: 100

Panjang sisi = 28 cm

Panjang aktual = 28 x 100 = 2.800 cm = 28 m

K = 4 × dtk

K = 4 · 28

K = 112 m

Jadi sekitar tambak adalah 112 m

Jawabannya: B

4. Maka :

Jarak pada peta = 6 cm

Skala = 1: 3.500.000

Diminta jarak yang sebenarnya?

Jb = Jp: S

Jb = 6: 1 / 3.500.000

Jb = 6 × 3.500.000

Jb = 21.000.000 cm = 210 km

Dengan demikian jarak sebenarnya adalah 210 km

Jawabannya: A

5. Maka disebut:

Jarak aktual = 3 1/2 km = 350.000 cm

Jarak pada peta = 5 cm

Ditanya tentang saldo?

S = Jp: Jb

S = 5: 350.000

S = 5 · 1 / 350.000

S = 1 / 70.000 = 1: 70.000

Jawabannya: C

6. Maka :

Skala = 1: 250

Paralelogram = 6 cm

Basis genjang sebenarnya 12,8 x 250 = 3.200 cm = 32 m

Tinggi paralogram = 6 cm

Ketinggian jajaran genjang adalah 6 x 250 = 1.500 cm = 15 m

Area jajaran genjang = a x t

Area jajaran genjang = 32 x 15 = 480 m²

7. Maka :

Tinggi aktual = jarak aktual = 60 m = 600 cm

Denah lantai = jarak di peta = 10 cm

S = 1: 600

8. Maka:

Jarak aktual = 265 km = 16.500.000 cm

Skala = 1: 1.000.000

Jp = S x Jb

Karena itu jarak pada peta adalah 26,5 cm

9. Maka :

35. contoh soal sifat asosiatif dan distributif kelas 5 sd mapel matematika


Kelas : V (5 SD)
Materi : bilangan bulat
Kata Kunci : bilangan, bulat, sifat-sifat, asosiatif, distributif

Pembahasan :
1. Penjumlahan
Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

(a + b) + c = a + (b + c)

contoh :

(2 + 6) + 8 = 2 + (6 + 8)

⇔ 8 + 8 = 2 + 14

⇔ 16 = 16


2. Pengurangan

tidak bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c tidak berlaku:

(a – b) – c ≠ a – (b – c).

contoh :

(3 - 5) - 7 ≠ 3 - (5 - 7)

⇔ -2 - 7 ≠ 3 - (-2)

⇔ -9 ≠ 5


3. Perkalian

Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c, berlaku :

(a x b) x c = a x (b x c)

contoh :

(2 x 5) x 6 = 2 x (5 x 6)

⇔ 10 x 6 = 2 x 30

⇔ 60 = 60.


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

a x (b + c) = (a x b) + (a x c)

contoh : 

–5 x [9 + (–1)] = [–5 x 9] + [–5 x (–1)] 

⇔ –5 x 8 = –45 + 5

⇔ –40 = –40


ii. distributif terhadap pengurangan

a x (b – c) = (a x b) – (a x c)

contoh :

7 x [–10 – 12] = [7 x (–10)] – [7 x 12]

⇔ 7 x -22 = -70 - 84

⇔ -154 = -154.


4. Pembagian

Tidak bersifat asosiatif, sehingga Untuk sembarang bilangan bulat a, b, dan c berlaku:

(a : b) : c ≠ a : (b : c)

contoh :

(8 : 4) : 2 ≠ 8 : (4 : 2)

⇔ 2 : 2 ≠ 8 : 2

⇔ 1 ≠ 4


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

(a + b) : c = (a : c) + (b : c)

contoh : 

(12 + 6) : 2 = (12 : 2) + (6 : 2)

⇔ 18 : 2 = 6 + 3

⇔ 9 = 9


ii. distributif terhadap pengurangan

(a – b) : c = (a : c) – (b : c)

contoh :

(21 – 9) : 3 = (21 : 3) – (9 : 3)

⇔ 12 : 3 = 7 - 3

⇔ 4 = 4
