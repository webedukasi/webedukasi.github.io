3. Air mengalir melalui pipa lingkaran berdiameter 2,0 m. Apabila kemiringan dasar saluran adalah 0,0025, hitung debit aliran apabila kedalaman aliran adalah 1,0 m. Koefisien manning n = 0,015.


Jawaban:

4,19 m^3/s

Penjelasan:

Untuk menghitung debit aliran air melalui pipa lingkaran, kita dapat menggunakan rumus debit aliran menurut Manning, yaitu:

Q = (1/n) x A x R^(2/3) x S^(1/2)

dimana:

Q adalah debit aliran (m^3/s)n adalah koefisien ManningA adalah luas penampang pipa (m^2)R adalah rasio tinggi air terhadap lebar penampang pipa (m)S adalah kemiringan dasar saluran (m/m)

Dari persamaan di atas, kita dapat menghitung debit aliran air melalui pipa lingkaran sebagai berikut:

Q = (1/0,015) x (π x (1 m)^2/4) x (1 m)^(2/3) x (0,0025 m/m)^(1/2)

Setelah menghitung, kita akan mendapatkan:

Q = 4,19 m^3/s

Jadi, debit aliran air melalui pipa lingkaran dengan diameter 2,0 m, kedalaman aliran 1,0 m, dan kemiringan dasar saluran 0,0025 adalah 4,19 m^3/s.

Penjelasan:

1. Air mengalir melalui pipa lingkaran berdiameter 2 m. Apabila kemiringan dasar saluran S = 0,0061 hitung debit aliran apa bila kedalaman aliran adalah 1, 0. Koefisien manning n = 0,0186.

Diketahui:

Diameter pipa => D = 2 m

Kemiringan dasar saluran => I = 0,0061

Kedalaman aliran => h = 1 m

Koefisien Manning => n = 0,0186

Ditanya:

Debit aliran => Q?

Jawab:

• Luas tampang aliran

A = ½ πD²/4

A = ⅛ π(2)²

A = 1,5708 m²

• Keliling basah

P = ½ πD

P = ½ π (2)

P = 3,1416 m

• Jari-jari hidraulis

R = A/P

R = 1,5708/3,1416

R = 0,5 m

• Debit Aliran

Q = AV = A × 1/n × R^⅔ × I^½

Q = 1,5708 × 1/0,0186 × (0,5)^⅔ × (0,0061)^½

Q = 4,155 m³/d

2. Air mengalir melalui pipa lingkaran berdiameter 3 m. Apabila kemiringan dasar saluran 0,0025 hitung debit aliran apabila kedalamannya 0,9 D. Koefisien Chezy C = 53,6.

Diketahui:

Diameter pipa => D = 3 m

Kemiringan dasar saluran => I = 0,0025

Kedalaman aliran => h = 0,9 D

Koefisien Chezy => C = 53,6

cos θ = OB/OC

cos θ = 0,4 D/0,5 D

cos θ = 0,8

θ = cos^-1 0,8 = 36,87°

Ditanya:

Debit aliran => Q?

Jawab:

• Luas tampang aliran

A = Luas ABCD

A = Luas AOCD + Luas AOC

A = πD²/4 × (360° - 2 × 36,87°)/360° + 2 × ½ × BC × OB

A = 0,62452 D² + 2 × ½ × 0,5 D sin (36,87°) × 0,5 D cos (36,87°)

A = 0,74452 D²

A = 0,74452 (3)²

A = 6,7 m²

• Keliling basah

P = panjang busur ADC

P = (360° - 2 × 36,87°)/360° πD

P = 2,4981 D

P = 7,49427 m

• Jari-jari hidraulis

R = A/P

R = 6,7/7,49427

R = 0,894 m

• Debit Aliran

Q = AV = AC√RI

Q = 6,7 × 53,6 √0,894×0,0025

Q = 16,978 m³/d
