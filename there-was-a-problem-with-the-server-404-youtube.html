18. __________ discussed by the board of directors when it was proposed again by the supervisors. A. The problem had already B. The problem is already C. The problem had already been D. The problem has already


PASSIVE VOICE

C. The problem had already been

ELABORATION

Jika dicermati dari konteksnya, ini merujuk pada pola kalimat pasif "Past perfect tense." "Past Perfect tense" digunakan untuk menjelaskan aksi atau tindakan di waktu tertentu pada masa lampau.

Ketika ingin membahas pelaku/subjek yang telah melakukan aksi pada waktu lampau sebelum kejadian lain muncul, ini dengan menggunakan pola kalimat aktif.

Pola/Struktur:

(+) S+had+V3+O

Contoh:

She had finished her homework before the deadline came.

Pada contoh kalimat di atas, ini memokuskan pada subjek/ si pelaku aksi (She).

Sebaliknya, ketika kalimat tersebut dirubah ke bentuk pasif, pembahasan yang awalnya fokus pada si pelaku aksi berubah ke apa atau siapa yang dikenakan aksi oleh subjek.

Pola/Struktur:

(+) O+had+been+V3+by+S

Contoh:

The homework had been finished (by her) before the deadline came.

Pada contoh kalimat di atas, ini fokus pada objek/si penerima aksi (the homework). Pada umumnya, jika Agent/subjek/pelaku aksi tidak diketahui dengan jelas, tidak dimunculkan.

25. __________ discussed by the board of directors when it was proposed again by the supervisors.A. The problem had alreadyB. The problem is alreadyC. The problem had already beenD. The problem has already


...... discussed by the board of directors when it was proposed again by the supervisors.


Pembahasan

Pertama, ketahui tense dari kalimat tersebut. Kata kuncinya terletak pada kalimat "when it was proposed". Kalimat ini merupakan bentuk past simple, yang artinya kejadian terjadi di masa lampau (telah usai).


Maka untuk melengkapi kalimat yang rumpang, tense yang digunakan juga haruslah dalam bentuk past.


A. The problem had already discussed → past perfect tense

B. The problem is already discussed → simple present tense

C. The problem had already been discussed → past perfect tense

D. The problem has already discussed → present perfect tense


Dari opsi jawaban di atas, maka kemungkinan jawabannya yaitu opsi A atau C, karena sama-sama dalam bentuk past.


Lalu manakah yang benar? Perhatikan pola kalimat "the problem ..... discussed". Kalimat ini merupakan bentuk passive voice, karena objek yang mengawali kalimat dan menggunakan kata kerja past participle (kata kerja verb 3).

Rumus past perfect tense untuk : Passive voice → O + had + been + Verb 3 (by S) Active voice → S + had + Verb 3 + O

Jadi, jawaban yang tepat adalah

[tex]\boxed{C.\:The \:problem \:had\: already \:been}[/tex]

Why You Keep Getting This Error?
“The specified module could not be found” error
message is ascribed to the following possibility:
1. Some dll. files are damaged or missing ;
2. Attack of malware, adware or other spyware;
3. Incorrect installation of applications;
4. Damaged or obsolete windows registry;
5. Outdated windows device drivers that require
updating;
6. Corrupted file allocation;
Note: If you want to fix this error, the first step is
to identify the reason why you get this error first.

RunDLL Manual Removal Guides:

1. Write down the exact error you see when the
computer starts up and then contact YooCare
Expert if you are not sure how to handle the
issue. It should be in a format such as “There
was a problem starting C:\ XXX. The specified
module could not be found,” with “XXX” being the
specific name of a file.

2. Open Control Panel from Start menu and
search for Folder Options. Go to Folder Options
window, under View tab, tick Show hidden files
and folders and non-tick Hide protected
operating system files (Recommended) and then
click OK..

3. Under Safe mode with Networking, please hit
Ctrl+Shift+Esc (Ctrl+Alt+Delete) keys together to
pull up Windows Task Manager; go to Processes
tab on top and scroll down the list to find
processes associated with “The specified module
could not be found” Message and click End
process.

4. Open Registry Editor by pressing Windows+R
keys and pull up Run box and searching force
regedit.

5. Find out and delete files and registry entries as
below:
*HKEY_LOCAL_MACHINE\Software\Microsoft
\Windows\CurrentVersion\Run
*HKEY_CURRENT_USER\Software\Microsoft
\Windows\CurrentVersion\Run
*HKEY_LOCAL_MACHINE\Software\Microsoft
\Windows\CurrentVersion\RunOnce
*HKEY_CURRENT_USER\Software\Microsoft
\Windows\CurrentVersion\RunOnce

6. Restart computer back and check out if the
error still shows up again.
