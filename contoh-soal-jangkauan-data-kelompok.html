9. suatu data yang terdiri dari 1,2,3,4,5 memiliki jangkauan j. Jika tiap data ditambah 1,maka berapakah jangkauan data yang baru?​


Untuk mencari jangkauan data, kita perlu mengurutkan data tersebut dari yang terkecil hingga terbesar dan menghitung selisih antara nilai terbesar dan nilai terkecil. Berikut ini adalah data yang telah diurutkan dari yang terkecil:

1, 2, 3, 4, 5

Jangkauan data awal adalah:

5 - 1 = 4

Jika setiap data ditambah 1, maka data baru menjadi:

2, 3, 4, 5, 6

Jangkauan data baru adalah:

6 - 2 = 4

Jadi, jangkauan data yang baru tetap 4.

37. Suatu kelompok data mempunyai jangkauan 16 dan rataan 24 jika setiap data dikalikan lalu ditambah 3 ditanya jangkauan dan rataan


Misalkan:
- data yang dimaksud adalah x(1), x(2), ... , x(n)
- data terbesar adalah x(n) dan data terkecil adalah x(1)

Oleh karena jangkauan data awal adalah 16, maka x(n) - x(1) = 16 ... (i)

Oleh karena rataan data awal adalah 24, maka 
[x(1) + x(2) + ... + x(n)] / n = 24 ... (ii)

Jika seluruh data dikalikan k, kemudian ditambah 3, maka akan diperoleh data baru sebagai berikut:

kx(1) + 3, kx(2) + 3, ..., kx(n) + 3

Jangkauan data yang baru adalah
kx(n) + 3 - [kx(1) + 3]
= kx(n) + 3 - kx(1) - 3
= kx(n) - kx(1)
= k(x(1) - x(n)) → lihat (i)
= k(16)
= 16 k

Rataan data yang baru adalah
[kx(1) + 3, kx(2) + 3, ... , kx(n) + 3] / n
= [kx(1) + kx(2) + ...  kx(n) + 3n] / n
= {k[x(1) + x(2) + ... + x(n)] + 3n} / n
= k[x(1) + x(2) + ... + x(n)] / n + 3n / n → lihat (ii)
= k(24) + 3 
= 24k + 3
