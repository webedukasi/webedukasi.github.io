Kaos = y

Ditanya :  2x + 5y

Dijawab :

x + 2y  = 150.000  ║ × 3

3x + 3y = 270.000  ║ × 1

-----------------------------------

3x + 6y = 450.000

----------------------------- -

       3y = 180.000

Ganti y ke salah satu persamaan

x + 2y = 150.000

x + 2(60.000) = 150.000

x + 120.000 = 150.000

x = 150.000 - 120.000

x = 30.000 ( Harga 1 jeans = Rp. 30.000 )

Soal ini membahas tentang SPLDV atau Sistem persamaan linear dua variabel, didalam SPLDV ada metode substitusi, eliminasi, grafik, determinan, dll.

misal :

x = celana jeans

2x + y = 165,000

2x + 25,000 = 165,000

2x = 165,000 - 25,000

2x = 140,000

Harga sebuah Celana jeans dan kaos yaitu :

31. Dilan membeli 2 kaos dan 1 jeans rp. 140.000 candra membeli 3 kaos dan 2 jeans rp. 235.000 maka harga 1 kaos dan 1 jeans adalah


Diketahui :

Kaos = x

Jeans = y

2x + y = 140.000 .......(1)

3x + 2y = 235.000 .....(2)

Ditanya :

x + y = ?

Penjelasan + Cara :

2x + y = 140.000 (Ubah menjadi persamaan y)

y = 140.000 - 2x

Substitusi y ke persamaan (2)

3x + 2y = 235.000

3x + 2(140.000 - 2x) = 235.000

3x + 280.000 - 4x = 235.000

-x + 280.000 = 235.000

-x = 235.000 - 280.000

-x = -45.000

x = 45.000

Substitusi x ke persamaan (1)

y = 140.000 - 2x

y = 140.000 - 2(45.000)

y = 140.000 - 90.000

y = 50.000

Harga x + y adalah sebagai berikut :

x + y =

= 45.000 + 50.000

= 95.000

Kesimpulan :

Maka, harga 1 kaos dan 1 jeans adalah Rp 95.000
