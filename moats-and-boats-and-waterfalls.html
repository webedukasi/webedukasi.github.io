13. There were two waterfalls at Sunny Island. The average height of the waterfalls was 805 m. The difference between the height of the waterfalls was 26 m. What is the height of the shorter waterfall?


Penjelasan dengan langkah-langkah:

diketahui :

the average of waterfalls : 805 m

the height's difference : 26 m

ditanya :

height of the shorter waterfall?

dijawab :

average = sum of all values ÷ total number of value

misal :

the longer waterfall = x

maka :

=> the height's difference (26 m)

26 = the longer waterfall - shorter waterfall

26 = x - shorter waterfall

sorter waterfall = x - 26

=> average of waterfalls (805 m)

805 = (height longer waterfall + shorter waterfall) ÷ 2

805 × 2 = height longer waterfall + shorter waterfall

1610 = x + (x - 26)

1610 = 2x - 26

1610 + 26 = 2x

1636 = 2x

818 = x

=> height of shorter waterfall

= x - 26

= 818 - 26

= 792

So, the height of the shorter waterfall is 792 m.
