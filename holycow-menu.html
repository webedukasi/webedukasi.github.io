13. menu pada Delphi 7 sebutkan menu-menu tersebut


Main Menu (Menu Utama)
Menu pada Delphi memiliki kegunaan yang sama seperti menu pada aplikasi windows. Dengan menggunakan fasilitas menu, Anda dapat memanggil atau menyimpan program, sehingga pada dasarnya semua perintah yang dikerjakan dapat ditemukan pada bagian menu utama. Nama-nama menu ditampilkan pada bagian menu bar yang terdapat pada bagian atas layar. Untuk bekerja dengan menggunakan menu, dapat menggunakan fasilitas mouse maupun keyboard.
Toolbar
Delphi memiliki beberapa toolbar yang masing-masing memiliki perbedaan fungsi dan setiap tombol pada bagian toolbar berfungsi untuk menggantikan suatu perintah menu yang sering digunakan. Toolbar terletak pada bagian bawah baris menu. Pada kondisi default, terdapat enam bagian toolbar, antara lain standard, view, debug, desktop, custom dan component palette.
Component palette
Berisi sekumpulan ikon yang melambangkan komponen-komponen yang terdapat pada VCL (Visual Component Library). Pada component palette terdapat beberapa page control seperti standard, additional, Win32, System, Data Access

24. sub menu dan menu pull down adalah menu


Dalam ilmu komputer, pointer adalah tipe data bahasa pemrograman yang nilainya mengacu untuk menunjuk langsung nilai lain yang tersimpan di tempat lain dalam suatu memori komputer menggunakan alamat yang ada. Untuk bahasa pemrograman tingkat tinggi, pointer secara efektif dapat mengambil tempat pada general purpose registers di bahasa tingkat rendah seperti bahasa assembly atau kode mesin, tetapi bila memori tersedia. 
Pointer adalah implementasi sederhana, lebih konkret tentang tipe data referensi yang lebih abstrak. Beberapa bahasa mendukung beberapa jenis pointer, walaupun ada yang memiliki lebih banyak pembatasan penggunaannya dari pada yang lain. Sebagai analogi, sebuah nomor halaman dalam buku dapat dianggap sebagai pointer ke halaman yang sesuai; dereferencing seperti pointer akan dilakukan dengan membalik ke halaman dengan nomor halaman yang diberikan. 
Pointer dapat meningkatkan kinerja data secara signifikan agar operasi berulang seperti traversing string, tabel lookup, tabel kontrol dan struktur pohon. Secara khusus, sering jauh lebih efisien dalam waktu dan space untuk pointer dari pada menggunakan variable tipe data biasa. 
Pointer juga digunakan untuk menyimpan alamat dari titik masuk untuk memanggil subrutin dalam pemrograman prosedural dan untuk run-time penghubung ke dynamic link libraries (DLL). Dalam pemrograman berorientasi objek, pointer ke fungsi yang digunakan untuk mengikat/menjilid method, ini sering disebut tabel method virtual.
Sementara "pointer" telah digunakan untuk merujuk pada referensi yang ada pada umumnya, hal ini berlaku lebih baik bagi struktur data yang secara eksplisit memungkinkan antarmuka (interface) pointer untuk dimanipulasi (secara aritmetik melalui aritmetik pointer) sebagai alamat memori. Karena pointer memungkinkan akses yang baik dilindungi maupun tidak dilindungi ke memori yang dituju. 
Pointer primitif adalah pointer sering disimpan dalam format yang sama dengan integer. Namun, mencoba untuk dereference atau "mencari" pointer yang nilainya tidak pernah menjadi alamat memori yang valid akan menyebabkan program menjadi crash.

28. menu menu yang terdapat dalam tab menu file terdiri dari


1. New.

New adalah submenu dalam menu file yang di dalam submenu tersebut terdapat beberapa jenis pemilihan file mana yang akan digunakan. Misalnya saja, blank document, blog post, recent templates, sample templates, my templates dan new from existing. Dari semua hal tersebut, terdapat jenis atau bentuk yang berbeda-beda untuk menulis dokumen. Jadi, tinggal pilih jenis mana yang akan digunakan. Fungsi dari submenu new adalah untuk membuat dokumen baru.

2. Open.

Open adalah salah satu submenu dalam menu file yang pengguna mengklik submenu tersebut maka akan muncul beberapa dokumen atau beberapa file yang sudah disimpan sebelumnya. Selanjutnya, tinggal memilih saja file atau dokumen mana yang akan dibuka. Fungsi dari submenu open dalam menu file ini adalah untuk membuka dokumen atau file yang sudah disimpan sebelumnya.

3. Save.

Save adalah salah satu submenu dalam menu file yang di jika pengguna mengklik submenu tersebut maka pengguna berarti akan menyimpan file atau dokumen mereka. Kemudian, pengguna harus memilih di mana file atau dokumen tersebut akan diletakkan. Pengguna juga dapat memilih dalam bentuk atau tipe apa file atau dokumen mereka akan disimpan. Fungsi dari submenu save dalam menu file ini adalah untuk menyimpan dokumen atau file.

4. Save as.

Save as adalah submenu dalam menu file yang fungsinya hampir sama dengan submenu save. Pengguna harus memilih mau diletakkan dimana dokumen atau file mereka. Pengguna juga akan bisa memilih dalam bentuk atau tipe apa dokumen atau file mereka yang akan disimpan. Fungsi dari submenu save as dalam menu file ini adalah untuk menyimpan dokumen atau file dengan format file yang berbeda dari penyimpanan yang pertama.

5. Print.

Print adalah submenu dalam menu file yang jika pengguna mengklik submenu tersebut berarti pengguna akan mencetak dokumen mereka. Dalam submenu print, terdapat beberapa hal untuk mengecek kembali apakah dokumen yang akan dicetak sudah benar atau belum. Fungsinya adalah untuk mencetak dokumen dengan printer.
