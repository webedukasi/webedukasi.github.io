2. arti onegaishimasu ohayougengoNIHONGO MANTAPPU​


お願いします (Onegaishimasu)

=> Mohon bantuannya, Tolong

Kata ini biasa digunakan saat ingin meminta tolong sesuatu kepada seseorang. お願い (Negai) ini memiliki arti "harapan" secara sopan. Sedangkan します (Shimasu) artinya "melakukan", dan biasa diletakkan setelah kata benda untuk mengubahnya menjadi kata kerja.

--

おはよう (Ohayou)

=> Selamat pagi

Ungkapan ini merupakan salah satu cara untuk mengungkapkan "selamat pagi" dalam Bahasa Jepang, secara kasual. Penggunaan ungkapan ini hanya ke orang yang setara, contoh nya seperti ke teman. Untuk ungkapan yang sopannya, dapat menambahkan ございます (Gozaimasu) di belakangnya.

--

言語 (Gengo)

=> Bahasa

Kata ini terdiri dari 2 kanji yaitu 言 (Gen) dan 語 (Go). Kanji 言 (Gen) dapat memiliki arti "kata", sedangkan 語 (Go) dapat memiliki arti "bahasa". Untuk menyebut nama-nama bahasa, dapat dengan menambahkan 語 (Go) setelah nama bahasa nya. Contoh, 日本語 (Nihon-go), artinya "Bahasa Jepang".

--

日本語マンタップ (Nihongo Mantappu)

=> Bahasa Jepang Mantap

Terdiri dari 2 kata yaitu 日本語 (Nihongo) yang berarti "Bahasa Jepang", dan マンタップ (Mantappu) yang merupakan serapan dari Bahasa Indonesia "mantap". Kata serapan dari luar Jepang ditulis dengan huruf Katakana.

5. Ayah nihongo de nan desuka?​


Jawaban

• お父さん ( Otousan )

‘Ayah’ untuk keluarga Orang Lain

• 父 ( ちち ) ( Chichi )

‘Ayah’ untuk keluarga Sendiri

———————

Pembahasan

Cara Menyebutkan Ayah dalam bahasa Jepang adalah ‘Otousan’ untuk keluarga Orang Lain sedangkan untuk ayah keluarga sendiri adalah ‘Chichi’

• お父さん ( Otousan )

‘Ayah’ untuk keluarga Orang Lain

• 父 ( ちち ) ( Chichi )

‘Ayah’ untuk keluarga Sendiri

Contoh :

( Otousan + ) あなたのお父さんは優しいです

Anata no Chichi wa yasashii desu!

· Ayah anda baik hati!

-------

( Chichi + ) 私の父はけんこうです

Watashi no Chichi wa kenkou desu

· Ayah saya Sehat

6. atashi wa nihongo wakarimasen


あたしは日本語わかりません

atashi wa nihongo wakarimasen

Aku tidak mengerti/tidak paham bahasa Jepang.

Pembahasan

Jika kalimat diatas ditulis dengan bentuk sepenuhnya hiragana tanpa kanji, yaitu :

あたし は にほんご わかりません (atashi wa nihongo wakarimasen).

-----------

—› あたし (atashi)

Diartikan aku atau saya. Pengucapan kata ganti ini tidak digunakan untuk lelaki, melainkan umum dipakai oleh perempuan. Bentuk kata ganti tersebut terdapat tingkatan yang hampir sama seperti kata わたし (watashi).

—› Partikel は (wa)

Hanya sekedar menjadi penanda kalimat ataupun bisa juga subjek tersebut.

—› 日本語 / にほんご (Nihongo)

Bentuk 日本語 ditulis dengan bentuk kanji, sedangkan にほんご ini ditulis dengan bentuk hiragana. Kalau dalam bahasa Indonesianya kata tersebut berarti bahasa Jepang.

—› わかりません (wakarimasen)

Berarti yaitu tidak paham/tidak mengerti. Kata ini adalah bentuk kata kerja yang sopan, juga merupakan bentuk kata kerja -Masu.

わかります (wakarimasu) : paham/mengerti, kemudian bentuk negatifnya yaitu わかりません (wakarimasen) : tidak paham/mengerti.

14. Nan nihongo watashinokazoku ?


Arti dari "Nan nihongo watashi no kazoku?"

"Nan nihongo watashi no kazoku?"

-› Apa Bahasa Jepangnya keluargaku?

Pembahasan

なん日本語私の家族.

(Nan nihongo watashi no kazoku).

Kita langsung saja pelajari kosakata-kosakatanya:

なん (Nan) artinya sama dengan 何 (Nani) yaitu "Apa", merupakan kata tanya yang berfungsi untuk menanyakan benda dan tindakan dalam situasi tertentu.日本語 (nihongo) artinya bahasa Jepang, berasal dari 日本 (Nihon) yang ditambahkan kata 語 (go) atau 言語 (gengo) yang artinya bahasa.私 (watashi) artinya saya atau aku, merupakan kata ganti (pronomina) subjek pertama yang jumlahnya tunggal menyatakan diri sendiri.の (no) tidak mempunyai makna yang khas, namun berfungsi sebagai partikel kepemilikan benda dalam kalimat apapun.家族 (kazoku) artinya keluarga, ini merupakan penyebutan untuk 'keluarga kandung/sendiri'.

23. Apa bahasa indonesia nihongo


Arti Nihongo 「日本語 」 adalah Bahasa Jepang.

Pembahasan:

Kosakata mengenai bahasa dalam Bahasa Jepang disebut dengan Gengo 「言語」. Penulisan kosakata ini disesuaikan dengan nama negara asal bahasa itu. Jika negara tersebut belum berdiri/ tidak dikenal sebelum munculnya tata penulisan Kanji, maka pada bagian negaranya ditulis dengan Katakana dan ditambah abjad Go 「語」. Contohnya sebagai berikut:

1) Bahasa Jepang:

日本 + 語 = 日本 語

(Nihon + go = Nihongo)

Semuanya ditulis dengan Kanji.

2) Bahasa Indonesia:

インドネシア + 語 = インドネシア語

(Indoneshia + go = Indoneshia-go)

Bagian Indonesia ditulis dengan Katakana.

3) Bahasa Korea:

韓国 + 語 = 韓国語

(Kankoku + go = Kankoku-go)

Semua ditulis dengan Kanji.

4) Bahasa China:

中国 + 語 = 中国語

(Chuugoku + go = Chuugoku-go)

Semua ditulis dengan Kanji.

5) Bahasa Inggris:

英 + 語 = 英語

Ei + go = Eigo

Semua ditulis dengan Kanji.

6) Bahasa Arab:

アラビア + 語 = アラビア語

Arabia + go = Arabia-go

Bagian Arabia ditulis dengan Katakana.

27. kalau " nihongo wo oshiemasu " berarti jadi " nihongo wo yonde kudasai?​


Jawaban:

Jawaban:

Bukan, Pelajaran tersebut merupakan perubahan K. Kerja bentuk ます menjadi K.Kerja bentuk て

Saya jawab pakai romaji saja ya, biar enak dibacanya..

-----------------------------------------------------------------------------------------------------------

[Perubahan K. Kerja bentuk ます menjadi K.Kerja bentuk て]

Dibagi menjadi beberapa kelompok yaitu:

1. Berakhiran huruf berikut sebelum Masu い(I), ち(Chi), り(Ri)

Maka ketiga huruf tersebut harus diganti dengan って(tte) dan ます(Masu) dihilangkan.

Contoh:

Kaimasu (Katte) Membeli

Tachimasu (Tatte) Berdiri

Machimasu (Matte) Menunggu

Hairimasu (Haitte) Masuk

-----------------------------------------------------------------------------------------------------------

2. Berakhiran huruf berikut sebelum Masu み(Mi), び(Bi), に(Ni)

Maka ketiga huruf tersebut harus diganti dengan んで(nde) dan ます(Masu) dihilangkan.

Contoh:

Yomimasu (Yonde) Membaca

Yobimasu (Yonde) Memanggil

Shinimasu (Shinde) Meninggal

*Bentuk て dari yomimasu dan yobimasu memang sama, tetapi tidak akan membuat salah paham karena kanjinya beda dan apa yang berkaitan dengan kata kerja tsb pasti beda.

-----------------------------------------------------------------------------------------------------------

3. Berakhiran dengan huruf し(Shi)

Maka cukup mengganti Masu dengan て(Te)

Contoh:

Hanashimasu (Hanashite) Berbicara

Kashimasu (Kashite) Meminjamkan

-----------------------------------------------------------------------------------------------------------

4. Kata kerja yang hurufnya bervokal え(E) sebelum Masu

Maka cukup mengganti Masu dengan て(Te)

Contoh:

Oshiemasu (Oshiete) Memberi tahu/Mengajari

Akemasu (Akete) Membuka

Tsukemasu (Tsukete) Menyalakan

Kaemasu (Kaete) Menukar

-----------------------------------------------------------------------------------------------------------

5. Berakhiran き(Ki) dan ぎ(Gi)

Maka き(Ki) dan ぎ(Gi) perlu diganti いて(Ite) dan いで(Ide) dan ます(Masu) dihilangkan.

Contoh:

Kakimasu (Kaite) Menulis

Kikimasu (Kiite) Mendengarkan

Oyogimasu (Oyoide) Berenang

-----------------------------------------------------------------------------------------------------------

6. Kata kerja bentuk Shimasu

Maka Masu pada Shimasu diganti て(Te)

Contoh:

Kekkon shimasu (Kekkon shite) Menikah

Kaimono shimasu (Kaimono shite) Berbelanja

Shokuji shimasu (Shokuji shite) Makan

-----------------------------------------------------------------------------------------------------------

Mungkin terdapat sebagian kecil K.Kerja yang memiliki "pengecualian" sehingga perubahan tidak menggunakan dasar-dasar diatas, namun dapat dibedakan dengan sering-sering membaca dan berlatih.

Contoh:

Mimasu (Mite) Melihat

Ikimasu (Itte) Pergi

Karimasu (Karite) Meminjam

Okimasu (Okite) Bangun

dan lain-lainnya..

-----------------------------------------------------------------------------------------------------------

先生 (sensei)

Artinya adalah guru.

は (wa)

Merupakan salah satu partikel yang dimana dalam konteks kalimat ini berfungsi sebagai penanda kalimat utama terhadap suatu objek dan subjek.

日本語 (nihon go)

Artinya dalam bahasa Indonesia adalah bahasa Jepang. Dimana kata 日本 (nihon) artinya adalah Jepang, 語 (go) yang berarti bahasa.

の (no)

Adalah salah satu jenis bentuk partikel yang umumnya digunakan untuk menyatakan suatu kepemilikan dan juga untuk menegaskan penegasan.

です (desu)

Merupakan salah satu partikel yang umumnya suka digunakan disetiap akhir kalimat dan juga sebagai menyatakan suatu penegasan.

38. Q. Bahasa Jepang dari “Belajar Bahasa Jepang” ...A. Nihongo O Manabu.B. Nihongo O Narau.C. Nihongo O Benkyou.D. Benkyou O Nihongo.​


Bahasa Jepang dari “Belajar Bahasa Jepang" adalah...

A. Nihongo o manabu.

                     

Penjelasan

Pilihan B saya rasa kurang tepat, karena kata 'narau' digunakan jika mempelajari suatu hal dari seseorang. Hal yang dipelajari biasanya juga berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.

Contoh kalimat menggunakan 'narau' ↴

   父から空手を習います。

   = Chichi kara karate o naraimasu.

   = Belajar karate dari ayah.

Pilihan C kurang tepat, karena kata 'benkyou' merupakan kata benda. Kalimat tersebut bisa tepat bila penulisannya menjadi 'Nihongo o benkyou suru'. Kata 'suru' digunakan untuk mengubah kata benda 'benkyou' menjadi kata kerja.

Pilihan D jelas salah, karena susunan kalimatnya yang tidak tepat dan kata 'benkyou' bukanlah kata kerja. Harusnya kata 'benkyou' diganti dengan 'benkyou suru', dan pola kalimat yang tepat harusnya "[Objek] o [Predikat]".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Perbedaan 'Benkyou suru', 'Manabu', dan 'Narau' :

・Benkyou suru = "Belajar" yang lebih fokus ke hal-hal yang berhubungan dengan akademik.

・Manabu = "Belajar" yang cakupannya lebih luas dari 'benkyou suru'.

・Narau = “Belajar” yang digunakan ketika jelas ada seseorang yang mengajarkan kita akan sesuatu, dan biasa berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.
