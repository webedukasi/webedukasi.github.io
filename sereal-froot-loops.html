1. jelaskan singkat fungsi dari:1. For loops2. while loops3. repeat-until loops​


for loop, while loop, dan do-while loop menggunakan prinsip yang sama hanya saja terdapat perbedaan diantara lainnya :

1. `for-loop`

for-loop dikhususkan untuk mencapai target dengan interval atau step yang telah ditentukan. Dalam hal ini seperti contoh dibawah

int target = 10;

for(int i = 0; i < target; i++){

}

dalam hal ini, variabel awal i merupakan inisiator untuk memulai tahapan, dilanjutkan dengan melakukan pengecekan terhadap variabel selanjutnya yakni target. apabila target terpenuhi maka looping akan dihentikan. Sebaliknya, apabila target belum terpenuhi maka variabel i akan bertambah dan loop akan berlanjut

2. `while` loop

while loop merupakan looping yang sifatnya tidak diketahui kapan akan berhenti apabila suatu kondisi belum terpenuhi, contoh :

while(boolean makan = true){

if(sudahMakan){

makan = makan;

}

apabila variabel makan bernilai false, maka esekusi while-loop akan dihentikan karena sesuai dengan kondisi yang diminta

3. `do-while`

do-while tidak jauh berbeda dengan while loop, hanya saja do-while melakukan pengecekan diawal dan dilanjutkan dengan looping apabila kondisi terpenuhi, contoh :

do{

mainGame();

}while(jam != "13:00");

disini perintah yang ada diblok do akan dilakukan sekali, lalu dicek kondisinya. apabila kondisi belum terpenuhi maka perintah pada blok do akan diulang kembali hingga kondisi jam terpenuhi

4. Buatlah program Java nested loops..mohon bantuannya ka​


Kode Program Java

import java.util.Scanner;

public class LatihanNestedLoops {
   // Method untuk meminta masukan berupa angka
   private static int inputAngka(String teks, boolean closeScanner) {
       int tmp = 0;
       Scanner in = new Scanner(System.in);
       try {
           System.out.print(teks);
           tmp = in.nextInt();
       } catch (Exception e) {
           System.out.println("Terjadi kesalahan.");
       } finally {
           if (closeScanner) in.close();
       }
       return tmp;
   }

   // Method untuk menentukan karakter X atau O
   // dari parameter baris dan kolom
   private static char xAtauO(int b, int k) {
       if (b % 2 != 0)
           // Untuk baris ganjil, X pada kolom ganjil
           return (k % 2 != 0) ? 'X' : 'O';
       else
           // Untuk baris genap, X pada kolom genap
           return (k % 2 == 0) ? 'X' : 'O';
   }

   // PROGRAM UTAMA
public static void main(String args[]) {
       // Meminta masukan jumlah baris dan kolom
       int baris = inputAngka("Input jumlah baris: ", false);
       int kolom = inputAngka("Input jumlah kolom: ", true);
       System.out.println("=======================");
       for (int b = 1; b <= baris; b++) {
           for (int k = 1; k <= kolom; k++) {
               System.out.print(xAtauO(b, k) + " ");
           }
           System.out.println();
       }
   }
}
__________________

Pembahasan

Karena tidak ada spesifikasi yang lebih rinci, maka diasumsikan karakter X dan O dicetak bergantian setiap berganti nomor kolom, juga setiap berganti nomor baris. Jadi, pada baris bernomor ganjil, karakter X dicetak pada kolom bernomor ganjil. Pada baris bernomor genap, karakter X dicetak pada kolom bernomor genap.

Algoritma utama penentuan karakter X atau O yang akan dicetak pada baris dan kolom tertentu terletak pada method xAtauO, sehingga pada method main, kita hanya perlu memanggil method xAtauO pada loop for bagian dalam, dengan parameter variabel loopin baris dan kolomnya.

Contoh hasil eksekusi program dapat dilihat pada gambar yang disertakan. Terdapat 3 contoh kasus, yaitu untuk jumlah baris dan kolom berturut-turut 3 dan 5, 4 dan 6, serta 11 dan 15.

10. tuliskan cara penyimpanan sereal, yang ada di kotak sereal (bantu aku plisss, yg ada di kotak sereal kalian aja)​


Jawaban:

Penyimpanan sereal yang ada dalam kotak sebaiknya dilakukan dengan baik untuk menjaga kesegaran dan kualitasnya. Berikut adalah cara penyimpanan sereal yang disarankan:

1. Simpan dalam Wadah Tertutup: Setelah membuka kotak sereal, segera pindahkan isinya ke dalam wadah penyimpanan yang kedap udara, seperti wadah plastik atau toples kaca dengan penutup yang rapat. Ini membantu mencegah sereal dari paparan udara yang dapat membuatnya cepat kehilangan kesegaran.

2. Jauhkan dari Suhu Panas dan Kelembapan: Sereal sebaiknya disimpan dalam tempat yang sejuk dan kering, jauh dari sinar matahari langsung dan kelembapan. Sereal yang terkena panas dan kelembapan dapat menjadi cepat basah atau kehilangan rasa krispnya.

3. Periksa Tanggal Kadaluarsa: Pastikan untuk memeriksa tanggal kadaluarsa pada kemasan sereal dan gunakan sereal sebelum tanggal tersebut untuk memastikan kesegarannya.

4. Gunakan Penutup Rapat: Pastikan wadah penyimpanan sereal tertutup dengan rapat setelah mengambil sejumlah sereal. Ini membantu menjaga kualitas sereal dan mencegah serangga atau kelembapan masuk ke dalamnya.

5. Hindari Menyimpan di Dekat Bahan-Bahan yang Berbau Kuat: Sereal memiliki kemampuan menyerap aroma, jadi hindari menyimpannya di dekat bahan-bahan yang berbau kuat seperti bawang atau bumbu lainnya yang dapat mempengaruhi rasanya.

Dengan menjaga sereal Anda dengan baik, Anda dapat memastikan bahwa mereka tetap segar dan nikmat untuk dikonsumsi dalam waktu yang lebih lama.

Banyak cereal yang Rofiq makan adalah 333,33 gram. Jumlah cereal dengan jumlah kalori pada cereal membentuk perbandingan senilai.

Pembahasan

PERBANDINGAN SENILAI

Perbandingan senilai adalah perbandingan dimana jika sebuah pembandingnya naik, maka pembanding lainnya juga akan naik. Sebaliknya, jika sebuah pembandingnya turun, maka pembanding lainnya akan turun.

Contoh jumlah buku dengan harga buku.

Jika jumlah buku bertambah, maka harga buku bertambah. Jika jumlah buku berkurang, maka harga buku pun akan berkurang.

Cara untuk menyelesaikannya:

Misalkan dahulu yang belum diketahui dengan sebuah variabel seperti a, m, atau p.
