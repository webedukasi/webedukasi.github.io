21. sebuah kereta api berangkat dari jakarta pukul 06.45 dan tiba di cirebon pukul 10.30 berapa kecepatan kereta api jika jarak jakarta cirebon 270 km​


Penjelasan dengan langkah-langkah:

Untuk mencari kecepatan rata-rata kereta api, kita perlu menggunakan rumus:

kecepatan = jarak / waktu

Jarak antara Jakarta dan Cirebon adalah 270 km, sedangkan waktu yang dibutuhkan kereta api untuk menempuh jarak tersebut adalah:

10 jam 30 menit - 6 jam 45 menit = 3 jam 45 menit = 3.75 jam

Dalam satuan waktu yang sama, waktu tersebut dapat diubah menjadi:

3.75 jam = 3 jam + 0.75 jam = 3 jam + 45 menit

Dengan demikian, kita bisa menghitung kecepatan rata-rata kereta api dengan rumus:

kecepatan = jarak / waktu = 270 km / 3,75 jam = 72 km/jam

Jadi, kecepatan rata-rata kereta api dari Jakarta ke Cirebon adalah 72 km/jam.
