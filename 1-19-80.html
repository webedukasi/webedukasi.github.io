22. Jika diketahui k + 1,2 =6/15 + 1 2/3 : 80% maka nilai k yang memenuhi persamaan tersebut adalah HOTS a.1 4/15 b. 1 17/60 C. 1 3/10 d. 1 19/60​


Jawaban:

b.117/60

Penjelasan dengan langkah-langkah:

Mari selesaikan persamaan ini:

k + 1,2 = 6/15 + 1 2/3 : 80%

Langkah pertama, kita akan mengevaluasi ekspresi di sebelah kanan tanda sama dengan:

1 2/3 : 80% = 5/3 : 80% = 5/3 : 0,8 = (5/3) / (8/10) = (5/3) * (10/8) = (5/3) * (5/4) = 25/12

Kemudian, kita akan menyederhanakan sisi kanan:

6/15 + 25/12 = 2/5 + 25/12

Untuk menjumlahkan fraksi ini, kita perlu menemukan denominasi bersama, yang dalam hal ini adalah 60:

(2/5) * (12/12) + (25/12) * (5/5) = 24/60 + 125/60 = 149/60

Jadi, persamaannya sekarang menjadi:

k + 1,2 = 149/60

Untuk menemukan nilai k, kita kurangkan 1,2 dari kedua sisi:

k = 149/60 - 1,2

Untuk mengurangkan 1,2 dari 149/60, kita perlu mengubah 1,2 menjadi pecahan dengan denominasi 60:

1,2 = 1 + 2/10 = 1 + 12/60 = 60/60 + 12/60 = 72/60

Jadi, k = (149/60) - (72/60) = 77/60

Sekarang, kita bisa menyederhanakan nilai k:

k = 77/60 = 1 17/60

Jadi, nilai k yang memenuhi persamaan tersebut adalah b. 1 17/60.

35. Berapa persentase komposisi massa dari halotan (C2HBrC1F3), suatu obat bius (anestesi)? Diketahui Ar: C = 12;H = 1 ; Br =80; C1 =35,5; F =19.


Persentase komposisi massa dari halotan adalah 12,152 % C, 0,506 % H, 40,506 % Br, 17,975 % Cl, dan 28,861 % F. Angka tersebut dapat ditentukan menggunakan rumus penentuan kadar zat. Pada penghitungannya dapat digunakan data Ar sebagai pendukung.

Pembahasan

Pada soal di saat diketahui beberapa data yaitu:

senyawa = C₂HBrClF₃

Ar C = 12; H = 1; Br = 80; Cl = 35,5; F = 19

Adapun yang ditanyakan adalah komposisi massa dar masing-masing komponen dalam halotan (C₂HBrClF₃).

Penghitungan kadar zat dapat ditentukan menggunakan beberapa cara yaitu:

1) kadar zat dalam persen (%)

Kadar zat dapat diukur dalam satuan %. Kadar zat ini dapat berasal dari perbandingan massa/massa (w/w) atau dalam perbandingan volume/volume (v/v). Berikut rumus yang dapat digunakan:

Kadar (%) = (massa komponen/massa campuran) x 100%

Kadar (%) = (volume komponen/volume campuran) x 100%

2) kadar zat dalam bpj

Dalam satuan bpj kadar zat ditandai dengan adanya perkalian perbandingan komponen dan campuran dengan 10⁶

Kadar (bpj) = (massa komponen/massa campuran) x 10⁶

Kadar (bpj) = (volume komponen/volume campuran) x 10⁶

Pada soal di atas diiginkan persentase unsur dalam suatu senyawa. Rumus yang digunakan adalah rumus kadar dalam satuan persen. Persentase suatu unsur dalam senyawa dapat ditentukan menggunakan perbandingan massa atom relatif dengan massa molekul relatif dari senyawa yang diketahui. Massa molek relatif dari halotan (C₂HBrClF₃) adalah:

Mr = 2 (Ar C) + Ar H + Ar Br + Ar Cl + 3 (Ar F)

    = 2 (12) + 1 + 80 + 35,5 + 3 (19)

    = 24 + 1 + 80 + 35,5 + 57

    = 197,5  

Selanjutnya persentase masing-masing unsur dapat ditentukan sebagai berikut:

% C   = (2. Ar C/ Mr C₂HBrClF₃) x 100%

        = (24/ 197,5) x 100%

        = 12,152 %

% H  = (Ar H/ Mr C₂HBrClF₃) x 100%

        = (1/ 197,5) x 100%

        = 0,506 %

% Br  = (Ar Br/ Mr C₂HBrClF₃) x 100%

         = (80/ 197,5) x 100%

         = 40,506 %

% Cl   = (Ar Cl/ Mr C₂HBrClF₃) x 100%

         = (35,5/ 197,5) x 100%

         = 17,975 %

% F    = (3. Ar F/ Mr C₂HBrClF₃) x 100%

         = (57/ 197,5) x 100%

         = 28,861 %

Jadi dapat disimpulkan halotan tersusun dari 12,152 % C, 0,506 % H, 40,506 % Br, 17,975 % Cl, dan 28,861 % F.
