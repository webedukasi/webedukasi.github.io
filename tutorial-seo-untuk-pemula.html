Kalor adalah bentuk energi yang berpindah dari suatu benda bersuhu tinggi ke benda yang bersuhu rendah jika ditempelkan, satuan kalor adalah joule atau kalori

Diketahui :

m = massa = 200 gr = 0,2 kg

L = Kalor lebur = 336.000 J/kg°C

Ces = Kalor jenis es = 2100 J/kg°C

Cair = Kalor jenis air = 4200 J/kg°C

T2 = Suhu 2 = 0°

T3 = Suhu 3 = 100°

∆T = T akhir - T awal= Perubahan suhu

Q = Kalor = 4200 joule

Ditanya :

A.) T1 = ...?

B.) Q1 = ...?

C.) Q2 = ...?

Jawab :

Rumus Kalor adalah

Q = m × c × ∆T (jika ada perubahan suhu)

Q = m × L atau U (jika tidak ada perubahan suhu)

A.) Dalam Q, terjadi perubahan suhu, maka :

Q = m × ces × (T2-T1)

4200 = 0,2 × 2100 × (0-T1)

4200 = 420 × (0-T1)

4200 = 0 - 420T1

4200/-420 = T1

-10 = T1

T1 = -10°

Kesimpulan :

Jadi nilai T1 adalah -10°

B.) Karena tidak ada perubahan suhu, maka :

Q = m × L

Q = 0,2 × 336.000

Q = 67.200 J

Q = 67,2 Kj

Kesimpulan :

Jadi nilai Q1 adalah 67.200 J

C.) Karena ada perubahan suhu, maka :

Q = m × c × ∆T

Q = 840 × 100

Q = 84000 J

Q = 84 Kj

Jadi nilai Q2 adalah 84000 J

11. Apa perbedaan antara SEO Onpage dan SEO Offpage?


SEO On Page adalah proses SEO yang dilakukan hanya di bagian internal situs web kamu saja. Jadi, dalam proses On Page ini, kamu benar-benar memaksimalkan proses SEO hanya di bagian dalam situs web saja tanpa melibatkan pihak luar. SEO On Page menjadi salah satu faktor terbesar untuk penentuan ranking situs web kamu. Jika SEO Off Page kamu bagus tetapi tidak didukung oleh On Page yang baik pula, maka semua usaha akan sia-sia.

Lain lagi dengan SEO Off Page. Proses SEO yang satu ini dilakukan benar-benar di luar bagian situs web atau eksternal. Salah satu contoh proses SEO Off Page adalah dengan meletakkan link situs web kita di situs web lain atau istilahnya backlink. Dengan adanya backlink ini, diharapkan pengguna dari berbagai situs web lain bisa mampir ke situs web kita.

tutorial (bimbingan) dasarr untuk pemula
