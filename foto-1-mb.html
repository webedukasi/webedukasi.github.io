11. 1.  konvesikan satuan ukuran data berikut ini :4 Byte = .......bit128 KB = ......bit1,5 MB = .......B8GB = ........ KB2.  jika rata rata file foto mempunyai ukuran 2,5 MB, sedangkan kamu mempunyai flashdisk ukuran 16 GB , maka berapa maksimal jumlah foto yang dapat disimpan di flashdisk kamu?​


Soal Pertama

Hasil Konversi

4 Byte = 32 bit128 KB = 1.024.000 bit1,5 MB = 1.500.000 Byte8 GB = 8.000.000 Byte

Soal Kedua

Foto yang dapat disimpan dengan ukuran 2,5 MB dan disimpan dalam Flashdisk berukuran 16 GB adalah 6.400 foto.

Soal tersebut merupaka soal tentang perangkat keras komputer.

Pembahasan dengan Langkah-Langkah

Soal tersebut merupakan soal TIK yang membahas tentang perangkat keras komputer. Flashdisk merupakan perangkat keras komputer yang berfungsi untuk menyimpan data-data yang kita butuhkan pada komputer. Untuk menyelesaikan soal tersebut kita harus mengkonversi ukuran file yang ada pada komputer.

Konversi

1 Byte (B) = 8 bit (b)1 Kilo bit (kb) = 1024 bit (b)1 Kilo Byte (KB) = 1024 Byte (B) = 8192 bit (b)1 Mega Byte (MB) = 1024 Kilo Byte (Biner) = 1000 Kilo Byte (Desimal) = 1.048.576 Bytes (Biner) = 1.000.000 Bytes (Desimal)1 Giga Byte (GB) = 1.000 MB = 1.000.000 KB

Penyelesaian soal
Soal Pertama

Diketahui:

Pertanyaan pada soal

Ditanyakan:

4 Byte = .......bit128 KB = ......bit1,5 MB = .......Byte8GB = ........ KB

Jawab:

4 Byte = 4 x 8 = 32 bit128 KB = 128.000 KB = 128.000 x 8 = 1.024.000 bit1,5 MB = 1,5 x 1.000.000 = 1.500.000 Byte8 GB = 8 x 1.000.000 = 8.000.000 KB

Soal Kedua

Diketahui:

Foto berukuran 2,5 MBFlashdisk berkapasitas 16 GB

Ditanyakan:
Tentukan banyak foto yang dapat disimpan dalam flashdisk tersebut!

Jawab:

n = 16 GB : 2,5 MBn = 16.000 : 2,5n = 6.400

Jadi, banyak foto berukuran 2,5 MB yang dapat disimpan dalam flashdisk berukuran 16 GB adalah 6.400 foto.

Total penggunaan saat ini = 250 MB (foto) + 425 MB (aplikasi) + 50 MB (dokumen) + 200 MB (audio) = 925 MB

Kapasitas memori yang tersedia = 1024 MB (1 GB) - 925 MB = 99 MB

Brian ingin menyimpan file video dengan ukuran 300 MB, namun hanya memiliki sisa 99 MB. Oleh karena itu, Brian harus menghapus salah satu file yang sudah ada.

Dari semua file yang ada, file dengan ukuran paling kecil adalah "Dokumen" yang berukuran 50 MB. Jadi, Brian harus menghapus file "Dokumen" agar dapat menyimpan file video berukuran 300 MB.

37. seorang siswa mempunyai flash disk dengan kapasitas penyimpanan sebesar 8 Gb, di dalam flashdisk tersebut diisi beberapa data yaitu 100 foto yang masing-masing foto berukuran 20 MB dan file video sebesar 1.450 MB serta 1 file aplikasi sebesar 4,12 GB. berapakah sisa kapasitas penyimpanan flashdisk siswa tersebut?pke cara ya.​


Jawaban:

3,701,518 MB.

Penjelasan:

Jumlah kapasitas penyimpanan yang digunakan oleh foto adalah 100 foto x 20 MB/foto = 2.000 MB.

Sedangkan jumlah kapasitas penyimpanan yang digunakan oleh video adalah 1.450 MB.

Total kapasitas penyimpanan yang digunakan adalah 2.000 MB + 1.450 MB + 4,12 GB = 7.570 MB + 4,12 GB = 7.570 MB + 4,290,912 MB = 4,298,482 MB.

Sisa kapasitas penyimpanan flashdisk siswa tersebut adalah 8 GB - 4,298,482 MB = 3,701,518 MB.

Jawaban:

Untuk mengetahui sisa kapasitas penyimpanan flash disk siswa tersebut, pertama-tama harus menghitung jumlah ukuran total dari semua file yang ada di dalam flash disk tersebut.

Jumlah ukuran total dari foto-foto tersebut adalah 100 foto x 20 MB/foto = 2.000 MB

Jumlah ukuran total dari semua file yang ada di flash disk tersebut adalah 2.000 MB + 1.450 MB + 4.12 GB = 7.570 MB + 4.12 GB = 4,93 GB.

Kemudian, untuk mengetahui sisa kapasitas penyimpanan flash disk tersebut, bisa mengurangi jumlah ukuran total file dari kapasitas penyimpanan flash disk yaitu 8 GB - 4,93 GB = 3,07 GB. Jadi, sisa kapasitas penyimpanan flash disk siswa tersebut adalah 3,07 GB.

Jika diinginkan, bisa juga mengubah jumlah ukuran total file yang ada di flash disk tersebut ke dalam satuan MB, sehingga sisa kapasitas penyimpanan flash disk siswa tersebut dapat dinyatakan dalam satuan MB juga.

Jumlah ukuran total file yang ada di flash disk tersebut adalah 4,93 GB = 4.930 MB.

Sisa kapasitas penyimpanan flash disk siswa tersebut adalah 8.000 MB - 4.930 MB = 3.070 MB.
