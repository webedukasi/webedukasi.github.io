Diketahui :

Perbandingan bilangan A : B : C = 5:4:1 jumlah A,B,C adalah 150

Ditanya :

Tentukan nilai setiap bilangan !

Dijawab :

5 : 4 : 1 = 5 + 4 + 1

= 10

A :

[tex] a= \frac{5}{10} \times 150 = 75[/tex]

B :

[tex]b = \frac{4}{10} \times 150 = 60[/tex]

C :

[tex]c = \frac{1}{10} \times 150 = 15[/tex]

Jadi banyak Bilangan A : B : C Adalah :

A = 75

B = 60

C = 15
