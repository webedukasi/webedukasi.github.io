37. (+) (-)(?) do Indonesia have nightmare this night​


Jawaban

(+) Indonesia have nightmare this night.

(-) Indonesia don't have nightmare this night.

(?) Do Indonesia have nightmare this night?

PenjelasanPresent Simple Tense

=> digunakan untuk menyatakan kejadian yang terjadi sekarang, kebenaran umum, dan kebiasaan.

Present Simple Tense, baik kalimat positif, negatif, atau tanya menggunakan Verb 1.

Rumus Present Simple Tense

1. Verbal, I/You/We/They

(+) S + V1 + O

(-) S + don't + V1 + O

(?) Do + S + V1 + O + ?

2. Verbal, He/She/It

(+) S + V1 + s/es + O

(-) S + doesn't + V1 + O

(?) Does + S + V1 + O + ?

3. Nominal

(+) S + is/am/are + Adjective/Adverb/Noun

(-) S + is/am/are + not + Adjective/Adverb/Noun

(?) Am/Is/Are + S + Adjective/Adverb/Noun

NB:

1) Am digunakan untuk Subject I.

2) Is digunakan untuk Subject He, She, It.

3) Are digunakan untuk Subject You, We, They.

Keterangan:

(+) = Kalimat positif (positive)

(-) = Kalimat negatif (negative)

(?) = Kalimat tanya (interrogative)

S = Subjek (Subject)

O = Objek (Object)

V1 = Verb 1

Adjective = Kata sifat

Noun = Kata benda

Adverb = Kata keterangan
