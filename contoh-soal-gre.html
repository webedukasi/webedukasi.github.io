Fisika - Fluida Dinamis (Persamaan Bernoulli)

﻿﻿Numpang coba:3

Diketahui : 
Massa Jenis = p(rho)
A1 = πr²
A2 = π (r/2)^2
v1 = v0
P1 = P0
P2 = ?

Pertama, dicari dulu kecepatan di posisi kedua menggunakan persamaan kontinuitas, yaitu :

Q1 = Q2
A1 v1 = A2 v2
πr² . v0 = π(r/2)² . v2
v2 = πr² . v0 / π (r/2)²
v2 = r² . v0 / (r/2)²
v2 = r² . v0 / r² / 4
v2 = 4v0

Masukan ke Persamaan Bernoulli yaitu :
(anggap p kecil adalah rho)

P1 + p g h + 1/2 p v1² = P2 + p g h + 1/2 p v2²
*karena ketinggian sama, kita bisa coret (p g h)*
P0 + 1/2 pv0² = P2 + 1/2 p 4v0²
P2 = P0 + 1/2 pv0² - 1/2 . p . 4v0²
P2 = P0 + 1/2 p ( v0²- (4v0)²)
P2 = P0 + 1/2 p ( -15v0²)
P2 = P0 + -15/2 p v0² 
P2 = P0 - 15/2 p v0² (D)
