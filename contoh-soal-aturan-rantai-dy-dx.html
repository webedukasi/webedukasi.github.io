8. jika y=in(sin(x^3-1)), tentukan dy/dx dengan menggunakan aturan rantai turunan (derivatif)​


Jika sebuah bilangan dengan nilai y=in(sin(x^3-1)). Tentukanlah nilai dy/dx dengan menggunakan aturan rantai turunan (derivatif)​. Niali dy/dx adalah [tex]3x^{2} .cot(x^{3} -1)[/tex].

Penjelasan dengan langkah-langkah

Diketahui :

Jika sebuah bilangan dengan nilai y=in(sin(x^3-1)).

Ditanya : Tentukanlah nilai dy/dx dengan menggunakan aturan rantai turunan (derivatif)​.

Jawab:

[tex]y=in(sin(x^{3} -1))\\\frac{dy}{dx} =\frac{1}{sin(x^{3}-1) } .\frac{d(sin(x^{3}e-1) }{dx} \\=\frac{1}{sin(x^{3}-1) } cos(x^{3}-1).\frac{d(x^{3}-1) }{dx} \\=\frac{1}{sin(x^{3}-1)} cos(x^{3} -1)(3x^{2} )\\=\frac{cos(x^{3}-1) }{sin(x^{3}-1 )}(3x^{2} )\\ = 3x^{2} cot(x^{3} -1)[/tex]

Jadi, Niali dy/dx adalah [tex]3x^{2} .cot(x^{3} -1)[/tex].

a. Untuk menemukan dy/dx dari persamaan y = cos^4(x^2 + 4), kita akan menggunakan aturan rantai. Pertama, kita identifikasi fungsi luar yaitu cos^4(u) dengan u = x^2 + 4, dan fungsi dalamnya adalah u = x^2 + 4.

Langkah-langkah:

1. Tentukan turunan parsial dy/du dengan memperhatikan fungsi luar.

dy/du = 4cos^3(u)

2. Selanjutnya, tentukan turunan parsial du/dx dengan memperhatikan fungsi dalam.

du/dx = 2x

3. Terakhir, kalikan turunan parsial dy/du dengan turunan parsial du/dx untuk mendapatkan dy/dx menggunakan aturan rantai.

dy/dx = (dy/du) * (du/dx)

= 4cos^3(u) * 2x

= 8x * cos^3(x^2 + 4)

b. Untuk menemukan dy/dx dari persamaan y = √((3 + 2x)/(2 - 3x)), kita akan menggunakan aturan rantai. Pertama, kita identifikasi fungsi luar yaitu √u dengan u = (3 + 2x)/(2 - 3x), dan fungsi dalamnya adalah u = (3 + 2x)/(2 - 3x).

dy/du = 1/(2√u)

dy/dx = (dy/du) * (du/dx)
