6. You cannot download all files because the assignment submission item greater than 500mb. Apa arti penggalan kode diatas?


ini dlm bahasa indonesia nya

Anda tidak dapat mengunduh semua file karena item pengiriman tugas lebih dari 500mb.

jadi, anda tidak bisa kirim tugas karena tugas yg anda kirim melebihi batas maksimal pengiriman, yaitu 500mb.

saran saya, anda bisa mengcompress dulu file tugas anda menjadi ukuran yang lebih kecil.

jika tugas anda berupa video, anda bisa mencari di google "compress video" lalu klik web paling atas dan masukkan file yang anda tuju, dan otomatis ukuran file anda akan mengecil.

jika tugas anda berupa file hasil scanning, di aplikasi scanning yang anda gunakan, pilih option compress size file, lalu tekan dan ukuran file anda akan mengecil.
