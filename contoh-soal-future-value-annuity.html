1. . Berapa Future dan Present Value dari uang sebesar Rp 150.000.000 lima tahun mendatang/yang akan diterima lima tahun mendatang jika tingkat suku bunga yang berlaku 15% per tahun? 2. Soal Future Value; pembayaran Rp 100.000.000 yang dilakukan selama 4 tahun berturut-turut jika tingkat suku bunga yang berlaku 12% Per tahun. Hitunglah berapa nilai Ordinary Annuity dan Annuity Due! 3. Soal Present Value: penerimaan dana sebesar Rp 50.000.000 setiap akhir/awal tahun selama 4 tahun dengan tingkat suku bunga 11% Per tahun. Hitunglah berapa nilai Ordinary Annuity dan Annuity Due!


Untuk menghitung Future Value dan Present Value dari suatu jumlah uang, kita dapat menggunakan rumus-rumus yang telah ditentukan.

   Future Value dari uang sebesar Rp 150.000.000 lima tahun mendatang/yang akan diterima lima tahun mendatang jika tingkat suku bunga yang berlaku 15% per tahun adalah Rp 255.762.500. Rumus yang digunakan adalah:

Future Value = Present Value x (1 + Interest Rate)^Number of Periods

Future Value = 150.000.000 x (1 + 0,15)^5

Future Value = 255.762.500

   Future Value dari pembayaran Rp 100.000.000 yang dilakukan selama 4 tahun berturut-turut jika tingkat suku bunga yang berlaku 12% Per tahun adalah Rp 547.731.132. Rumus yang digunakan adalah:

Future Value = Payment x (1 + Interest Rate)^Number of Periods - 1/[(1 + Interest Rate)^Number of Periods - 1]

Future Value = 100.000.000 x [(1 + 0,12)^4 - 1]/[(1 + 0,12)^4 - 1]

Future Value = 547.731.132

Nilai Ordinary Annuity dari pembayaran tersebut adalah Rp 491.541.465, yaitu hasil dari pembayaran selama 4 tahun tanpa memperhitungkan suku bunga. Rumus yang digunakan adalah:

Ordinary Annuity = Payment x [((1 + Interest Rate)^Number of Periods - 1)/Interest Rate]

Ordinary Annuity = 100.000.000 x [((1 + 0,12)^4 - 1)/0,12]

Ordinary Annuity = 491.541.465

Sedangkan nilai Annuity Due dari pembayaran tersebut adalah Rp 547.731.132, yaitu hasil dari pembayaran selama 4 tahun dengan memperhitungkan suku bunga. Rumus yang digunakan adalah:

Annuity Due = Payment x [(1 + Interest Rate)^Number of Periods - 1]/Interest Rate

Annuity Due = 100.000.000 x [(1 + 0,12)^4 - 1]/0,12

Annuity Due = 547.731.132

   Present Value dari penerimaan dana sebesar Rp 50.000.000 setiap akhir/awal tahun selama 4 tahun dengan tingkat suku bunga 11% Per tahun adalah Rp 181.818.182. Rumus yang digunakan adalah:

Present Value = Payment x (1 - (1 + Interest Rate)^-Number of Periods)/Interest Rate

Present Value = 50.000.000 x (1 - (1 + 0,11)^-4)/0,11

Present Value = 181.818.182

Nilai Ordinary Annuity dari penerimaan dana tersebut adalah Rp 181.818.182, yaitu hasil dari penerimaan dana selama 4 tahun tanpa memperhitungkan suku bunga. Rumus yang digunakan adalah:

Ordinary Annuity = Payment x [(1 - (1 + Interest Rate)^-Number of Periods)/Interest Rate]

Ordinary Annuity = 50.000.000 x [(1 - (1 + 0,11)^-4)/0,11]

Ordinary Annuity = 181.818.182

Sedangkan nilai Annuity Due dari penerimaan dana tersebut adalah Rp 200.000.000, yaitu hasil dari penerimaan dana selama 4 tahun dengan memperhitungkan suku bunga. Rumus yang digunakan adalah:

Annuity Due = Payment x [(1 - (1 + Interest Rate)^-Number of Periods)/Interest Rate] x (1 + Interest Rate)

Annuity Due = 50.000.000 x [(1 - (1 + 0,11)^-4)/0,11] x (1 + 0,11)

Annuity Due = 200.000.000


2. 1. Berapa Future dan Present Value dari uang sebesar Rp 150.000.000 lima tahun mendatang/yang akan diterima lima tahun mendatang jika tingkat suku bunga yang berlaku 15% per tahun? 2. Soal Future Value; pembayaran Rp 100.000.000 yang dilakukan selama 4 tahun berturut-turut jika tingkat suku bunga yang berlaku 12% Per tahun. Hitunglah berapa nilai Ordinary Annuity dan Annuity Due! 3. Soal Present Value: penerimaan dana sebesar Rp 50.000.000 setiap akhir/awal tahun selama 4 tahun dengan tingkat suku bunga 11% Per tahun. Hitunglah berapa nilai Ordinary Annuity dan Annuity Due!


Untuk menjawab pertanyaan pertama, kita perlu menggunakan rumus Present Value untuk menghitung nilai sekarang dari uang yang akan diterima lima tahun kemudian. Rumusnya adalah sebagai berikut:

Present Value = Future Value / (1 + Interest Rate)^Number of Years

Jadi, dengan menggunakan nilai yang disediakan dalam pertanyaan, kita bisa menghitung Present Value sebagai berikut:

Present Value = Rp150.000.000 / (1 + 0,15)^5

Setelah melakukan perhitungan, kita mendapatkan Present Value sebesar Rp89.322.814,67.

Untuk menjawab pertanyaan kedua, kita perlu menggunakan rumus Future Value untuk menghitung nilai masa depan dari uang yang dibayarkan selama empat tahun berturut-turut. Rumusnya adalah sebagai berikut:

Future Value = Present Value x (1 + Interest Rate)^Number of Years

Jadi, dengan menggunakan nilai yang disediakan dalam pertanyaan, kita bisa menghitung Future Value sebagai berikut:

Future Value = Rp100.000.000 x (1 + 0,12)^4

Setelah melakukan perhitungan, kita mendapatkan Future Value sebesar Rp145.947.653,96.

Untuk menjawab pertanyaan ketiga, kita perlu menggunakan rumus Present Value untuk menghitung nilai sekarang dari uang yang akan diterima setiap akhir tahun selama empat tahun berturut-turut. Rumusnya adalah sebagai berikut:

Present Value = Future Value / (1 + Interest Rate)^Number of Years

Jadi, dengan menggunakan nilai yang disediakan dalam pertanyaan, kita bisa menghitung Present Value sebagai berikut:

Present Value = Rp50.000.000 / (1 + 0,11)^4

Setelah melakukan perhitungan, kita mendapatkan Present Value sebesar Rp180.527.071,24.

Rumus Simple Future Tense
1. Positif
Subject + will/shall + verb I + object
Subject + am/is/are + going to + verb I + object
Contoh :
We will go to school soon
The students are going to have a test next month
2. Negatif
Subject + will/shall + verb I + object
Subject + am/is/are + not + going to + verb I + object
Contoh :
We will not go to school soon
The students are not going to have a test next
3. Pertanyaan
Will/shall + subject + verb I + object
Am/is/are + subject + going to + verb I + object

14. Kenapa annuity due selalu memberikan nilai yang lebih besar dibandingkan dengan ordinary annuity? jika kamu mengetahui nilai dari ordinary annuity, mengapa perlu dikalikan dengan (1+i) untuk menemukan due annuity?


Jawaban:

Karena due annuity dibayar lebih banyak 1 kali dibandingkan ordinary annuity.

Penjelasan:

Jadi, kalau due annuity kan dibayarnya setiap awal periode, sedangkan ordinary annuity dibayar di akhir periode. Ibarat kamu beli motor. Kalau pakai due annuity, saat kamu menerima motornya, kamu harus bayar dulu baru bisa pakai. Sedangkan ordinary annuity itu kamu bisa pake dulu motornya 1 bulan baru dibayar di akhir bulan. Jadi kalau dibikin garis yg ada angka 0, 1, 2, 3, due annuity itu dibayar dari angka 0 yaitu saat kamu menerima manfaatnya, sedangkan ordinary annuity itu dibayar dari angka 1. Makanya due annuity itu harus pakai (1+i) karena ada penambahan 1 periode pembayaran jika dibandingkan dengan ordinary annuity.

17. pengertian future value apa?


Future value yaitu nilai uang yang akan diterima dimasa yang akan datang dari sejumlah modal yang ditanamkan sekarang dengan tingkat discount rate ( bunga ) tertentu.Nilai uang di masa yang akan datang dari uang yang diterima atau dibayarkan pada masa sekarang dengan memperhitungkan tingkat bunga setiap periode selama jangka waktu tertentu.

35. Deskripsikan apa yang dimaksud future value dan tuliskan salah satu rumusnya


Deskripsikan apa yang dimaksud future value dan tuliskan salah satu rumusnya!

Jawaban :  

Future value (nilai yang akan datang) adalah nilai uang di masa yang akan datang dari uang yang diterima atau dibayarkan pada masa sekarang dengan memperhitungkan tingkat bunga setiap periode selama jangka waktu tertentu.

Berikut adalah rumus untuk menghitung future value :

a. Jika pembayaran bunga dilakukan per tahun

FV = Po (1+i)n

b. Jika pembayaran bunga dilakukan per hari, per triwulan, per kuartal, dan per semester

FV = Po (1+(i/m))m.n

Pembahasan

Time value atau nilai waktu yang dimiliki oleh uang adalah salah satu teori dasar mengenai manajemen keuangan.

Time value ada lima hal yang perlu diperhatikan :

1. suku bunga

2. jangka waktu atau berapa kali suku bunga atau bagi hasil diberikan

3. pembayaran

4. nilai uang saat ini (present value)

5. nilai uang di masa mendatang (future value)

Future value (nilai yang akan datang) : Nilai uang di masa yang akan datang dari uang yang diterima atau dibayarkan pada masa sekarang dengan memperhitungkan tingkat bunga setiap periode selama jangka waktu tertentu.

Ada 2 macam suku bunga yang dapat digunakan untuk menghitung future value, yaitu :

1. simple interest adalah bunga yang didapatkan dari nilai investasi saat ini.

2. compound interest (bunga berbunga, bunga majemuk) adalah bunga yang didapatkan dari investasi saat ini dan dari akumulasi investasi tersebut.

Berikut adalah rumus untuk menghitung future value :

a. Jika pembayaran bunga dilakukan per tahun

FV = Po (1+i)n

b. Jika pembayaran bunga dilakukan per hari, per triwulan, per kuartal, dan per semester

FV = Po (1+(i/m))m.n

Keterangan :

FV : Nilai pada masa yang akan datang

Po : Nilai pada saat ini

i : Tingkat suku bunga

n : Jangka waktu

m : Frekuensi pembayaran bunga per tahun

Present value (nilai sekarang) : merupakan besarnya jumlah uang pada permulaan periode atas dasar tingkat tertentu dari sejumlah uang yang baru akan diterima beberapa waktu/periode yang akan datang.

Present value merujuk pada nilai yang dimiliki uang pada saat ini. Bila saat ini Anda memiliki Rp 1.000.000,00, maka nilai uang tersebut adalah sebesar Rp 1.000.000,00. Bila Anda hanya menyimpan uang tersebut di rumah, maka nilainya akan berkurang di masa mendatang karena berarti Anda menghilangkan nilai bunga yang bisa Anda dapatkan melalui investasi.

Jumlah tersebut juga akan berkurang nilainya karena inflasi—yang menaikkan harga barang. Inilah risiko menyimpan uang ketimbang menginvestasikannya.

Berikut adalah rumus untuk menghitung present value :

a. Jika pembayaran bunga dilakukan per tahun

Po = FV/(1+i)n

b. Jika pembayaran bunga dilakukan per hari, per triwulan, per kuartal, dan per semester

Po = FV/(1+(i/m))m.n

Keterangan:

FV: jumlah investasi dimasa yang akan datang

Po: jumlah investasi sekarang atau awal

i: tingkat bunga pertahun

n: jumlah tahun

m: frekuensi pembayaran bunga dalam setahun

Untuk menghitung PV dan FV bisa menggunakan fungsi pv() dan fungsi fv() yang ada dimicrosoft excel. Ada lima parameter yaitu :

1. Rate, tingkat suku bunga pada periode tertentu bisa per bulan ataupun per tahun.

2. Nper, jumlah angsuran yang dilakukan.

3. Pmt, besar angsuran yang dibayarkan.

4. Fv, nilai akan datang yang akan dihitung nilai sekarangnya.

5. Type, jika bernilai 1 pembayaran dilakukan diawal periode, jika bernilai 0 pembayaran dilakukan diakhir periode

38. Jelaskan perbedaan Present Value (PV) dengan Future Value (FV) dan berikan contoh simulasi perhitungannya!


Jawaban:

Perbedaan Present Value (PV) dengan Future Value (FV) dan contoh simulasi perhitungannya yaitu jika present value menunjukkan nilai uang saat ini dan future value menunjukkan nilai uang dimasa yang akan datang dan menggambarkan manfaat dari sebuah investasi.

Contohnya dapat dilihat pada penjelasan dibawah ini.

Penjelasan:

Present value: Anda memiliki uang saat ini Rp.2.000.000 jika anda menginvestasikannya selama 5 tahun dengan suku bunga 5% maka jumlah yang dimiliki adalah?

Present value yaitu jumlah uang saat ini sebesar Rp.2.000.000

Future value :

Future value = PV x (1+i)n

PV : Present value

i : suku bunga

n: jangka waktu/periode

Future value = PV x (1+i)n

                      = Rp.2.000.000 x (1 + 5%) 5

                      = Rp.2.000.000 x [1+(0,005 x 5)]

                      =  Rp.2.050.000

Maka besarnya future value adalah sebesar Rp.2.050.000
