24. 2. Tinjau himpunan A, B, dan C, buktikanlah hubungan berikut dengan contoh himpunan: a) An (B-C) = (AnB) - (AnC) b) A − (B − C) = (A - B) U (ANC) c) Jika An BC, maka B-CCA​


Jawaban:

Untuk membuktikan hubungan-hubungan tersebut, kita akan menggunakan contoh himpunan.

Misalkan kita memiliki himpunan-himpunan berikut:

A = {1, 2, 3, 4}

B = {3, 4, 5}

C = {4, 5, 6}

a) Kita akan membuktikan pernyataan An (B-C) = (AnB) - (AnC)

An (B-C) = An {3,4,5} - An {4,5,6}

= {3,4} - {4,5}

= {3}

AnB = {1,2,3,4} AnC = {1,2,3,4}

(AnB) - (AnC) = {1,2,3,4} - {1,2,3,4}

= {}

Dalam contoh ini, hubungan An (B-C) = (AnB) - (AnC) tidak terbukti.

b) Kita akan membuktikan pernyataan A − (B-C) = (A - B) U (ANC)

A − (B-C) = {1,2,3,4} − ({3,4,5} - {4,5,6})

= {1,2,3,4} − {3}

A-B = {1,2,3,4} - {3,4,5} = {1,2}

ANC = {1,2,3,4} ∩ {4,5,6} = {4}

(A - B) U (ANC) = {1,2} U {4} = {1,2,4}

Dalam contoh ini, hubungan A − (B-C) = (A - B) U (ANC) terbukti.

c) Kita akan membuktikan bahwa jika An BC, maka B-CCA

Dalam contoh ini, karena An BC (yaitu {1,2,3,4} ∩ {4,5,6} = {4}) terpenuhi, kita perlu membuktikan hubungan B-CCA.

B-CCA = {3,4,5} - {1,2,3,4}

= {5}

Dalam contoh ini, hubungan jika An BC, maka B-CCA terbukti.

Jadi, dalam contoh ini, hubungan a) An (B-C) = (AnB) - (AnC) tidak terbukti, hubungan b) A − (B - C) = (A - B) U (ANC) terbukti, dan hubungan c) jika An BC, maka B-CCA terbukti
