26. Tentukanlah suku umum dari barisan berikut. 6, 24, 60, 120, 210, 336, 504, ...


Jawaban:

A. Rumus suku ke - n pola bilangan balok adalah Un = n(n + 1)(n + 2).

B. Sepuluh suku pertama dari pola bilangan tersebut adalah 6, 24, 60, 120, 210, 336, 504, 720, 990 dan 1.320.

Penjelasan dengan langkah-langkah:

A. Rumus suku ke - n pola bilangan balok adalah Un = n(n + 1)(n + 2).

B. Sepuluh suku pertama dari pola bilangan tersebut.

Maka, penyelesaiannya adalah :

Un = n(n + 1)(n + 2)

U1 = 1(1 + 1)(1 + 2)

U1 = 6

U2 = 2(2 + 1)(2 + 2)

U2 = 24

U3 = 3(3 + 1)(3 + 2)

U3 = 60

U4 = 4(4 + 1)(4 + 2)

U4 = 120

U5 = 5(5 + 1)(5 + 2)

U5 = 210

U6 = 6(6 + 1)(6 + 2)

U6 = 336

U7 = 7(7 + 1)(7 + 2)

U7 = 504

U8 = 8(8 + 1)(8 + 2)

U8 = 720

U9 = 9(9 + 1)(9 + 2)

U9 = 990

U10 = 10(10 + 1)(10 + 2)

U10 = 1.320

Dengan demikian, sepuluh suku pertama dari pola bilangan tersebut adalah 6, 24, 60, 120, 210, 336, 504, 720, 990 dan 1.320.
