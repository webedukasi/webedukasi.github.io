4. Apa yg dimaksud dengan wordperfect & fungsinya ?


pengertian wordperfect           Corel WordPerfect adalah program pengolah kata serbaguna itu, sampai beberapa tahun yang lalu, adalah yang paling populer program pengolah kata di pasar. Microsoft Word saat ini memegang Nomor 1 tempat, bagaimanapun, WordPerfect memiliki basis penggemar yang besar mati-keras pengguna dan umumnya digunakan oleh firma hukum.
Sejarah
WordPerfect ini diciptakan pada tahun 1979 oleh satelit Software International dan saat ini dimiliki oleh Corel Corporation. Sekarang menjadi bagian dari WordPerfect Office X4, fungsi-lengkap bundel perangkat lunak perkantoran yang meliputi program untuk spreadsheet, presentasi, notebook digital dan database.
Kemampuan
Corel WordPerfect efisien dapat menciptakan berbagai jenis dokumen yang diperlukan untuk rumah atau kantor, termasuk surat-menyurat, laporan, tabel, bagan, selebaran, brosur, newsletter, daftar riwayat hidup dan dokumen-dokumen hukum.
Persyaratan Sistem
Komputer Anda harus dilengkapi dengan Windows Vista atau XP, Microsoft Internet Explorer 6. 0 atau yang lebih tinggi, 256 MB RAM (512 MB untuk Windows Vista), sebuah prosesor 466 MHz (800 MHz processor untuk Windows Vista), 600 MB ruang hard disk, sebuah 16-bit warna monitor dengan resolusi 800 --- 600 atau lebih dan CD-ROM.
Keunikan
Fungsi eksklusif untuk Corel WordPerfect adalah "kode mengungkapkan," yang memungkinkan pengguna untuk dengan mudah mengedit dokumen dengan menunjukkan semua behind-the-adegan kode. Baru untuk WordPerfect X4 adalah hitam keluar fitur dan hukum redacting toolbar.
Kompatibilitas
Corel WordPerfect sepenuhnya kompatibel dengan dokumen Microsoft Word sehingga konversi tidak masalah.

6. Apa yang dimaksud Corel draw?


CorelDraw adalah sebuah aplikasi grafis yang dapat digunakan untuk membuat maupun memodifikasi seni grafis berbasis vektor.

Pembahasan

CorelDraw adalah salah satu aplikasi pengolah grafis yang sangat populer selain Adobe Photoshop. Aplikasi ini merupakan perangkat lunak yang dikembangkan dan dipasarkan oleh perusahaan ternama Corel Corporation. Aplikasi ini juga dikenal dengan nama lainnya yaitu Corel Graphics Suite. Versi terakhir dari aplikasi ini diluncurkan pada akhir Maret 2019 yakni versi 21. Sebagai sebuah aplikasi pengolah grafis, aplikasi CorelDraw dikhususkan untuk seni grafis dua dimensi.

Aplikasi ini bekerja dengan berbasis vektor yaitu pengolahan berbasis garis, arah, panjang, dan ukuran untuk membuat maupun mengolah sebuah gambar.

26. 7. Fungsi menu text pada Corel Draw adalah ... . 8. Fungsi menu Help pada Corel Draw adalah ... .


Jawaban:

fungsi menu teks

Text Properties berfungsi untuk menampkan text atau menghiden text.

Tabs digunankan untuk memberikan tab pada text yang kita buat.

Columns berfungsi untuk memberikan kolom di lembar kerja CorelDraw.

Bullet digunakan sebagai pemberi petunjuk atau tanda seperti lingkaran kecil berwarna black.

Drop Cap berfungsi untuk merubah huruf awal dari huruf kecil menjadi huruf kapital.

Font Palyground berfungsi untuk menyeleksi suatu font yang akan digunakan dengan cara menyatukan atau mengumpulkan banyak font, kemudian memilih font mana yang akan digunakan.

Edit Text berfungsi untuk merubah tulisan pada text.

Insert Caracter berfungsi untuk memasukan karakter pada suatu text.

Insert Formating Code berfungsi untuk mamasukan code pada text.

Convert dapat digunakan untuk mengganti text menjadi paragraf dan mengganti text menjadi artistik.

Show Non-Printing Characters berfungsi untuk mengemukakan karakter menjadi non print.

Paragraph Taxt Frame digunakan untuk memilih kolom pada sebuah fram text.

Fit Text To Path berfungsi untuk mengatur text dalam bidang lurus dirubah menjadi melengkung dengan menggunakan bangun lingkarang atau dengan bangun lain sebagainya.

Straighten Text fungsi yang digunakan untuk mengatur text.

Align to Baseline berfungsi untuk mengatur ulang dari font menjadi bentuk aslinya.

Align to Baseline Grid berfungsi untuk mengatur ulang font pada grid di lembar kerja CoralDraw.

Use Hyphenation berfungsi untuk mengatur panjang suatu kata pada kalimat dan jarak  pada line pinggir.

Hyphenation Settings berfungsi untuk menentukan panjang suatu kata dan jarak antar garis pinggir secara tidak otomatis.

Writing Tools berfungsi untuk mengecek text yang telah dibuat pada lembar kerja CorelDraw.

Change Case untuk mengatur kata menjadi huruf kapital , kecil, huruf besar kecil, dan huruf besar pada  huruf vokal.

Make Taxt Web-compatible berfungsi untuk membuat text mejadi dapat bersaing  pada Web.

Encode Berfungsi untuk membuat kata sandi pada objek.

What the Font berfungsi untuk mencari font yang langsung terhubung dengan internet pada web CorelDraw.

Text Statistics berfungsi untuk melihat text yang dipilih.

Font List Option berfungsi sebagai penyusun urutan font secara  tidak otomatis.

fungsi menu help:

Poduct Help digunakan sebagai petunjuk penggunaan Aplikasi CorelDraw.

Welcome Screen berfungsi untuk menampilkan awal lembar kerja sebelum membuka lembar kerja yang baru, di sana terdapat beberapa daftar nama file yang pernah kita buat.

Video Tutorials berfungi untuk menampilkan Video Tutorial sebagai media pemblajaran ,bagimana cara penggunakan tool serta teknik mendesign.

Hints berfungsi untuk memunculkan fungsi dan kegunakan dari masing-masing tool.

Quick Start Guide berfungsi untuk menampilkan langkah penggunaan secara cepat.

Insights Form the Eperts berfungsi sebagai media pembelajaran berformat PDF.

Whats' new berfungsi sebagai pengirim pemberitahuan yang berkaitan dengan terupdate pada CorelDraw.

Highlight Whats New fungsinya hampir sama dengan fungsi pada what's new, serta dapat memberi informasi pada versi yang lama.  

Updates berfungsi sebagai media untuk mengupdate ke versi yang lebih terbaru dari versi yang sebelumnya.

CorelDraw.com berfungsi untuk bertemu dengan  komunitas CorelDraw lainya yang disediakan pada link yang ada pada CorelDraw.

Corel Support diguankan untuk memberikan informasi menyeluruh tentang CorelDraw.

About CorelDraw MemberShip berfungsi langkah menjadi member secara gratis atau berbayar.  

Account Settings berfungsi sebagai tempat untuk masuk ke account CorelDraw atau membuat Account pada CorelDraw.

Sing In fungsinya sama dengan account setting.

About CorelDraw digunakan untuk mengetahui tentang seputar CorelDraw.
