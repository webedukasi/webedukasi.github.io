Keliling lingkaran tersebut adalah 220 cm.

Rumus:

Luas lingkaran = π x r²

Keliling Lingkaran = 2 x π x r

dengan

r = jari-jari

π = ²²/₇ atau π = 3,14

Penjelasan dengan langkah-langkah:

Diketahui:

Luas lingkaran = 3.850 cm²

π = ²²/₇

Ditanya:

Keliling lingkaran tersebut adalah

Jawab:

Luas lingkaran = π x r²

3.850 = ²²/₇ x r²

r² =  [tex]\frac{3.850\times 7}{22}[/tex]

   =  175 x 7

   = 1.225

r = √1225

 = 35 cm

Keliling Lingkaran = 2 x π x r

= 2 x ²²/₇ x 35

= 2 x 22 x 5

= 220

Jadi, Keliling lingkaran tersebut adalah 220 cm.

23. QUIZ (1552)Materi: KesebangunanPerhatikan gambar berikut!Panjang TR adalah ...A. 2 cmB. 3 cmC. 4 cmD. 6 cm(Contoh Soal Ujian SMP)​​​​​


Diketahui :

PT = 12 cm

PQ = 9 + 11 = 20 cm

PR = 12 + x

TR = x = ? cm

Penyelesaian :

→ [tex] \tt \frac{tp}{pq} = \frac{ps}{PR} [/tex]

→ [tex] \tt \frac{12}{20} = \frac{9}{12 + x} [/tex]

→ 20(9) = 12(12 + x)

→ 180 = 144 + 12x

→ -12x = 144 - 180

→ -12x = -36 cm

→ x = 3 cm

Kesimpulan :

Maka, panjang TR adalah B.) 3 cm.

29. Contoh soal kedua Sebuah drum berbentuk tabung memiliki diamter 70 cm dan tinggi 80 cm. Tentukan luas permukaan drum tersebut


Jawab:

jadi luas permukaan drum tersebut adalah 25.300 cm²

Penjelasan dengan langkah-langkah:

Sebuah drum berbentuk tabung memiliki diameter 70 cm dan tinggi 80 cm. Tentukan luas permukaan drum tersebut

_____

dik.

d = 70 cm

t = 80 cm

dit.

luas permukaan tabung = ?

jawab :

rumus luas permukaan tabung

→ tabung dengan tutup :

2 x [tex]\pi[/tex] x r x r + 2 x [tex]\pi[/tex] x r x  t 2 x [tex]\pi[/tex] x r x (r + t)

→ tabung tanpa tutup  

[tex]\pi[/tex] x r x r + 2 x [tex]\pi[/tex] x r x  t

r = d/2 = 70/2 = 35 cm

untuk soal ini bisa menggunakan 2 cara yaitu

cara pertama

L = 2 x [tex]\pi[/tex] x r x r + 2 x [tex]\pi[/tex] x r x  t

L = 2 x 22/7 x 35 x 35 + 2 x 22/7 x 35 x 80

L = 2 x 22 x 5 x 35 + 2 x 22 x 5 x 80

L = 44 x 5 x 35 + 22 x 2 x 5 x 80

L = 220 x 35 + 2 x 22 x 5 x 80

L = 7.700 + 2 x 22 x 5 x 80

L = 7.700 + 44 x 5 x 80

L = 7.700 + 220 x 80

L = 7.700 + 17.600

L = 25.300 cm

cara kedua

2 x [tex]\pi[/tex] x r x (r + t)

2 x 22/7 x 35 x (35 + 80)

2 x 22 x 5 x 115

44 x 5 x 115

220 x 115

25.300

kesimpulan :

cara yang pertama cenderung lebih panjang dari cara yang kedua, lagi pula jika melihat rumus keduanya sudah dapat dilihat bahwa cara yang pertama jauh lebih panjang

jadi luas permukaan drum tersebut adalah 25.300 cm²
