= 12 + 72

= 84.

18. ksih tau yaaa bgi yg tau uuuuu​


6.500×50=325.000+30.000×75=2.575.000

Jawaban:

Rp. 2.575.000,-

Penjelasan dengan langkah-langkah:

Dik : 1 kg = Rp. 6.500,- untuk 50 bungkus.

5 kg = Rp. 30.000,- untuk 75 bungkus.

Dit : Berapa jumlah uang yang akan diperoleh jika paket terjual semua?

Jwb :

a. Untuk paket 50 bungkus.

50 bungkus × Rp. 6.500,- = Rp. 325.000,-

b. Untuk paket 75 bungkus.

75 bungkus × Rp. 30.000,- = Rp. 2.250.000,-

Jadi, untuk paket 50 bungkus = Rp. 325.000,- + untuk paket 75 bungkus = Rp. 2.250.000,- =

Rp. 2.575.000,- seluruhnya.
