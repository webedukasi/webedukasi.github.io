3. Sebuah perusahaan membutuhkan beberapa karyawan baru. Tes seleksi karyawan diadakan dan dari seluruh peserta tes hanya 20% yang lulus. Jika dari parabpesertabtes tersebut diambil sampel secara acak sebanyak 10 peserta, peluang paling banyak terdapat 3 peserta lulus tes adalah


Sebuah perusahaan membutuhkan beberapa karyawan baru. Tes seleksi karyawan diadakan dan dari seluruh peserta tes hanya 20% yang lulus. Jika dari parabpesertabtes tersebut diambil sampel secara acak sebanyak 10 peserta, peluang paling banyak terdapat 3 peserta lulus tes adalah 0,879 atau 87,9%

PEMBAHASAN

Peluang suatu kejadian di definisikan sebagai kemungkinan terjadinya suatu kejadian terhadap segala kemungkinan yang ada ( ruang sampel ).

[tex]\textbf{Peluang = banyak kejadian yang di maksud / ruang sampel}[/tex]

Distribusi Probabilitas Binomial :

P(X=a) = ⁿCₐ pᵃ ( 1-p ) ⁿ⁻ᵃ

dengan :

p = kemungkinan suatu kejadian sukses

Okay lah dengan menggunakan prinsip ini kita akan menyelesaikan soal yang ditanyakan.

Diketahui dari soal :

p = 20/100 = 1/5

n = 10

Ditanyakan :

P ( X ≤ 3 )

Jawab:

P ( X =0 ) = ¹⁰C₀ (1/5)⁰ ( 1 - 1/5 ) ¹⁰⁻⁰ = ( 4/5 )¹⁰

P ( X = 1 ) = ¹⁰C₁ (1/5)¹ ( 1 - 1/5 ) ¹⁰⁻¹ = 10 ( 1/5 ) ( 4/5 )⁹ = 2 ( 4/5 )⁹

P ( X =2 ) = ¹⁰C₂ (1/5)² ( 1 - 1/5 ) ¹⁰⁻² = 45 ( 1/25 ) ( 4/5 )⁸ = ( 9/5 ) ( 4/5 )⁸

P ( X = 3 ) = ¹⁰C₃ (1/5)³ ( 1 - 1/5 ) ¹⁰⁻³ = 120 ( 1/125 ) ( 4/5 )⁷ = (24/25) ( 4/5 )⁷

P ( X ≤ 3 ) =    P ( X =0 ) + P ( X = 1 ) + P ( X = 2 ) + P ( X = 3 )

P ( X ≤ 3 ) =    ( 4/5 )¹⁰ + 2 ( 4/5 )⁹ + ( 9/5 ) ( 4/5 )⁸ +  (24/25) ( 4/5 )⁷

P ( X ≤ 3 ) =    0,879 = 87,9%

8. Sebuah perusahaan membutuhkan beberapa karyawan baru melalui tes seleksi karyawan. dari seluruh peserta tes hanya 30% yang lolos. dari para peserta tersebut diambil sampel secara acak sebanyak 10 peserta. peluang peserta lolos sebanyak 4 orang adalahâ¦. *


Peluang peserta lolos sebanyak 4 orang adalah 0,2001

Penjelasan:

Masalah yang diberikan di atas merupakan masalah distribusi binomial. Dalam hal ini, masalah di atas hanya memiliki 2 kemungkinan yaitu mendapat peserta lolos atau tidak.

Misalkan:

p = peluang lolos

q = peluang gagal = 1 - p

n = banyaknya percobaan

x = banyaknya percobaan berhasil

Diketahui:

p = 30% = 0,3

q = 1 - p = 0,7

n = 10

x = 4

Sehingga dapat ditentukan peluang peserta lolos sebanyak 4 orang sebagai berikut.

P = C(n,x) · pˣ · qⁿ⁻ˣ

  = C(10,4) · (0,3)⁴ · (0,7)⁶

  = [tex]\frac{10!}{6!4!}[/tex] · (0,3)⁴ · (0,7)⁶

  = 210 · 0,0081 ·  0,1176

  ≈ 0,2001

Jadi, peluang peserta lolos sebanyak 4 orang adalah 0,200.

10. Terdapat 60 orang pelamar yang harus mengikuti tes tertulis dan tes wawancara agar dapat diterima sebagai karyawan di Universitas Terbuka. Ternyata 32 orang karyawan lulus tes wawancara, 48 orang lulus tes tertulis, dan 6 orang tidak mengikuti tes tersebut. Banyak pelamar yang diterima sebagai karyawan perusahaan adalah?


Banyak pelamar yang diterima sebagai karyawan perusahaan adalah 26 orang.

Pendahuluan

Himpunan adalah kumpulan benda atau objek yang dapat didefinisikan dengan jelas.

Pembahasan

Diketahui :

Banyak pelamar n (S) = 60 orangLulus tes wawancara n(A) = 32 orangLulus tes tertulis n(B) = 48 orangTidak mengikuti tes n (A U B)⁽ = 6 orang

Ditanya :

Banyak pelamar yang diterima ?

Penyelesaian :

Menentukan banyak pelamar yang diterima

n (A ∩ B) = x

n (S) = n (A) - n(A∩B) + n (B) - n(A∩B) + n(A∩B) + n (A ∪ B)⁽

60 = 32 - x + 48 - x + x + 6

60 = 86 - x

x = 86 - 60

x = 26

Jadi banyak pelamar yang diterima sebagai karyawan perusahaan adalah 26 orang.

Banyak karyawan baru yang diterima adalah sebanyak 10 orang.

Penjelasan dengan langkah-langkah:

Diketahui : Rata-rata kinerja dari 5 karyawan lama sebesar 60. untuk menaikkan rata-rata kinerja menjadi 68, perusahaan menerima karyawan baru dan hasil tes kinerja diperoleh rata-rata sebesar 72.

x1 = 60

n1 = 5

x2 = 68

x gab = 72

Ditanya : Berapa banyak karyawan baru yang diterima?

Jawab :

[tex]X gab = \frac{ x_{1}. n_{1}+ x_{2}. n_{2} +... + x_{n}.n_{n} }{n_{1} +n_{1}}[/tex]

[tex]72 = \frac{5.60 + n_{2}. 68}{5+n_{2}}[/tex]

[tex]72 = \frac{300 + n_{2}. 68}{5+n_{2}}[/tex]

72 (5 + n2) = 300 + 68n2

360 + 72 n2 = 300 + 68 n2

360 - 300 = 68 n2 - 72 n2

60 = - 4 n2

n2 = 15

n = 15 -5

n = 10

Jadi, tambahan karyawan barunya yang diterima adalah 10 orang karyawan.
