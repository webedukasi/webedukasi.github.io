14. Tentukan nilai diskriminan pada soal no.1


Tentukan nilai diskriminan pada soal no.1

a.  3x² - 12 = 0b.  x² + 7x + 6 = 0c.  -3x² - 5x + 2 = 0

Persamaan kuadrat adalah persamaan dengan variabel berpangkat maksimal 2 dengan bentuk umum persamaan kuadrat ax² + bx + c = 0.

Jenis akar-akar persamaan kuadrat ax² + bx + c = 0 ditentukan oleh diskriminan

[tex]\boxed {\boxed {D = b^{2} - 4ac}}[/tex]

Pembahasan

Nilai Diskriminan

a.  3x² - 12 = 0

a = 3

b = 0

c = -12

D = b² - 4ac

   = 0² - 4 (3) (-12)

   = 0 - (12) (-12)

   = 144

b.  x² + 7x + 6 = 0

a = 1

b = 7

c = 6

D = b² - 4ac

   = 7² - 4 (1) (6)

   = 49 - 24

   = 25

c.  -3x² - 5x + 2 = 0

a = -3

b = -5

c = 2

D = b² - 4ac

   = (-5)² - 4 (-3) (2)

   = 25 + 24

   = 49

~Persamaan Kuadrat

_____________________

Diskriminan dari persamaan kuadrat x² + 5x + 6 = 0 adalah 1

» Pembahasan

Bentuk umum persamaan kuadrat adalah ax² + bx + c = 0

dimana:

a = koefisien x²b = koefisien xc = konstantax = variabel

Untuk menentukan akar-akar pada persamaan kuadrat, dapat menggunakan beberapa metode, diantaranya:

PemfaktoranMelengkapkan kuadratMenggunakan rumus ABC atau Kuadratis

---

DiketahuiBentuk persamaan kuadrat → x² + 5x + 6 = 0

Ditanya

Nilai diskriminan ( D )

Solusi

x² + 5x + 6 = 0

1x² + 5x + 6 = 0

didapat:

a = 1b = 5c = 6

Untuk menentukan diskriminan persamaan kuadrat gunakan rumus:

D = b² - 4ac

Maka:

D = b² - 4ac

D = 5² - 4(1)(6)

D = 25 - 24

D = 1

Kesimpulan

Jadi, Diskriminan dari persamaan kuadrat x² + 5x + 6 = 0 adalah 1

34. jelaskan konsep diskriminan dan berikan contohnya​


Jawaban:

Konsep diskriminan bisa kita lihat pada saat mencari kedudukan garis terhadap persamaan lingkaran, dimana kita harus menentukan apakah kedudukannya menyinggung, memotong lingkaran di dua titik ataupun tidak memotong lingkaran. Diskriminan dilambangkan huruf D.

D = b² - 4ac

Contoh soal:

Tentukan kedudukan garis y = x - 2 terhadap lingkaran (x² - 3)² + (y + 4)² = 9!

Pembahasan:

→ Substitusi garis y = X - 2 ke persamaan lingkaran.

» (x² - 3)² + ((x - 2) + 4)² = 9

» (x² - 3)² + (x + 2)² = 9

» x² - 6x + 9 + x² + 4x + 4 = 9

» 2x² - 2x + 4 = 0

» x² - x + 2 = 0 sehingga didapat komponen diakriminan, yaitu:

a = 1, b = -1, dan c = 2

D = b² - 4ac → D = (-1)² - 4(1)(2) = 1 - 8 = -7

Untuk D < 0 berarti tidak memotong lingkaran

Untuk D = 0 berarti menyinggung lingkaran

Untuk D > 0 berarti memotong lingkaran di dua titik

Karena nilai D = -7 artinya -7 kurang dari 0. Berarti kedudukan lingkaran nya ialah tidak memotong lingkaran.
