30. selain menduplikasi slide,ada juga menyalin slide apa perbedaan menduplikasi slide?​


Tidak ada perbedaan dalam fungsinya. Tujuan akhirnya sama-sama untuk menduplikasi/menciptakan salinan.

Perbedaannya, fitur duplicate adalah operasi tunggal. Ketika anda klik duplicate, maka slide otomatis terduplikat di dalam file tersebut juga.

Kalau fitur salin, sesuai namanya anda hanya baru menyalin. Anda masih perlu menempel (paste) untuk menghasilkan salinan/duplikasi dari slide tersebut. Anda bisa menempel (paste) slide tersebut pada file yang sama ataupun file yang berbeda.

Jadi, menurut saya tidak ada perbedaan. Hanya tergantung situasi. Jika anda ingin menduplikasi dalam file yang sama, tinggal duplicate saja agar lebih mudah. Jika anda ingin menyalin slide tersebut ke file yang berbeda maka anda harus mengcopy terlebih dahulu.

Semoga membantu

32. untuk menambah slide dengan bentuk desain yang sama klik kanan pada mouse...a. duplicate slideb. publish slide c. delete slide d. new slide​


Jawaban singkat:

a. duplicate slide

Penjelasan tambahan:

Misalkan pada Ms. Powerpoint 2013, terdapat aneka pilihan yg tampil saat melakukan klik kanan pada slide. Antara lain:

→ duplicate slide: untuk menambah slide dg bentuk desain yg sama

→ publish slide: untuk menampilkan kotak dialog Publish slides

→ delete slide: untuk menghapus slide

→ new slide​: untuk menambah slide baru dg bentuk desain berbeda

Amatilah contoh tampilannya pada lampiran:

→ duplicate slide ditunjukkan tanda panah hijau

→ publish slide ditunjukkan tanda panah hitam

→ delete slide ditunjukkan tanda panah biru

→ new slide​ ditunjukkan tanda panah merah

→ kotak dialog Publish Slides​ ditunjukkan tanda panah ungu
