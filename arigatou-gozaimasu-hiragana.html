2. Tulisan Jepang untuk "Felix", bisa dalam Katakana, Hiragana, maupun Kanji, terserahArigatou Gozaimasu


Penulisan nama "Felix" dalam bahasa Jepang adalah フェリックス (Ferikkusu).

Pembahasan:

Nama フェリックス (Ferikkusu) = Felik dalam bahasa Jepang, tersusun dari beberapa huruf antara lain:

フェ (U + e *kecil*) = feリ = riック = kkuス = su

Sistem penulisan bahasa Jepang ada tiga macam, yaitu Hiragana, Katakana, dan Kanji. Hiragana adalah huruf-huruf dalam bahasa Jepang yang lebih umum digunakan, untuk menulis kosakata asli pada umumnya, menulis partikel, dsb. Katakana adalah huruf-huruf dalam bahasa Jepang yang penggunaannya lebih sempit daripada Hiragana, hanya dipakai untuk menuliskan nama orang atau marga luar Jepang, kota luar Jepang, negara, kosakata pinjaman, penulisan onomatope, istilah-istilah ilmiah, dsb. Kanji adalah sistem penulisan bahasa Jepang yang berasal dari penyesuaian abjad Hanzi dari Tionghoa, namun jumlah Kanji sudah dibatasi. Jadi, tidak semua Hanzi juga merupakan abjad Kanji. Pembacaan Kanji, ada dua macam yaitu dengan menggunakan cara baca Jepang dan cara baca China. Kanji berjumlah ribuan.

Pendahuluan :

Sistem Penulisan Bahasa Jepang

Huruf yang digunakan di Jepang ada 3 jenis yaitu Hiragana, Katakana, dan Kanji. Saat menulis kalimat dalam Bahasa Jepang, huruf yang digunakan biasanya campuran dari huruf Kanji dan Hiragana. Bila ada kata-kata serapan, ditulis dalam huruf Katakana. Kalau Romaji itu hanya digunakan orang luar Jepang yang lebih terbiasa membaca huruf abjad (abcd), agar mudah membaca huruf Jepang.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Huruf yang saya bahas kali ini hanya huruf Kana saja, yang meliputi huruf Hiragana dan huruf Katakana.

HIRAGANA「平仮名・ひらがな」

Hiragana merupakan salah satu huruf Jepang yang memiliki beberapa kegunaan antara lain seperti untuk menulis kata-kata asli Jepang yang Kanjinya terlalu sulit atau tidak ada, untuk menulis partikel, untuk menulis Furigana (huruf untuk memudahkan cara baca Kanji), untuk menulis Okurigana (huruf yang mengakari Kanji), dan lain sebagainya. Huruf ini memiliki ciri yaitu garis-garis hurufnya yang kebanyakan melengkung dan lebih elegan.

KATAKANA「片仮名・カタカナ」

Katakana merupakan salah satu huruf Jepang yang memiliki beberapa kegunaan antara lain seperti untuk menulis kata-kata dari luar Jepang, untuk menulis kata-kata ilmiah, untuk memberi penekanan pada kata, untuk menulis onomatope, dan lain sebagainya. Huruf ini memiliki ciri yaitu garis-garis hurufnya yang kebanyakan lurus dan lebih kaku.

- - - - - - -

Hiragana dan Katakana memiliki banyak kesamaan, contohnya seperti masing-masing huruf nya hanya memiliki satu bunyi saja dan tidak memiliki arti, berbeda dengan Kanji yang berkebalikannya. Lalu jenis hurufnya juga sama-sama dibagi menjadi beberapa jenis yaitu Seion, Dakuon, Handakuon, dan Youon. Daftar hurufnya sebagai berikut.

1. SEION「清音・せいおん」

Jenis huruf ini biasa disebut juga sebagai huruf dasar, karena memang merupakan kumpulan huruf-huruf yang menjadi dasarnya. Hurufnya terdiri dari 48 huruf, antara lain sebagai berikut.

Hiragana :

Katakana :

2. DAKUON「濁音・だくおん」

Jenis huruf ini merupakan kumpulan huruf dasar deret [K-S-T-H] yang diberikan tanda Tenten (seperti tanda kutip), agar bunyi konsonannya berubah menjadi [G-Z-D-B]. Hurufnya terdiri dari 20 huruf, antara lain sebagai berikut.

Hiragana :

が ぎ ぐ げ ご (ga - gi - gu - ge - go)ざ じ ず ぜ ぞ (za - ji - zu - ze - zo)だ ぢ づ で ど (da - ji - dzu - de - do)ば び ぶ べ ぼ (ba - bi - bu - be - bo)

Katakana :

3. HANDAKUON「半濁音・はんだくおん」

Jenis huruf ini merupakan kumpulan huruf dasar deret [H] yang diberikan tanda Maru (lingkaran kecil), agar bunyi konsonannya berubah menjadi [P]. Hurufnya terdiri dari 5 huruf, antara lain sebagai berikut.

Hiragana :

ぱ ぴ ぷ ぺ ぽ (pa - pi - pu - pe - po)

Katakana :

パ ピ プ ペ ポ (pa - pi - pu - pe - po)

4. YOUON「拗音・ようおん」

Jenis huruf ini biasa disebut juga sebagai huruf campuran, karena huruf-hurufnya merupakan hasil campuran dari huruf dasar berbunyi i dengan huruf [Ya-Yu-Yo] berukuran kecil. Hurufnya terdiri dari 36 huruf, antara lain sebagai berikut.

Katakana :

Sebelum membahas mengenai perbedaannya, perlu kalian ketahui dahulu mengenai persamaan kedua kalimatnya.

Persamaannya

Untuk letak kesamaannya adalah bahwa kedua kalimat arigatou gozaimasu dan arigatou gozaimashlta adalah terima kasih, dan juga untuk pemakaiannya termasuk formal dikarenakan terdapat akhiran -gozaimasu/gozaimashlta sebagai bentuk penghalus atau sopan.

Lanjut mengenai perbedaannya di bagian pembahasan ya.

----------------

Pembahasan

Perbedaannya

Arigatou gozaimasu「ありがとうございます」

Pemakaiannya ini termasuk bentuk yang umum, terdapatnya bentuk akhiran ます (masu) sehingga membuat kalimat atau suatu frasa terdengar memiliki kesan yang halus sekaligus sopan. Biasanya penggunaan kata ありがとうございます (arigatou gozaimasu) pada ketika waktu kini, sekarang, atau sedang terjadi (present).

Arigatou gozaimashlta「ありがとうございました」

Adanya akhiran ました (mashlta) yang merupakan bentuk kata lampau dari ます (masu) sehingga untuk fungsinya tetap sama, namun bentuk kata ました (mashlta) ini hanya untuk digunakan dalam menyatakan waktu yang sudah lewat atau lampau. Sehingga kalimat ありがとうございました (arigatou gozaimashlta) dipakai dalam waktu past (lalu/lampau).

39. huruf hiragananya gozaimasu


Huruf hiragana "gozaimasu" adalah ござちいます

Pembahasan

Gozaimasu「ございます」

Merupakan sejenis kata bantu sehingga terkadang tidak memiliki arti, lantas apa fungsi dari kata ござちいます？

Fungsi kata gozaimasu「ございます」:

╰➝ Untuk memberikan kesan terhadap suatu kata agar menjadi kata yang formal dan terdengar sopan dan halus.

Contoh :

Arigatou gozaimasu「ありがとうございます」

╰➝ Dalam bunpou, penggunaan kata ini berfungsi sebagai pengganti dari kata です (desu) menjadi でございます (de gozaimasu) yang jauh penggunaannya lebih sopan dan formal.

Watashi wa ginkouin de gozaimasu「私は銀行員でございます」: saya seorang pegawai bank.
