Cara Menentukan Sumbu Simetri dan Titik Puncak dari Grafik Fungsi Kuadrat (Parabola)

Bentuk umum fungsi kuadrat adalah y = ax2 + bx + c, dengan x adalah variabel dan a, b, c adalah bilangan dan a tidak sama dengan 0. Pada umumnya grafik kuadrat berbentuk parabola. Pada grafik parabola dapat dilihat titik puncak dan sumbu simetri. Bagaimana cara menentukan sumbu simetri dan titik puncak grafik fungsi Kuadrat?

Marilah kita bahas di sini.

Untuk melihat grafik fugsi kuadrat, simaklah gambar berikut.

Tampak bahwa grafik fungsi kuadrat berbentuk parabola.

Parabola di atas memiliki titik puncak atau dinamakan titik ekstrim.

Coba perhatikan:

Pada Grafik : y = x2 + 2x – 1 memiliki titik puncak (-1, -2) dan sumbu simetri x = -1.

Pada Grafik : y = x2 - 4x – 2memiliki titik puncak (2, -2) dan sumbu simetri x = 2.

32. sumbu simetri dari fungsi kuadrat f(X)=x² -7​


Sumbu simetri dari fungsi kuadrat f(x) = x² - 7 adalah x = 0.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

PENDAHULUANFungsi Kuadrat

Fungsi kuadrat adalah fungsi yang memiliki variabel berpangkat dua.

Bentuk umum fungsi kuadrat adalah

[tex]f(x)=ax^{2}+bx+c[/tex]

dengan

a = koefisien x²

b = koefisien x

c = konstanta

Grafik Fungsi Kuadrat

Grafik fungsi kuadrat f(x) berupa kurva lengkung (parabola) yang menghadap ke atas atau ke bawah.

Grafik fungsi kuadrat memiliki sumbu simetri yang dirumuskan sebagai berikut:

[tex]\boxed{x=\frac{-b}{2a}}[/tex]

PEMBAHASAN

Diketahui:

f(x) = x² - 7

Ditanya:

sumbu simetri

Jawab:

Dari f(x) = x² - 7 kita peroleh a = 1, b = 0, dan c = -7.

Selanjutnya, kita substitusi nilai a dan b ke dalam rumus sumbu simetri.

[tex]x=\frac{-b}{2a}[/tex]

[tex]x=\frac{0}{2\times1}[/tex]

[tex]x = 0[/tex]

Kesimpulan:

Jadi, sumbu simetri dari fungsi kuadrat f(x) = x² - 7 adalah x = 0.
