Penjelasan:

nah kan sudah diketahui untuk mencari ∆H°C atau reaksi pembakarannya dapat menggunakan rumus

∆Hr = ∆Hproduk - ∆Hreaktan

∆Hr maksudnya perubahan entalpi saat reaksi. entah itu pembakaran, pembentukan, penguraian atau sebagainya.

reaksi pembakarannya sebagi berikut

C(s) + O2(g) >>> CO2(g)

∆Hr = (-393,509) - (716,682 + 0)

∆Hr = -1110,191 Kj/mol

jadi ∆H°C = -1110,191 Kj/mol

27. Latihan Soal KIMIA Bab : Stokiometri Kelas : XI SMA


Fe : O = 70 / 56 : 30 / 16 = 1.25 : 1.875 = 2 : 3
Oksida besi itu adalah Fe2O3.

4 Fe(s) + 3 O2(g) --> 2 Fe2O3(s)
mol Fe = 42 / 56 = 0.75 mol
mol O2 = 42 / 32 = 1.3125 mol

Tampak bahwa Fe adalah pereaksi pembatas.

Perbandingan koefisien = perbandingan mol, maka:

mol Fe2O3 = 2/4 × 0.75 = 0.375 mol
Mr Fe2O3 = 2 × 56 + 3 × 16 = 160 g/mol
Massa Fe2O3 = 0.375 × 160 = 60 gram

34. tolong bantu jawab soal kimia kelas xi


a. Persamaan reaksi pembakaran Metanol

   CH3OH (l)   +  2 O2(g)  ---->  CO2(g)   + 2 H2O( l)

b. ΔH reaksi = ∑ Entalpi produk - ∑ Entalpi pereaksi

   =  (ΔH CO2 + 2 ΔH H2O ) -  ( ΔH CH3OH + 2 ΔH O2  )  

   =  ( - 393,5 + 2 x - 286 ) kj/mol - ( - 238,6 + 0 ) kj/mol

   =  (( - 393,5 - 572 )  + 238,6) kj/mol

   =   - 965,5 + 238,6 kj/mol

   = - 726,9 kj/mol

c. Kalor yang dibebaskan 10 gram metanol

  Mr CH3OH = 32

  mol = gram metanol / Mr metanol = 10 gram / 32 = 0,3125 mol

  maka kalor yang dibebaskan = 0,3125 mol x 726,9 kj/mol = 227,16 kj.

35. Kimia kelas XI- Larutan penyangga.


Kode : 11.7.7 [Kelas 11 Kimia Bab 7 - Buffer]

Larutan penyangga dengan pH = 10 bersifat basa.

pOH = 14 - pOH
pOH = 14 - 10
pOH = 4

[tex][OH]^-=Kb \times \frac{mol \ LOH}{mol \ L_2SO_4 \times valensi}} [/tex]

[tex]10^{-4}=10^{-5} \times \frac{(100).M}{(100)(0,05) \times 2} [/tex]

[tex]10= \times \frac{M}{(0,05) \times 2} [/tex]

M = 10 x 0,05 x 2

Diperoleh konsentrasi LOH sebesar 1 M
