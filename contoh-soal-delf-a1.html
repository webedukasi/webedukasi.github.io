Jika a1 a2 a3 merupakan barisan geometri dimana a3>a2>a1 terdapat hubungan B. a2^2= a1 a3

PEMBAHASAN :

Barisan Geometri adalah barisan yang memiliki rasio atau perbandingan yang konstan antar suku yang berturutan. Rumus yang bisa di pakai adalah sebagai berikut:

[tex]\mathbf{Un = a(r)^{n-1}}[/tex]

dimana :

Un = Rumus Suku ke - n

a = Suku Pertama

r = Rasio

Okay marilah kita menggunakan rumus di atas untuk mengerjakan soalnya.

Jika a₁ , a₂ , a₃ merupakan barisan geometri maka dapat di tuliskan :

r = a₂ / a₁ = a₃ / a₂  

a₂ / a₁ = a₃ / a₂

a₂ x a₂ = a₃ x  a₁

(a₂)² = a₃ x a₁

Jadi kesimpulannya :

B. (a₂)² = a₁ . a₃

17. jika a1,a2,a3 adalah barisan aritmatika dan a1,a2,a1+a3 adalah barisan geometri, maka a3/a1 = tolong dibantu yaa :(


*Cara 1 (dengan rumus)

ciri2 deret aritmatika
2a₂ = a₁ + a₃
a₂ = (a₁ + a₃) / 2 ...................(1)

ciri2 deret geometri
(a₂)² = a₁ × (a₁ + a₃)

[(a₁ + a₃) / 2]² = a₁ × (a₁ + a₃)     ⇒ masukkan persamaan (1)
  (a₁ + a₃)² / 4 = a₁ × (a₁ + a₃)
          a₁ + a₃ = 4a₁
                 a₃ = 3a₁
           a₃ / a₁ = 3


*Cara 2 (dengan logika)
jika a₁ = 1, a₂ = 2, a₃ = 3

pada deret aritmatika 1, 2, 3     beda = 1
pada deret geometri 1, 2, 1+3   rasio = 2 ..............Terbukti

jadi a₃ / a₁ = 3
