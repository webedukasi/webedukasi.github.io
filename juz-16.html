Jika jangkauan jumlah juz yang telah di hafal tersebut adalah 16 , maka rata-rata jumlah juz yang telah dihafal oleh putra-putra ibu Ida adalah 11

PEMBAHASAN

Berikut beberapa rumus penting dalam bab Statistika adalah :

Jika diketahui data [tex]x_1,x_2,x_3,...,x_n[/tex] , maka :

[tex]Rata-Rata=\overline{x}=\frac{x_1+x_2+x_3+...+x_n}{n}[/tex]

[tex]Simpangan~Rata-Rata=\frac{|x_1-\overline{x}|+|x_2-\overline{x}|+|x_3-\overline{x}|+...+|x_n-\overline{x}|}{n}[/tex]

[tex]Varians=\frac{|x_1-\overline{x}|^2+|x_2-\overline{x}|^2+|x_3-\overline{x}|^2+...+|x_n-\overline{x}|^2}{n}[/tex]

Mari kita mencoba menyelesaikan soalnya.

Diketahui data - data terurut sebagai berikut ini :

a , b , c , d

Median = ( b + c ) / 2

b = ( 1/2 ) Median

b = ( 1/2 ) ( b + c ) / 2

4b = b + c

3b = c → Persamaan 1

a = ( 1/6 ) Median

a = ( 1/6 ) ( b + c ) / 2

b + c = 12a

b + 3b = 12a ← Persamaan 1

4b = 12a

b = 3a

a = ( 1/3 ) b → Persamaan 2

Jangkauan = 16

d - a = 16

c - a = 16

3b - ( 1/3 ) b = 16 ← Persamaan 1 dan Persamaan 2

( 8/3 ) b = 16

b = 16 : ( 8/3 )

b = 6

a = ( 1/3 ) b = ( 1/3 ) 6 = 2

c = 3b = 3 (6) = 18

d = c = 18

Kesimpulan :

Urutan datanya adalah 2 , 6 , 18 , 18

Rata - rata = ( 2 + 6 + 18 + 18 ) / 4 = 44 / 4 = 11

40. 63. Atho'illah, Hilmy, Hakim, dan Choirul adalahputra-putra ibu Ida dan mereka memilih untukbelajar di pondok Tahfidz, Atho'illah telah hafal ajuz, Hakim telah hafal b juz, Choirul telah hafal cJuz dan Hilmy telah hafal d Juz. Jika jangkauanjumlah juz yang telah dihafal tersebut adalah 16,a = median, b = median, dan c = d, makarata-rata jumlah juz yang telah dihafal oleh putra-putra ibu Ida adalah ....(A) 10(B) 11(C) 12(D) 13​


Rata-rata jumlah juz yang telah dihafal oleh putra-putra ibu Ida adalah 11 (B). Untuk mendapatkan hasilnya, mari kita simak dalam pembahasan.

Pembahasan

Dalam statistika, kita mengenal istilah median dan mean. Median adalah nilai tengah dari data yang terurut. Sedangkan, rata-rata adalah jumlah nilai dibagi banyak data. Rata-rata kalau dirumuskan menjadi

Rata-rata = Jumlah nilai/banyak data

Sebelum masuk ke dalam penyelesaian, berikut ada koreksi soal sedikit. Mungkin maksudnya a = 1/6 median, dan b = 1/2 median.

Kini, kita masuk dalam penyelesaian soal.

Penyelesaian

Diketahui data-datanya yang terurut adalah

a, b, c, d

maka mediannya adalah (b + c)/2 ⇒ pers 1

sehingga

b = 1/2 median

b = 1/2 x (b + c)/2

4b = b + c

c = 3b ⇒ pers 2

a = 1/6 median

a = 1/6 x (b + c)/2

12a = b + c

c kita substitusi dengan pers 2

12a = b + 3b

12a = 4b

a = 1/3 b ⇒ pers 3

Jangkauan = 16

d - a = 16

karena d = c dan c = 3b, maka

3b - a = 16

karena a = 1/3 b, maka

3b - 1/3 b = 16

8/3 b = 16

b = 16 x 3/8 = 6

a = 1/3 b

a = 1/3 x 6 = 2

c = 3b = 3 x 6 = 18

d = c = 18

Rata-rata = (a + b + c + d) / 4

Rata-rata = (2 + 6 + 18 + 18) / 4

Rata-rata = 44/4 = 11
