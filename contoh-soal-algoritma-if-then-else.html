3. Jelaskan perbedaan if-then dengan if-then-else (berikan contoh untuk masing-masingnya)


Perbedaan yang berada diantara if then dengan if then else adalah

If Then

If then memiliki artian sebagai adanya satu bidang kondisi yang dimana akan menjadi sebuah syarat untuk melakukan satu maupun satu blok dari sekelompok pertanyaan. Secara umum, bentuk algoritma daripada teks standar sebuah percabangan dengan satu macam kondisi yaitu:

if <kondisi> then

If Then Else

If then else memiliki artian sebagai dua macam bentuk kondisi yang dimana akan menjadi sebuah sayat untuk dilakukan pengerjaan dari dua jenis bentuk pernyataan. Secara umum, bentuk daripada percabangan dengan menggunakan dua kondisi adalah seperti:

if <kondisi> then

pernyataan1

else

pernyataan2

2. Selection menggunakan if-then-else:

```cpp

#include <iostream>

using namespace std;

int main() {

// Langkah 1: Mendapatkan input dari pengguna

int nilai;

cout << "Masukkan nilai: ";

cin >> nilai;

// Langkah 2: Melakukan seleksi menggunakan if-then-else

cout << "Nilai A" << endl;

} else if (nilai >= 70) {

cout << "Nilai B" << endl;

} else if (nilai >= 60) {

cout << "Nilai C" << endl;

} else if (nilai >= 50) {
```

```cpp

#include <iostream>

using namespace std;

int main() {

// Langkah 1: Inisialisasi variabel pengulangan

// Langkah 2: Melakukan pengulangan menggunakan do-while loop

} while (n <= 5); // Melakukan pengulangan selama n masih kurang dari atau sama dengan 5

return 0;

}
```

flowchart percabangan IF ELSE

Fungsi ELSE

Fungsi else merupakan lanjutan dari fungsi if, dimana jika kondisi benar makan statement pertama yang akan dilaksanakan, tetapi apabila kondisi salah maka statement yang ada pada blok else yang akan dilaksanakan.
