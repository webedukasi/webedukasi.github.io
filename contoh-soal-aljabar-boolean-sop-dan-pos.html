1. Fungsi boolean F (x,y,z)=xy+yz' dalam bentuk kanonik sop dan pos


Jawab:

Fungsi boolean F (x,y,z) dapat dituliskan dalam bentuk sum-of-products (SOP) sebagai berikut:

F = xy + yz'

Dan dapat juga dituliskan dalam bentuk product-of-sums (POS) sebagai berikut:

F = (x+y')(y+z)

Dalam bentuk SOP, kami menyatakan F sebagai penjumlahan dari produk-produk dari variabel-variabel boolean. Dalam hal ini, kami memiliki dua produk: xy dan yz'. Dalam bentuk POS, kami menyatakan F sebagai produk dari penjumlahan-penjumlahan dari variabel-variabel boolean. Dalam hal ini, kami memiliki dua penjumlahan: (x+y') dan (y+z).

SOP dan POS adalah dua cara yang berbeda untuk menyatakan fungsi boolean. Bentuk SOP lebih mudah dibaca, sementara bentuk POS lebih mudah untuk diolah oleh komputer. Kedua bentuk ini dapat diubah satu sama lain dengan menggunakan aturan De Morgan.

9. tentukan bentuk katonik sop&pos


Bentuk Kanonik

• Ada dua macam bentuk kanonik:

1. Penjumlahan dari hasil kali (sum-of-product atau SOP)

2. Perkalian dari hasil jumlah (product-of-sum atau POS)

Contoh: 1. f(x, y, z) = x’y’z + xy’z’ + xyz à SOP

Setiap suku (term) disebut minterm

2. g(x, y, z) = (x + y + z)(x + y’ + z)(x + y’ + z’)

(x’ + y + z’)(x’ + y’ + z) à POS

Setiap suku (term) disebut maxterm

• Setiap minterm/maxterm mengandung literal lengkap

Contoh 1. Nyatakan tabel kebenaran di bawah ini dalam bentuk kanonik SOP dan POS.

Penyelesaian:

(a) SOP

Kombinasi nilai-nilai peubah yang menghasilkan nilai fungsi sama dengan 1 adalah 001, 100, dan 111, maka fungsi Booleannya dalam bentuk kanonik SOP adalah

f(x, y, z) = x’y’z + xy’z’ + xyz

atau (dengan menggunakan lambang minterm),  

f(x, y, z) = m1 + m4 + m7 = å (1, 4, 7)

(b) POS  

Kombinasi nilai-nilai peubah yang menghasilkan nilai fungsi sama dengan 0 adalah 000, 010, 011, 101, dan 110, maka fungsi Booleannya dalam bentuk kanonik POS adalah

f(x, y, z) = (x + y + z)(x + y’+ z)(x + y’+ z’)

(x’+ y + z’)(x’+ y’+ z)

atau dalam bentuk lain,  

f(x, y, z) = M0 M2 M3 M5 M6 = Õ(0, 2, 3, 5, 6)

11. Nyatakan fungsi Boolean f(x,y,z) =x+y’z dalam bentuk kanonik SOP dan POS.


Jawaban:

(a) SOP

      x  = x(y + y’)

          = xy + xy’

          = xy (z + z’) + xy’(z + z’)

          = xyz + xyz’ + xy’z + xy’z’

      y’z = y’z (x + x’)

            = xy’z + x’y’z

      Jadi  f(x, y, z)   = x + y’z

                                        = xyz + xyz’ + xy’z + xy’z’ + xy’z + x’y’z

                                        = x’y’z + xy’z’ + xy’z + xyz’ + xyz

      atau  f(x, y, z)   = m1 + m4 + m5 + m6 + m7 = S (1,4,5,6,7)

(b) POS

            f(x, y, z) = x + y’z

                          = (x + y’)(x + z)

            x + y’ = x + y’ + zz’

          = (x + y’ + z)(x + y’ + z’)

            x + z = x + z + yy’       

                    = (x + y + z)(x + y’ + z)

            Jadi, f(x, y, z) = (x + y’ + z)(x + y’ + z’)(x + y + z)(x + y’ + z)

                                = (x + y  + z)(x + y’ + z)(x + y’ + z’)

CMOS atau Complementary MetalOxide Semiconductor adalah salah satu perangkat keras komputer, yang merupakan  chip bertenaga baterai  pada motherboard, dan daya dari baterai membantu mencegah komputer  menyimpan data.

Sebagian orang pasti akan terkecoh dengan kata 'Pos' dan spontan menjawab 'pos'. Kalau kamu juga menjawab 'pos', artinya kamu sudah berhasil terjebak oleh jawaban yang salah :)


Jawaban sebenarnya adalah TUMPAH.


Mengapa tumpah? Karena kata 'sop' pada kalimat tebak - tebakan itu merupakan kata benda alias makanan sup atau sayur sop. Bayangkan saja bila kita membawa  mangkuk yang berisi sayur sop lalu mangkuk tersebut kita balikkan, maka kuah sop beserta daging dan sayur - mayurnya pasti akan tumpah. Benar bukan?


Jadi jika ada temanmu yang menanyakan tebak-tebakkan ini, jangan jawab "POS" ya!

40. . Nyatakan fungsi Boolean f(x, y, z) = z’ dalam bentuk kanonik SOP


Jawaban:

Jawaban:

(a) SOP

      x  = x(y + y’)

          = xy + xy’

          = xy (z + z’) + xy’(z + z’)

          = xyz + xyz’ + xy’z + xy’z’

      y’z = y’z (x + x’)

            = xy’z + x’y’z

      Jadi  f(x, y, z)   = x + y’z

                                        = xyz + xyz’ + xy’z + xy’z’ + xy’z + x’y’z

                                        = x’y’z + xy’z’ + xy’z + xyz’ + xyz

      atau  f(x, y, z)   = m1 + m4 + m5 + m6 + m7 = S (1,4,5,6,7)

(b) POS

            f(x, y, z) = x + y’z

                          = (x + y’)(x + z)

            x + y’ = x + y’ + zz’

          = (x + y’ + z)(x + y’ + z’)

            x + z = x + z + yy’       

                    = (x + y + z)(x + y’ + z)

            Jadi, f(x, y, z) = (x + y’ + z)(x + y’ + z’)(x + y + z)(x + y’ + z)

                                = (x + y  + z)(x + y’ + z)(x + y’ + z’)
