Dalam layanan e-health, yang tidak termasuk dalam salah satu layanan yang diberikan adalah (C) pengecekan gula darah secara mandiri yang terhubung dengan sistem online.

Pembahasan

Pilihan (C) benar karena pengecekan gula darah memang bisa dilakukan secara mandiri dengan menggunakan alat glukometer, namun berdasarkan jurnal yang diterbitkan oleh National Library of Medicine, pengecekan gula darah dengan menggunakan glukometer ini cenderung tidak akurat sehingga tidak bisa digunakan oleh pasien sebagai tolak ukur kadar gula yang ada pada darahnya. Maka dari itu, pengecekan gula darah secara mandiri ini tidak termasuk dalam salah satu layanan e-health karena pengecekan gula darah sebaiknya dilakukan secara langsung.

Pilihan (A), (B), dan (D) salah karena:

Pilihan (A): adanya layanan rumah sakit 24 jam adalah salah satu bentuk dari layanan e-health.Pilihan (B): konsultasi online dengan dokter adalah salah satu layanan utama dari e-health.Pilihan (D): layanan Clinical Decision Support System (CDSS) adalah salah satu layanan e-health, seperti adanya aplikasi Athena.

3. Nilai x yang memenuhi persamaan l2-xl²-l2-xl=3l2-xl+5 adalah​


Penjelasan dengan langkah-langkah:

|2 - x|² - |2 - x| = 3|2 - x| + 5

misal |2 - x| = p, maka :

p² - p = 3p + 5

p² - p - 3p - 5 = 0

p² - 4p - 5 = 0

(p + 1)(p - 5) = 0

p = -1 atau p = 5

• untuk p = -1

|2 - x| = -1 (tidak memenuhi, karena nilai mutlak tidak mungkin bernilai negatif)

• untuk p = 5

|2 - x| = 5

2 - x = -5 atau 2 - x = 5

-x = -5 - 2 atau -x = 5 - 2

-x = -7 atau -x = 3

x = 7 atau x = -3

Jadi, HP = {-3, 7}

34. identitas trigonometri kelas xl ​


Pembahasan

Identitas Trigonometri

[tex] \boxed{ \sin(2 \alpha ) = 2 \sin( \alpha ) \cos( \alpha ) } \\ \boxed{ \cos(2 \alpha ) = 1 - 2 { \sin }^{2} ( \alpha ) } \\ \boxed{ \tan(2 \alpha ) = \frac{ \sin(2 \alpha ) }{ \cos(2 \alpha ) } } \\ \\ [/tex]

Diketahui :

[tex] \sin( \alpha ) = \frac{24}{25} \: \: \: , \: \: 0 < \alpha < \frac{\pi}{2} \\ \\ [/tex]

Ditanya :

[tex]a. \sin(2 \alpha ) \\ \\ b. \cos(2 \alpha ) \\ \\ c. \tan(2 \alpha ) \\ \\ [/tex]

Penyelesaian :

[tex] \sin( \alpha ) = \frac{24}{25} \\ \\ \cos( \alpha ) = \frac{ \sqrt{ {25}^{2} - {24}^{2} } }{25} = \frac{ \sqrt{49} }{25} = \frac{7}{25} \\ \\ [/tex]

[tex]a. \\ \sin(2 \alpha ) = 2 \sin( \alpha ) \cos( \alpha ) \\ \\ \sin(2 \alpha ) = 2 \cdot \frac{24}{25} \cdot \frac{7}{25} = \frac{336}{625} \\ \\ [/tex]

[tex]b. \\ \cos(2 \alpha ) = 1 - 2 { \sin }^{2} ( \alpha ) \\ \\ \cos(2 \alpha ) = 1 - 2 { \left ( \frac{24}{25} \right )}^{2} \\ \\ \cos(2 \alpha ) = 1 - 2 \left ( \frac{576}{625} \right ) \\ \\ \cos(2 \alpha ) = 1 - \left ( \frac{1152}{625} \right ) \\ \\ \cos(2 \alpha ) = - \frac{527}{625} \\ \\[/tex]

[tex]c. \\ \tan(2 \alpha ) = \frac{ \sin(2 \alpha ) }{ \cos(2 \alpha ) } \\ \\ \tan(2 \alpha ) = \frac{ \frac{336}{625} }{ - \frac{527}{625} } \\ \\ \tan(2 \alpha ) = - \frac{336}{527} \\ \\ [/tex]
