1. uraikan penggunaan OLE pada excel dan powerpoint​


Jawaban:

OLE (Object Linking and Embedding) adalah teknologi yang digunakan dalam aplikasi Microsoft Office seperti Excel dan PowerPoint untuk menggabungkan atau mengaitkan objek dari satu dokumen ke dalam yang lain. Berikut adalah penggunaan OLE dalam Excel dan PowerPoint:

1. OLE dalam Excel:

- Embedding: Anda dapat menyisipkan objek dari aplikasi lain (seperti gambar, grafik, atau dokumen Word) ke dalam lembar kerja Excel. Objek yang disisipkan akan menjadi bagian dari file Excel Anda. Anda dapat mengedit objek tersebut tanpa membuka aplikasi asalnya.

- Linking: Anda juga dapat membuat tautan ke objek dari aplikasi lain. Ini berarti bahwa data atau objek tersebut tetap berada dalam dokumen asalnya, dan perubahan di dokumen asal akan tercermin dalam file Excel Anda jika diperbarui.

2. OLE dalam PowerPoint:

- Embedding: Dalam PowerPoint, Anda dapat menyisipkan objek seperti gambar, grafik, atau dokumen Word ke dalam slide presentasi. Ini memungkinkan Anda untuk menambahkan konten dari aplikasi lain langsung ke presentasi Anda.

- Linking: Anda juga dapat membuat tautan ke objek dari aplikasi lain, mirip dengan Excel. Ini memungkinkan Anda untuk memperbarui konten dalam slide PowerPoint jika dokumen asalnya berubah.

Contoh penggunaan OLE dalam Excel: Menyisipkan grafik dari program lain ke dalam lembar kerja Excel Anda.

Contoh penggunaan OLE dalam PowerPoint: Menyisipkan tabel atau dokumen Word ke dalam slide PowerPoint Anda atau membuat tautan ke data eksternal yang akan diperbarui secara otomatis.

Penting untuk diingat bahwa ketika Anda mengggunakan OLE, dokumen sumber (dalam hal ini, dokumen asal) harus tetap ada untuk mempertahankan tautan atau objek yang disisipkan. Jika dokumen asal dihapus atau berubah, itu juga akan memengaruhi dokumen yang menggunakan OLE.

8. Tipe dalam penyimpanan yang digunakan untuk menyimpan file yang terdapat macro didalamnya dan macro yang sudah dibuat akan tersimpan di dalam file, yaitu tipe…. Pilih salah satu: a. Excel Binary Workbook b. Excel Macro-Enable Workbook c. Excel Workbook d. Excel Add-In​


Jawaban:

b. Excel Macro-Enable Workbook

Penjelasan:

Berikut ini sedikit penjelasan dalam penggunaannya.

1. Excel Workbook, Menyimpan dengan tipe ini seperti menyimpan file excel atau workbook pada umumnya. Pada file ini tidak terdapat Macro.

2. Excel Macro-Enable Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya. Macro yang sudah dibuat akan tersimpan di dalam file.

3. Excel Binary Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya akan tetapi macro akan tersimpan pada system Microsoft Office Excel.

4. Excel Add-In, Menyimpan dengan tipe ini digunakan untuk menyimpan file macro yang dapat anda gunakan sebagai Add-in atau program tambahan dalam Program Excel.

Langkah-langka Cara menyimpan file Macro :

1. Lakukan Penyimpanan dengan tombol SAVE

2. Akan tampil peringatan “the following features cannot be saved in Macro-free workboks” peringatan ini menginformasikan bahwa file ini terdapat VB Project (macro). Jika anda pilih Yes maka Macro yang sudah dibuat akan dihapus, dikarenakan workbook biasa tidak diperkenenkan untuk menyimpan macro.

3. Pilih No Untuk menyimpan macro.

4. Lakukan penyimpanan didalam folder tujuan. Ubahlah Save as Type Menjadi Excel macro-enable Workbook (xlsm).

5. Klik Save

39. convert decimal time to hours and minutes in excel


1. Select a blank cell which you want to output the converted result, and enter this formula =A2/24, drag fill handle over the cells you need. See screenshot:


2. Keep the formula cells highlighted, and right click to show context menu and select Format Cells. And in the Format Cells dialog, select Time from the Category list, and choose one time type as you need from right section. See screenshot:




3. Click OK. Now the decimal hours have been converted to time format.


Note: if you want to convert decimal minutes to time format, you can apply this formula =A2/1440, and then format the cells as the time format you want. See screenshot:
