35. Suatu software yang digunakan untuk memonitor layanan GPON disebut…*A down timeB. reportingC. flex manageD. SplitterE. multiple path​


Jawaban:

C. flex manage

Penjelasan:

Flex manage adalah software yang digunakan untuk memonitor layanan GPON (Gigabit Passive Optical Network). Software ini memberikan kemampuan untuk mengelola dan memantau jaringan GPON secara efisien, termasuk pemantauan kinerja, manajemen perangkat, pemecahan masalah, dan pengaturan konfigurasi. Flex manage memungkinkan operator jaringan untuk melacak kinerja jaringan, mengidentifikasi masalah potensial, dan mengambil tindakan yang diperlukan untuk memastikan layanan GPON berjalan dengan baik.
