Jawab:

Untuk mencari faktor-faktor dari persamaan x² - 1, kita perlu mencari nilai-nilai x yang memenuhi persamaan tersebut.

Dalam kasus ini, kita dapat menggunakan rumus faktor perbedaan kuadrat, yaitu a² - b² = (a + b)(a - b).

Persamaan x² - 1 dapat ditulis sebagai (x - 1)(x + 1) = 0.

Berdasarkan rumus faktor perbedaan kuadrat, kita dapat melihat bahwa faktor-faktor dari persamaan ini adalah (x - 1) dan (x + 1).

Sehingga, jawaban yang benar adalah:

b. x = 0 atau x = -1

atau

x = 1 atau x = -1
