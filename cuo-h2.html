3. hasil reduksi dan reduktor dari Cuo+H2→Cu+H2o


Hasil reduksi dan reduktor dari CuO + H₂ → Cu + H₂O adalah

Reduktor = mengalami oksidasi = biloksnya naik = H₂

Hasil Reduksi = Cu

Pembahasan :

(Reaksi Oksidasi/ Reduktor)

CuO + H₂ → Cu + H₂O

           |________|  

          0                +1

Atom H pada reaksi diatas mengalami kenaikan bilangan oksidasi dari 0 berubah menjadi +1, sehingga mengalami reaksi oksidasi atau reduktor.  

(Reaksi Reduksi/ Oksidator)

CuO + H₂ → Cu + H₂O

|_________|  

+2                0

Atom Cu pada reaksi diatas mengalami penurunan bilangan oksidasi dari +2 berubah menjadi 0, sehingga mengalami reaksi reduksi atau oksidator.  

Maka diperoleh :

Oksidator = mengalami reduksi = biloksnya turun = CuO

Reduktor = mengalami oksidasi = biloksnya naik = H₂

Hasil Oksidasi = H₂O

Hasil Reduksi = Cu

Pengertian Oksidasi dan Reduksi sebagai reaksi peningkatan dan penurunan bilangan oksidasi  

Oksidasi adalah reaksi dengan peningkatan bilangan oksidasi.  

Zat yang mengalami kenaikan bilangan oksidasi disebut reduktor.  

Reduksi adalah reaksi dengan penurunan bilangan oksidasi.  

Zat yang mengalami penurunan bilangan oksidasi disebut oksidator.

31. Pada reaksi : cuo + h2 → cu+h2o. cu mengalami reduksi karena


Pembahasan

Diketahui reaksi redoks sebagai berikut

CuO + H₂ → Cu + H₂O

[a]. Perubahan bilangan oksidasi

CuO + H₂ → Cu + H₂O

+2, (-2)..|.0...|......0...|...2(+1), (-2)

Cu ⇒ +2 di ruas kiri, menjadi 0 di ruas kanan

O ⇒ -2 di ruas kiri, tetap -2 di ruas kanan

H₂ ⇒ 0 di ruas kiri, menjadi +1 di ruas kanan

[b].

Cu pada CuO mengalami penurunan bilangan oksidasi (reduksi)

H₂ mengalami kenaikan bilangan oksidasi (oksidasi)

Dapat disimpulkan beberapa hal sebagai berikut:

⇒ Zat yang bertindak sebagai reduktor (mereduksi zat lain) adalah zat yang mengalami oksidasi, yakni H₂

⇒ Zat yang bertindak sebagai oksidator (mengoksidasi zat lain) adalah zat yang mengalami reduksi, yakni CuO

⇒ Hasil reduksi adalah Cu

⇒ Hasil oksidasi adalah H₂O


32. tentukan oksidator dan reduktor dari CuO + H2 -->Cu H2O


Oksidator adalah zat yang mengalami reduksi.
Reduktor adalah zat yang mengalami oksidasi.

atom Cu dalan CuO memiliki bilangan oksidasi +2, berubah menjadi logam Cu yang memiliki bilangan oksidasi 0. Karena bilangan oksidasi berkurang, berarti Cu mengalami reduksi. Karena mengalami reduksi maka CuO bertindak sebagai oksidator.

Atom H dalam H2 memiliki bilangan oksidasi 0. atom H dalam molekul H2O memiliki bilangan okaidasi +1. Karena bilangan oksidasi bertambah, berarti H mengalami oksidasi. Karena mengalami oksidasi maka H2 bertindak sebagai reduktor.

Jawaban:

Dalam reaksi redoks, senyawa yang mengalami reaksi reduksi disebut oksidator, sedangkan senyawa yang mengalami reaksi oksidasi disebut reduktor.

a. CuO+H2 -> Cu+H2O

biloks : +2 0 0 +1

Terjadi penurunan biloks pada atom Cu, sehingga CuO mengalami reduksi ( oksidator )

Terjadi peningkatan biloks pada atom H, sehingga H2 mengalami oksidasi ( reduktor)

b. SO2+H2S->S2+H2O

biloks : +4 -2 0

Terjadi penurunan biloks pada atom S, sehingga SO2 mengalami reduksi ( oksidator )

Terjadi peningkatan biloks pada atom S, sehingga H2S mengalami oksidasi ( reduktor)

c. Fe+2H+ -> Fe2+ + H2

biloks : 0 +1 +2 0
