18. bantu kak lagi placement test​


Rumus suku ke-n

Un = Sn - Sn₋₁

Un = 3n² + 6n - (3(n - 1)² + 6(n - 1))

Un = 3n² + 6n - (3(n² - 2n + 1) + 6n - 6)

Un = 3n² + 6n - 3n² + 6n - 3 - 6n + 6

Un = 6n + 3

Suku ke-10

U₁₀ = 6(10) + 3 = 60 + 3 = 63
