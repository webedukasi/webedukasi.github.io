Pendahuluan

Aljabar adalah suatu cabang matematika yang menggunakan huruf sebagai perwakilan angka yang belum diketahui secara jelas.

- Unsur unsur aljabar -

•Variabel adalah huruf yang digunakan sebagai perwakilan angka yang belum diketahui secara jelas. Contoh dalam bentuk aljabar 2x + 10, x adalah variabel

• Koefisien adalah angka yang diikuti oleh variabel.Contohnya dalam bentuk aljabar 2x + 10, 2 adalah koefisien

• Konstanta adalah angka yang tidak diikuti oleh variabel.Contohnya dalam bentuk aljabar 2x + 10, 10 adalah konstanta

• suku adalah variabel serta koefisien atau konstanta yang dipisahkan oleh tanda operasi hitung.Contohnya dalam bentuk aljabar 2x + 10, Jumlah suku ada 2, yaitu 2x dan 10

-Sifat sifat operasi hitung aljabar-

• Sifat Komutatif

a + b = b + a

a × b = b × a

• Sifat Asosiatif

(a + b) + c = a + (b + c)

(a × b) × c = a × (b × c)

• Sifat Distributif

a(b + c) = a × b + a × c

a(b - c) = a × b - a × c

-Perkalian bentuk aljabar-

Perkalian bentuk aljabar dilakukan dengan mengalikan koefisien dengan koefisien dan variabel dengan variabel.Pada operasi perkalian suku tidak harus sejenis.

-Pembagian bentuk aljabar-

• Pembagian bentuk aljabar suku tunggal

Dengan cara membagi koefisien dengan koefisien dan variabel dengan variabel.

• Pembagian bentuk aljabar dua suku atau lebih Dengan cara pembagian bersusun.

Pembahasan

Contoh soal perkalian bentuk aljabar :

(4a + 5b) (4 + 2a - b)

= ((4a × (4 + 2a - b) + 5b × (4 + 2a - b))

= 16a + 8a² - 4ab + 5b × (4 + 2a - b)

= 16a + 8a² - 4ab + 20b + 10ab - 5b²

= 16a + 8a² + 6ab + 20b - 5b²

Contoh soal pembagian bentuk aljabar :

(3x² + 8x - 35) : (3x - 7)

= (3x² + 15x - 7x - 35) : (3x - 7)

= 3x(x + 5) - 7(x + 5) : 3x - 7

= (x + 5) (3x - 7) : 3x - 7

= x + 5

KESIMPULAN

• (4a + 5b) (4 + 2a - b) = 16a + 8a² + 6ab + 20b - 5b²

• (3x² + 8x - 35) : (3x - 7) = x + 5

Sifat-Sifat Pada Bilangan Berpangkat :

[tex] \sf a^m \times  a^n = a^{(m+n)} [/tex][tex] \sf a^m \div a^n = a^{(m-n)} [/tex][tex] \sf (a^m)^n = a^{m\times n} [/tex][tex] \sf (ab)^n =a^nb^n [/tex][tex] \sf (\frac{a}{b})^n = \frac{a^n}{b^n} [/tex][tex] \sf a^{-n} = \frac{1}{a^n} [/tex][tex] \sf \sqrt[n]{ {a}^{m} } = {a}^{ \frac{m}{n} } [/tex][tex] \sf a^0 = 1 [/tex]

[tex] \: [/tex]

Pembahasan:Diketahui : [tex] \bold { \purple { \sf - 16 {(ab)}^{x} \div 4 {a}^{x} \times {b}^{x} }}[/tex]

[tex] \: [/tex]

Ditanya : Penyederhanaan?

[tex] \: [/tex]

Dijawab :

[tex] = \bold { \purple { \sf - 16 {(ab)}^{x} \div 4 {a}^{x} \times {b}^{x} }}[/tex]

[tex] = \sf (- 16 \div 4) \: {a}^{x} {b}^{x} \times {a}^{x} \times {b}^{x} [/tex]

[tex] = \sf - 4 \: {a}^{(x + x)} {b}^{(x + x)} [/tex]

[tex] = \bold { \red { \sf - 4 \: {a}^{2x} {b}^{2x} }}[/tex]

[tex] \: [/tex]

Kesimpulan:

Jadi, Penyederhanaan dari [tex] \bold { \purple { \sf - 16 {(ab)}^{x} \div 4 {a}^{x} \times {b}^{x} }}[/tex] adalah [tex] \bold { \red { \sf - 4 \: {a}^{2x} {b}^{2x} }}[/tex].
