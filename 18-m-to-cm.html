Persamaan umum gas ideal  

Persamaan umum gas ideal dinyatakan dengan :  

PV = nRT  

Dengan nilai n = N/NA, maka persamaan diatas bisa ditulis dalam bentuk lain :  

PV = NkT  

Dengan menguraikan nilai n = m/Mr dan ρ = m/V, maka didapat bentuk persamaan berikutnya :  

P.Mr = ρ.R T  

Keterangan simbol :  

P : Tekanan ( Pa )  

V : Volume ( m3 )  

n : Jumlah mol  

N : Jumlah partikel / molekul  

NA : Bilangan Avogadro ( 6,02 x 1023 )  

T : Temperatur ( K )  

R : 8,31 J/mol K = 0,082 atm.L/mol.K  

k : 1,38 x 10-23

[tex]\frac{PV}{T}=c[/tex]
