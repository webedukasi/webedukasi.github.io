1. contoh soal singkat materi tentang getaran harmonik


Jawaban:

1) Simpangan

Simpangan pada gerak harmonik sederhana dapat dianggap sebagai proyeksi partikel yang bergerak melingkar beraturan pada diameter lingkaran, dengan rumus sebagai berikut.

2) Kecepatan

Kecepatan dalam gerak harmonik sederhana didapat dari turunan pertama persamaan simpangan, dengan rumus:

V = dy/dt = d(A sin t)/dt

3) Percepatan

Percepatan benda yang bergerak harmonik sederhana diperoleh dari turunan pertama persamaan kecepatan atau turunan kedua persamaan simpangan dengan rumus berikut.

a = dv/dt= d2y/dt2 = d2 (A sin t)/dt2 = -A2sin t = -2y = -A2

Keterangan:

y = simpangan getaran (m)

A = amplitudo/simpangan maksimum (m)

= kecepatan sudut (rad/s)

t = waktu tempuh (s).

Penjelasan:

y=A sin (t)

y=A sin (2Tt)

y=5 sin (20,4 115)

y= 53 cm

2. berikan 5 contoh soal singkat materi tentang getaran harmonik


1. Sebuah pegas memiliki konstanta restitusi sebesar 2 N/m. Jika pegas tersebut digantungkan dengan massa 0,5 kg, maka frekuensi getaran harmonik pada sistem tersebut adalah?

Jawab: f = 1/(2pi) x sqrt(k/m) = 1/(2pi) x sqrt(2/0,5) = 3,18 Hz

2. Sebuah ayunan benda memiliki panjang 1 m dan massa 0,2 kg. Jika benda digantungkan dan diberi dorongan awal sehingga bergetar harmonik sederhana, maka periode getaran pada sistem tersebut adalah?

Jawab: T = 2pi x sqrt(l/g) = 2pi x sqrt(1/9,8) = 0,63 s

3. Sebuah pegas memiliki periode getaran sebesar 0,5 s. Jika massa yang digantungkan pada pegas tersebut adalah 0,6 kg, maka konstanta restitusi pegas adalah?

Jawab: T = 2pi x sqrt(m/k); 0,5 = 2pi x sqrt(0,6/k); k = 24,61 N/m

c = 2 x pi x sqrt(m x k); c = 13,82 Ns/m

4. Sebuah benda dengan massa 0,1 kg digantungkan pada sebuah pegas. Posisi kesetimbangan awal benda berada di titik tengah antara dua ujung pegas. Jika pegas ditarik sehingga benda bergeser sejauh 4 cm dari posisi kesetimbangan dan dilepaskan, maka amplitudo dan periode getaran harmonik pada sistem tersebut adalah?

Jawab: A = 4 cm; T = 2pi x sqrt(m/k) = 2pi x sqrt(0,1/2) = 0,89 s

5. Percepatan maksimum sebuah benda yang bergetar harmonik sederhana dengan amplitudo 5 cm dan periode 0,2 s adalah?

Jawab: a_maks = 2 x pi x A / T^2 = 2 x pi x 0,05 / 0,2^2 = 1,96 m/s^2

28. pada peristiwa getaran harmonik besaran yang sebanding dengan simpangan getaran adalah ​


Jawaban:

Pembahasan materi tentang Getaran Harmonis Sederhana ini bisa di lihat pada link - link " Pelajari Lebih Lanjut " di bawah ini .

Okay mari kita sekarang mencoba menyelesaikan soalnya.

Persamaan simpangan dari benda yang bergetar harmonis sederhana bisa di tuliskan dalam bentuk umum :

x = A sin ( ωt )

dimana :

x = simpangan getaran ( m )

A = simpangan maksimum getaran ( m )

ω = frekuensi sudut getaran ( rad/s )

t = selang waktu ( s )

Sedangkan untuk mencari kecepatan dari getaran harmonis sederhana adalah dengan cara menurunkan persamaan simpangan di atas sehingga menjadi :

v = dx / dt

v = A ω cos ωt

dimana :

v = kecepatan getaran ( m )

A = simpangan maksimum getaran ( m )

ω = frekuensi sudut getaran ( rad/s )

t = selang waktu ( s )

Dan terakhir untuk mencari percepatan getaran harmonis ini bisa di cari dengan cara menurunkan lagi persamaan kecepatan di atas sehingga menjadi :

a = dv / dt

a = - A ω² sin ωt

dimana :

a = percepatan getaran ( m )

A = simpangan maksimum getaran ( m )

ω = frekuensi sudut getaran ( rad/s )

t = selang waktu ( s )

Nah dari rumus persamaan simpangan dan persamaan getaran ini bisa di hubungkan dengan cara :

a = - A ω² sin ωt

a = - ω² ( A sin ωt )

a = - ω² x

a ∝ x → Percepatan(a) berbanding lurus dengan Simpangan Getaran(x)
