1. Untuk menghitung harga obligasi kita dapat menggunakan rumus Present Value of Annuity:

Harga obligasi = (C × (1 - (1 + r/n)^(-n × t))) / (r/n) + (F / (1 + r/n)^(n × t))

di mana:

C = kupon per tahun (dalam persen dari nilai nominal)

r = tingkat bunga per tahun (dalam persen)

n = jumlah pembayaran kupon per tahun

t = jumlah tahun

a. Untuk tingkat bunga pasar 8%:

C = 10% dari Rp. 1.000.000 = Rp. 100.000

r = 8% = 008

n = 2 (kupon dibayarkan tengah tahunan)

t = 5 tahun

Harga obligasi = (100.000 × (1 - (1 + 008/2)^(-2 × 5))) / (008/2) + (1.000.000 / (1 + 008/2)^(2 × 5))

b. Untuk tingkat bunga pasar 10%:

C = 10% dari Rp. 1.000.000 = Rp. 100.000

r = 10% = 010

n = 2 (kupon dibayarkan tengah tahunan)

t = 5 tahun

Harga obligasi = (100.000 × (1 - (1 + 010/2)^(-2 × 5))) / (010/2) + (1.000.000 / (1 + 010/2)^(2 × 5))

c. Untuk tingkat bunga pasar 12%:

C = 10% dari Rp. 1.000.000 = Rp. 100.000

r = 12% = 012

n = 2 (kupon dibayarkan tengah tahunan)

t = 5 tahun

Harga obligasi = (100.000 × (1 - (1 + 012/2)^(-2 × 5))) / (012/2) + (1.000.000 / (1 + 012/2)^(2 × 5))

Setelah menghitung harga obligasi menggunakan rumus di atas kita dapat mengetahui apakah obligasi diperdagangkan dengan diskon premium atau setara dengan nilai pari. Jika harga obligasi kurang dari nilai nominal maka obligasi diperdagangkan dengan diskon. Jika harga obligasi lebih besar dari nilai nominal maka obligasi diperdagangkan dengan premium. Jika harga obligasi sama dengan nilai nominal maka obligasi diperdagangkan setara dengan nilai pari.

2. Untuk menghitung nilai YTM (kd kita dapat menggunakan rumus Present Value of Bond:

Nilai pasar obligasi = (C × (1 - (1 + kd/n)^(-n × t))) / (kd/n) + (F / (1 + kd/n)^(n × t))

di mana:

C = kupon per tahun (dalam persen dari nilai nominal)

kd = tingkat diskon yield to maturity per tahun (dalam persen)

n = jumlah pembayaran kupon per tahun

t = jumlah tahun

F = nilai nominal obligasi

Dalam kasus ini:

C = 8% dari $5.000 = $400

Harga pasar obligasi = $1.000

n = 2 (kupon dibayarkan setiap semester)

t = 10 tahun

F = $5.000

Secara umum nilai YTM (kd) adalah tingkat diskonto yang diperoleh ketika harga pasar obligasi sama dengan nilai sekarang dari pembayaran kupon dan nilai nominal obligasi.

Kita dapat mencoba berbagai tingkat diskonto (kd) dalam rumus yang diberikan dan mengiterasi sampai harga pasar obligasi (yang diketahui) mendekati nilai sebenarnya ($1000).

Hambatan komunikasi inovasi dapat bersifat internal.

a. Unsur-unsur yang terlibat dalam proses difusi inovasi adalah sebagai berikut:

Inovasi itu sendiri, yaitu ide atau teknologi baru yang akan diterapkan atau diadopsi oleh individu atau kelompok.

Individu atau kelompok yang menerima dan mengadopsi inovasi tersebut, disebut sebagai "penerima inovasi".

Individu atau kelompok yang menyebarkan atau mengkomunikasikan inovasi tersebut kepada penerima inovasi, disebut sebagai "komunikator inovasi".

b. Difusi inovasi merupakan bagian dari komunikasi karena proses tersebut melibatkan pertukaran informasi dan ide antara komunikator dan penerima inovasi. Proses tersebut juga melibatkan interaksi sosial antara kedua belah pihak, sehingga dapat terjadi persuasi atau pengaruh terhadap penerima inovasi untuk mengadopsi inovasi tersebut.

Hambatan komunikasi inovasi dapat bersifat internal, yaitu hambatan yang terjadi di dalam pikiran atau sikap penerima inovasi sendiri. Hambatan tersebut dapat berupa ketidakpercayaan terhadap inovasi, sikap skeptis, atau ketidakmampuan untuk memahami atau mengimplementasikan inovasi tersebut. Hambatan internal ini dapat diatasi dengan memberikan penjelasan yang jelas dan detail tentang inovasi tersebut, serta memberikan dukungan dan bantuan teknis untuk memahami dan mengimplementasikannya.

Obligasi "AA" memiliki nominal Rp 2.000.000,-, kupon 10%, maturity dalam 5 tahun, dan pembayaran kupon dilakukan setiap triwulan. YTM = 6%. Berapa kurs obligasi "AA" ? 5. Obligasi B "at premium" memiliki kurs 250 dengan kupon 10% dan nominalnya Rp

Untuk menghitung kurs obligasi "AA", Anda perlu menggunakan rumus present value (nilai sekarang) dari pembayaran-pembayaran yang akan datang dari obligasi tersebut. Nilai sekarang merupakan ukuran nilai suatu obligasi pada saat ini berdasarkan pada tingkat suku bunga yang berlaku pada saat itu.

Rumus present value adalah sebagai berikut:

Present value = Kupon * (1 - (1 + YTM)^(-n)) / YTM + Nilai nominal / (1 + YTM)^n

Keterangan:

Kupon: tingkat bunga obligasi yang dibayarkan setiap periode. Dalam kasus ini, kupon obligasi "AA" adalah 10%.

YTM: tingkat yield to maturity, yaitu tingkat suku bunga yang diharapkan akan diterima investor jika memegang obligasi hingga jatuh tempo. Dalam kasus ini, YTM obligasi "AA" adalah 6%.

n: jumlah periode pembayaran kupon. Dalam kasus ini, kupon dibayarkan setiap triwulan selama 5 tahun, sehingga n = 4 per tahun * 5 tahun = 20 periode.

Nilai nominal: nominal obligasi, yaitu jumlah uang yang akan dibayarkan pada saat jatuh tempo. Dalam kasus ini, nilai nominal obligasi "AA" adalah Rp 2.000.000,-.

Jadi, dengan menggunakan rumus di atas, kurs obligasi "AA" dapat dihitung sebagai berikut:

Present value = 10% * (1 - (1 + 6%)^(-20)) / 6% + Rp 2.000.000,- / (1 + 6%)^20

= Rp 1.864.185,-

Jadi, kurs obligasi "AA" adalah Rp 1.864.185,-.

Untuk obligasi B yang "at premium", kurs obligasi tersebut lebih besar dari nilai nominalnya. Jadi, jika kurs obligasi B adalah Rp 250,- dengan kupon 10% dan nilai nominal Rp 100.000,-, maka obligasi tersebut dianggap "at premium" karena kursnya lebih besar dari nilai nominalnya.

Jawaban:

Untuk menghitung nilai obligasi, kita dapat menggunakan rumus:

Nilai obligasi = (Bunga kupon × Nilai nominal) + (Nilai nominal / (1 + YTM)^n)

Dimana:

- Bunga kupon adalah persentase bunga yang dibayarkan secara periodik (6% per tahun)

- Nilai nominal adalah nilai nominal obligasi ($1000)

- YTM adalah tingkat pengembalian pasar (6% per tahun)

- n adalah jangka waktu obligasi (10 tahun)

Mari kita gantikan nilai-nilai yang diketahui ke dalam rumus:

Nilai obligasi = (0.06 × $1000) + ($1000 / (1 + 0.06)^10)

Saat menghitung nilai obligasi, bandingkanlah tingkat pengembalian pasar (YTM) dengan bunga kupon. Jika YTM lebih rendah dari bunga kupon, nilai obligasi akan lebih tinggi dari nilai nominal. Jika YTM sama dengan bunga kupon, nilai obligasi akan sama dengan nilai nominal. Jika YTM lebih tinggi dari bunga kupon, nilai obligasi akan lebih rendah dari nilai nominal.

Dalam kasus ini, YTM (6%) sama dengan bunga kupon (6%), jadi nilai obligasi akan sama dengan nilai nominal.

Jadi, nilai obligasi adalah $1000.

19. Berapa coupon rate untuk sebuah obligasi dengan nilai nominal Rp 10 juta, mempunyai 5 tahun sampai jatuh tempo, YTM 8%, dan sekarang dijual pada harga Rp 11.197.810,00. (jangan gunakan rumus mencari YTM, gunakan formula menghitung obligasi biasa​


Jawaban:
Untuk menghitung coupon rate obligasi, kita dapat menggunakan rumus sebagai berikut:

Coupon Rate = (Coupon Payment / Face Value) x 100%

Dimana:

   Coupon Payment: pembayaran kupon tahunan

   Face Value: nilai nominal obligasi

Pertama, kita dapat menghitung pembayaran kupon tahunan dengan menggunakan YTM dan nilai nominal obligasi:

PV = C / (1+r)^1 + C / (1+r)^2 + ... + C / (1+r)^n + F / (1+r)^n

Di mana:

   PV: harga obligasi saat ini (present value)

   C: pembayaran kupon tahunan

   r: yield to maturity (atau tingkat pengembalian yang diharapkan)

   n: jumlah tahun hingga jatuh tempo

   F: nilai nominal obligasi (future value)

Dalam kasus ini, PV = Rp 11.197.810,00, F = Rp 10.000.000,00, r = 8%, dan n = 5 tahun. Kita akan mencari C:

Rp 11.197.810,00 = C / (1+0.08)^1 + C / (1+0.08)^2 + C / (1+0.08)^3 + C / (1+0.08)^4 + C / (1+0.08)^5 + Rp 10.000.000,00 / (1+0.08)^5

Rp 11.197.810,00 = C x (3.9936)

C = Rp 11.197.810,00 / (3.9936) = Rp 2.801.347,24

Jadi, pembayaran kupon tahunan adalah sebesar Rp 2.801.347,24.

Kemudian, kita dapat menghitung coupon rate dengan membagi pembayaran kupon tahunan dengan nilai nominal obligasi:

Coupon Rate = (Rp 2.801.347,24 / Rp 10.000.000,00) x 100% = 28,01%

Jadi, coupon rate untuk obligasi tersebut adalah 28,01%.
