31. Jelaskan perbedaan antara IMK dan GUI!


Perbedaan antara IMK dan GUI :

CLI atau Command Line Interface ialah tipe antarmuka dimana pengguna berinteraksi dengan sistem operasi melalui text terminal.  

Sedangkan GUI atau Graphics User Interface ialah tipe antarmuka yang dipakai oleh pengguna untuk berinteraksi dengan sistem operasi melalui gambar grafik, ikon, menu, serta menggunakan perangkat penunjuk atau pointing device seperti mouse ataupun track ball.  

Pembahasan CLI :  

Pengguna menjalankan perintah dan program di sistem operasi dengan cara mengetikkan baris-baris tertentu. Meskipun mempunyai konsep yang sama, tiap-tiap sistem operasi mempunyai nama atau istilah yang berbeda untuk CLI-nya. UNIX memberi nama CLI-nya sebagai bash, ash, ksh, dan yang lainnya. Microsoft Disk Operating System atau MS-DOS memberikan nama command.com atau Command Prompt. Sedangkan pada Windows Vista, Microsoft menamakannya dengan PowerShell. Pengguna Linux mengenal CLI pada Linux sebagai terminal, sedangkan pada Apple dinamakan dengan commandshell.

GUI :

Elemen-elemen utama dari GUI dapat diringkas dalam konsep WIMP (window, icon, menu, pointing device). Sebuah sistem operasi dianggap bagus jika tampilan luarnya atau GUI-nya juga bagus, walaupun komponen sistem operasi tidak hanya GUI saja. Jadi, penilaian terhadap sebuah sistem operasi tidak bisa hanya dari satu komponen saja dikarenakan GUI ialah kesan pertama pengguna dengan sistem operasi tersebut. Setiap pengembang sistem operasi berlomba-lomba mengembangkan GUI-nya dengan keunggulannya masing-masing.
