28. Jelaskan langkah-langkah untuk menyimpan dokumen dengan nama tugas praktek pada program Microsoft Word 2007 ​


Jawaban:

Langkah 1: ketikkan naskah hingga selesai di program Ms. Word 2007

Langkah 2: tekan tombol kombinasi di keyboard, yaitu Ctrl+S

Langkah 3: tunggu hingga muncul kotak dialog Save As

Langkah 4: di bagian File Name, ketikkkan tugas praktek

Langkah 5: klik tombol Save

Penjelasan panjang:

Amatilah lampiran:

kotak dialog Save As ditunjukkan tanda panah coklat

bagian File Name ditunjukkan tanda panah merah

tombol Save ditunjukkan tanda panah biru

bila langkah 1 sampai kelima sudah dilakukan, maka telah sukses menyimpan dokumen dg nama tugas praktek
