2. 50F°= °K Tolong jawab kak besok di kumpulkan


Jawaban:

50 °F = 283 ⁰K

Penjelasan:

50 °F ubah ke Celcius

C = 5/9. ( F - 32 )

C = 5/9. ( 50 - 32 )

C = 5/9. 18 = 10 °C

Ubah Celcius ke Kelvin

K = C + 273

K = 10 + 273 = 283 ⁰K

Jadi

50 °F = 283 ⁰K

Jawaban:

77⁰F = x (77-32) = x 45 = 20⁰C

3) Fahrenheit - Kelvin

77⁰F = x (77-32) + 273 = x 45 = 25 + 273 = 298⁰K

b. 1) Kelvin - Celcius

333⁰K = 333 - 273 = 60⁰C

2) Kelvin - Reamur

333⁰K = x (333-273) = x 60 = 48⁰R

3) Kelvin - Fahrenheit

333⁰K = x (333-273) + 32 = x 60 = 108 + 32 = 140⁰F

6. suhu ruangan diukur menggunakan termometer fahrenheit menunjukkan skala 50F berapakah skala yang ditunjukan oleh termometer celcius​


Jawaban:

10 derajat Celcius

Penjelasan:

Untuk mengubah Fahrenheit ke Celcius, pertama-tama kurangi dahulu derajat Fahrenheit menggunakan 32

50 - 32 = 18

Lalu, gunakan perbandingan ini

5/9 x n

Kita sudah memiliki "n" yaitu hasil pengurangan derajat Fahrenheit oleh 32

5/9 x n = 5/9 x 18 = 10

8. 5. Jika angka pada bilangan 100100100100100... diteruskan dengan pola yang sama, tentukan:a. Angka ke-100b. Angka ke-1000c. Angka ke-3000d. Angka ke-2016e. Banyak angka 1 hingga angka ke 50f. Banyak angka 0 hingga angka ke 102g. Banyak angka 1 hingga angka ke 300 h. Banyak angka 0 hingga angka ke 103


Jika angka pada bilangan 100100100100100......diteruskan dengan pola yg sama,

Pembahasan :

Bilangan tersebut selalu berulang tiap tiga angka
100 100 100 100 .....

Angka ke : 1, 4, 7, 10, 13, ... adalah 1
1, 4, 7, 10, 13, ... adalah bilangan jika dibagi 3 bersisa 1
Un = 3n - 2
Banyak angka 1 = 1, 2, 3, 4, 5, ..., n
Banyak angka 0 = 0, 2, 4, 6, 8, ..., (2n - 2)

Angka ke : 2, 5, 8, 11, 14, ... adalah 0
2, 5, 8, 11, 14, ... adalah bilangan jika dibagi 3 bersisa 2
Un = (3n - 1)
Banyak angka 1 = 1, 2, 3, 4, 5, ..., n
Banyak angka 0 = 1, 3, 5, 7, 9, ..., (2n - 1)

Angka ke : 3, 6, 9, 12, 15, .... adalah 0
3, 6, 9, 12, 15 adalah bilangan yang habis dibagi 3 (bersisa 0)
Un = 3n
Banyak angka 1 = 1, 2, 3, 4, 5, ..., n
Banyak angka 0 = 2, 4, 6, 8, 10, ..., 2n

Jadi berdasarkan penjelasan di atas :

a. Angka ke 100
100 : 3 = 33 bersisa 1 (33 × 3 = 99)
berarti angka ke 100 adalah 1 (karena bersisa 1)

b. Angka ke 1000
1000 : 3 = 333 bersisa 1 (333 × 3 = 999)
berarti angka ke 1000 adalah 1 (karena bersisa 1)

c. Angka ke 3000
3000 : 3 = 1000 bersisa 0 (3 × 1000 = 3000)
berarti angka ke 3000 adalah 0 (karena bersisa 0)

d. Angka ke 2016
2016 : 3 = 672 bersisa 0 (672 × 3 = 2016)
berarti angka ke 2016 adalah 0 (karena bersisa 0)

e. Banyak angka 1 hingga angka ke 50 adalah
50 : 3 = 16 bersisa 2 (16 × 3 = 48)
Un = 50
(3n - 1) = 50
3n = 51
n = 17
Banyak angka 1 = n = 17

f. Kemungkinan maksudnya adalah 10² bukan 102
Banyak angka 0 hingga angka ke 10²
100 : 3 = 33 bersisa 1 (33 × 3 = 99)
Un = 100
3n - 2 = 100
3n = 102
n = 34
Jadi banyak angka 0
= 2n - 2
= 2(34) - 2
= 68 - 2
= 66

Tetapi jika yang ditanya memang banyak angka 0 hingga angka ke 102
102 : 3 = 34 bersisa 0 (34 × 3 = 102)
Un = 102
3n = 102
n = 34
Jadi banyak angka 0
= 2n
= 2(34)
= 68

g. Banyak angka 1 hingga angka ke 300
300 : 3 = 100 bersisa 0 (100 × 3 = 300)
Un = 300
3n = 300
n = 100
Jadi banyak angka 1 = n = 100

h. Kemungkinan maksudnya adalah 10³ bukan 103
Banyak angka 0 hingga angka ke 10³
1000 : 3 = 333 bersisa 1 (333 × 3 = 999)
Un = 1000
3n - 2 = 1000
3n = 1002
n = 334
Jadi banyak angka 0
= 2n - 2
= 2(334) - 2
= 668 - 2
= 666

Tetapi jika yang ditanya tetap : banyak angka 0 hingga angka ke 103
103 : 3 = 34 bersisa 1 (34 × 3 = 102)
Un = 103
3n - 2 = 103
3n = 105
n = 35
Jadi banyak angka 0
= 2n - 2
= 2(35) - 2
= 70 - 2
= 68

20. 60C=...F 60C=...K 30R=...C 40R=...C 40R=...K 40C=...F 50F=...C

60C = 140 F

60C = ... K

K = 60 + 273 = 333
60C = 333 K

30R = ... C
C = 5/4 R
C = 5/4 x 30
C = 37,5
30R = 37,5 C

40R = ... C
C = 5/4 R
C = 5/4 x 40
C = 50
40R = 50 C

40R = ... K
K = (5/4 x R) + 273
K = (5/4 x 40) + 273
K = 50 + 273
K = 323
40R = 323 K

40C = ... F

F = (9/5 x 40) + 32
F = 72 + 32
F = 104
40C = 104 F

50F = ... C
C = 5/9 x (F - 32)
C = 5/9 x (50 - 32)
C = 5/9 x 18
C = 10
50F = 10 C

60+ 273= 333°K

5/4 × 30= 37,5°C

4.) °R - °C= (5/4) °R= ....°C
5/4 × 40= 50°C

5.) °R - °K=  C + 273 = (5/4) R + 273
=50 + 273= 323°K

=((9/5) × 40) + 32
