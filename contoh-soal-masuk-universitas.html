4. Soal latihan UM UGM Ujian Masuk Universitas Gadjah Mada


sin t = 1/a - 1/b
sin²t = (1/a - 1/b)²
sin²t = (1/a² + 1/b² - 2/ab)

cos t = 1/a + 1/b
cos²t = (1/a + 1/b)²
cos²t = (1/a² + 1/b² + 2/ab)

sin²t + cos²t = (1/a² + 1/b² - 2/ab) +  (1/a² + 1/b² + 2/ab)
1 = (2/a² + 2/b²)
1 = (2b² + 2a²)/ab²
(ab)² = 2(a² + b²)
a² + b² = (ab)²/2
a² + b² = (a²b²)/2

23. Soal latihan UM UGM Ujian Masuk Universitas Gadjah Mada


U5/U10 = (a + 4b)/(a + 9b)

4(U10/U5) = 4((a + 9b)/(a + 4b))
= (4a + 36b)/(a + 4b)

S5/S10 = 2/3
(a + a + b + a + 2b + a + 3b + a + 4b)/(a + a + b + a + 2b + ... + a + 9b) = 2/3

(5a + 10b)/(10a + 45b) = 2/3
3(5a + 10b) = 2(10a + 45b)
15a + 30b = 20a + 90b
20a - 15a = 30b - 90b
5a = -60b
a = -12b

U5/U10 = (a + 4b)/(a + 9b)
= (-12b + 4b)/(-12b + 9b)
= -8b/-3b
= 8/3

4(U10/U5) = (4a + 36b)/(a + 4b)
= (4(-12b) + 36b)/(-12b + 4b)
= (-48b + 36b) / (-8b)
= (-12b)/(-8b)
= 3/2

log(U5/U10 - 4(U10/U5)) = log(8/3 - 3/2)
= log((16 - 9)/6)
= log(7/6)
= log7 - log6

30. Soal latihan UM UGM Ujian Masuk Universitas Gadjah Mada


x - 3 = x'
x = x' + 3

f(x) = (x + 3 - 6)/(x + 3 + 3)
f(x) = (x - 3)/(x + 6)
f(x) = 1/2

(x - 3)/(x + 6) = 1/2
2(x - 3) = x + 6
2x - 6 = x + 6
2x - x = 6 + 6
x = 12

f(12) = 1/2
f⁻¹(1/2) = 12

35. APA Yang kalian lakukanb pada saat masuk kedalam universitas


Jawaban:

Penjelasan:

1. Keliling area fakultas dan kampus, lalu hafalkan letak gedung dan lokasi sarana umum lainnya

Kegiatan yang pertama ini wajib untuk kalian lakukan ketika pertama masuk kuliah. Mengapa demikian? Agar nantinya kalian tak perlu bingung lagi ketika memiliki urusan dengan pihak kampus dan mencari lokasinya. Umumnya kampus memiliki luas yang cukup besar sehingga kamu pun tak akan mudah untuk menghapalnya jika kamu tidak mengelilinginya.

2. Mencari tahu nama panggilan khas untuk lokasi sarana umum tertentu di kampus

Kegiatan yang kedua ini kamu lakukan agar kamu dapat lebih mudah untuk menghapal lokasi-lokasi penting di kampus. Biar serasa kayak anak kampus sini banget gitu kalau kata anak muda. Jadi kalian pun tak perlu bingung ketika harus mencari nama lokasi di kampus yang mungkin sebelumnya terdengar cukup asing ditelinga.

3. Mengunjungi sub bagian akademik kampus untuk tanya-tanya seputar regulasi formal perkuliahan

Nah, untuk kegiatan yang satu ini, jangan lupa untuk mengakrabkan diri dengan para pegawai sub akademiknya juga ya! Agar nantinya kamu juga cukup mudah mengurus urusan-urusan lainnya yang berhubungan dengan perkuliahan.

4. Pelajari istilah-istilah asing perkuliahan yang super banyak

Umumnya di jenjang perkuliahan kalian akan menemukan berbagai istilah asing yang sering dipergunakan oleh dosen dan para mahasiswa.

6. Susun catatan kuliah sesuai dengan tipe dosen kamu

Mengapa harus mencatat serta menyusunnya dengan tipe dosen? Karena biasanya beda dosen juga akan berbeda juga dengan teknik mengajar. Perlu kamu ketahui bahwa tidak semua materi yang disampaikan bisa dicatat dengan dengan metode yang sama.

7. Mencari kost dekat kampus

Buat kamu yang memutuskan untuk merantau dan tinggal di kost dekat kampus ugm ataupun sewa apartemen dekat uny, ada baiknya kamu memutuskan terlebih dahulu lokasi tempat tinggal kamu ini. Mencari kost Jogja murah bukanlah hal yang mudah, terkadang butuh berhari-hari untuk sekedar mencari informasi mana-mana saja kost yang sesuai. Oleh karena itu lakukanlah kegiatan ini di minggu pertama kamu masuk kuliah ya! Atau jika ada yang ingin melakukannya sebelum masuk kuliah juga akan jauh lebih baik lagi.

8. Khusus anak kosan, cari tahu informasi seputar tempat makan, laundry, dan fotokopian terdekat di sekitar kosan

Karena anak kost selalu diidentikkan dengan mandiri, tentu tempat-tempat tersebut sangat berpegaruh dalam menunjang kehidupan kalian selama kuliah di kota orang. Tempat makan akan menjadi tempat dimana kalian mengisi energi setiap harinya, sedangkan laundry akan menjadi tempat dimana kalian mengantar pakaian kotor ketika kalian tidak memiliki cukup waktu untuk mencucinya.

9. Gali informasi baru seputar kampus

Sebagai mahasiswa baru, kamu diharuskan lebih aktif mencari tau terkait informasi mahasiswa baru. Kapan jadwal masa orientasi, apa saja yang perlu dibawa saat orientasi, penting juga kamu mencari tau organisasi apa saja yang ada di kampusmu tersebut. Jadi saat masuk kuliah, kamu sudah punya banyak informasi dan tinggal memilih mau masuk organisasi yang mana.
