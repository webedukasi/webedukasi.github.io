5. Oi Oipersamaan garis g adalah​


Jawab:

3x - 4y - 48 = 0

Penjelasan dengan langkah-langkah:

Persamaan garis g melewati

(16, 0) dan (0, -12)

x1 = 16, y1 = 0

x2 = 0, y2 = -12

maka

m = (y2-y1)/(x2-x1)

m = (-12-0)/(0-16)

m = -12/-16

m = 3/4

------------

Pers. garis

y-y1 = m(x-x1)

y-0 = (3/4)(x-16)

y = (3/4)x - 48/4

4y = 3x - 48

3x - 4y - 48 = 0

17. Oi Oi bantu kk jagoo​


Jawaban:

Himpunan =

A - B = Selisih Himpunan

A n B = Irisan Himpunan

A u B = Gabungan Himpunan

Soal :

P = {2,3,4,5,6}

Q = {3,5,7}

R = {4,5,6,7}

Tentukan

a. (P - Q) U R

b. (P - Q) n R

Penyelesaian :

P = {2,3,4,5,6}

Q = {3,5,7}

R = {4,5,6,7}

(P - Q) U R

(2,4,6) U R

=> (P - Q) U R = {2,4,5,6,7}

(P - Q) n R

(2,4,6) n R

=> (P - Q) n R = {4,6}

25. Oi Oi digambar yoo jawab​


Persamaan Garis Lurus

_________

2x - y + 5 = 0

a = 2

b = -1

m = - a/b = - 2/-1

m = 2

[tex] \\ [/tex]

Karena tegak lurus, maka hasil kali gradien kedua garis adalah -1

m¹ × m² = -1

2 × m² = -1

m² = -½

[tex] \\ [/tex]

Persamaan garis lurus yang melalui titik ( 2, -1 ) dan bergradien -½

( 2, -1 ) ====> ( x¹, y¹ )

y - y¹ = m( x - x¹ )

y - ( -1 ) = -½( x - 2 )

y + 1 = -½x + 1

y = -½x + 1 - 1

y = -½x | ×2

2y = -x

[tex] \boxed{ \sf x + 2y = 0}[/tex]

______________

Jadi, persamaan garis yang melalui titik ( 2, -1 ) dan tegak lurus dengan persamaan 2x - y + 5 = 0 adalag

x + 2y = 0
