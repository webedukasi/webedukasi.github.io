1. Aturan konversi desimal (basis 10) ke biner (basis 2):

- bagi bilangan desimal dg angka 2

- tulis hasilnya & sisa baginya

- hentikan pembagian bila hasilnya 1

- penulisan konversi: dari bawah (hasil) ke atas (sisa bagi)

- bagi bilangan desimal dg angka 8

- tulis hasilnya & sisa baginya

- hentikan pembagian bila hasilnya 7 atau kurang dari 7

- penulisan konversi: dari bawah (hasil) ke atas (sisa bagi)

Cara perhitungannya bisa dilihat di lampiran (di sebelah tabel sistem bilangan)

3. Aturan konversi desimal (basis 10) ke heksadesimal (basis 16):

- bagi bilangan desimal dg angka 16

- tulis hasilnya & sisa baginya

- hentikan pembagian bila hasilnya 15 atau kurang dari 15

- penulisan konversi: dari bawah (hasil) ke atas (sisa bagi)

Cara perhitungannya bisa dilihat di lampiran (di bawah tabel sistem bilangan)

Cermatilah bahwa 14 (desimal) = E (heksadesimal)

4. Perhitungan konversi biner (basis 2) ke desimal (basis 10):

111011 = 1x2⁵+1x2⁴+1x2³+0x2²+1x2¹+1x2⁰ = 32 + 16 + 8 + 0 + 2 + 1 = 59

Jadi 111011 (biner) = 59 (desimal)

6. konversikan bilangan biner 10111 ke bilangan desimal​


Jawaban:

Bilangan biner 10111 dapat dikonversikan menjadi bilangan desimal dengan menggunakan metode perpangkatan 2. Setiap digit pada bilangan biner merepresentasikan nilai 2 pangkat n, dimulai dari nol pada digit paling kanan.

Maka, untuk bilangan biner 10111:

Digit paling kanan merepresentasikan nilai 2 pangkat 0 = 1

Digit kedua dari kanan merepresentasikan nilai 2 pangkat 1 = 2

Digit ketiga dari kanan merepresentasikan nilai 2 pangkat 2 = 4

Digit keempat dari kanan merepresentasikan nilai 2 pangkat 3 = 8

Digit kelima dari kanan merepresentasikan nilai 2 pangkat 4 = 16

Jumlahkan nilai-nilai pangkat 2 dari masing-masing digit yang bernilai 1:

1 + 2 + 16 = 19

Maka, bilangan biner 10111 setara dengan bilangan desimal 19.

18. -Konversi bilangan desimal ke biner,N: 1353 -Konversi bilangan desimal ke oktal, N 4355 -Konversi bilangan biner ke desimal,N 1000101010 -Konversi bilangan oktal ke desimal, N: 6245Mohon bantuannya yang bisa.​


cara 1:

Tentu, saya akan membantu Anda dengan konversi bilangan tersebut.

1. Konversi bilangan desimal ke biner (Decimal to Binary):

  Bilangan desimal N = 1353

  Langkah-langkah:

  1. Bagi bilangan desimal N dengan 2.

  2. Catat sisa pembagian sebagai digit biner.

  3. Bagi hasil pembagian dengan 2 lagi, dan ulangi langkah 2, hingga hasil pembagian menjadi 0.

  Mari kita lakukan konversi ini:

  1353 / 2 = 676 sisa 1

  676 / 2 = 338 sisa 0

  338 / 2 = 169 sisa 0

  169 / 2 = 84 sisa 1

  84 / 2 = 42 sisa 0

  42 / 2 = 21 sisa 0

  21 / 2 = 10 sisa 1

  10 / 2 = 5 sisa 0

  5 / 2 = 2 sisa 1

  2 / 2 = 1 sisa 0

  1 / 2 = 0 sisa 1

  Jadi, bilangan biner untuk 1353 adalah 10101011001.

2. Konversi bilangan desimal ke oktal (Decimal to Octal):

  Bilangan desimal N = 4355

  Langkah-langkah:

  1. Bagi bilangan desimal N dengan 8.

  2. Catat sisa pembagian sebagai digit oktal.

  3. Bagi hasil pembagian dengan 8 lagi, dan ulangi langkah 2, hingga hasil pembagian menjadi 0.

  Mari kita lakukan konversi ini:

  4355 / 8 = 544 sisa 3

  544 / 8 = 68 sisa 0

  68 / 8 = 8 sisa 4

  8 / 8 = 1 sisa 0

  1 / 8 = 0 sisa 1

  Jadi, bilangan oktal untuk 4355 adalah 10343.

3. Konversi bilangan biner ke desimal (Binary to Decimal):

  Bilangan biner N = 1000101010

  Langkah-langkah:

  1. Tulis bilangan biner.

  2. Mulai dari digit paling kanan, kalikan setiap digit dengan 2 pangkat posisinya (dimulai dari pangkat 0) dan jumlahkan.

  Mari kita lakukan konversi ini:

  1 * 2^9 + 0 * 2^8 + 0 * 2^7 + 0 * 2^6 + 1 * 2^5 + 0 * 2^4 + 1 * 2^3 + 0 * 2^2 + 1 * 2^1 + 0 * 2^0

  = 512 + 0 + 0 + 0 + 32 + 0 + 8 + 0 + 2 + 0

  = 554

  Jadi, bilangan desimal untuk 1000101010 adalah 554.

4. Konversi bilangan oktal ke desimal (Octal to Decimal):

  Bilangan oktal N = 6245

  Langkah-langkah:

  1. Tulis bilangan oktal.

  2. Mulai dari digit paling kanan, kalikan setiap digit dengan 8 pangkat posisinya (dimulai dari pangkat 0) dan jumlahkan.

  Mari kita lakukan konversi ini:

  5 * 8^0 + 4 * 8^1 + 2 * 8^2 + 6 * 8^3

  = 5 + 32 + 128 + 3072

  = 3237

  Jadi, bilangan desimal untuk 6245 adalah 3237.

Semoga penjelasan di atas membantu! Jika Anda memiliki pertanyaan lebih lanjut, jangan ragu untuk bertanya.

Cara 2:

1.

Konversi bilangan desimal ke biner, N = 1353:

   Langkah pertama, bagi angka desimal (1353) dengan 2 dan catat sisa pembagian.    Bagi hasil pembagian sebelumnya lagi dengan 2 dan catat sisa pembagiannya.    Ulangi langkah ini hingga hasil bagi menjadi 0.

   Balik urutan sisa-sisanya untuk mendapatkan representasi binernya.

Hasil: Bilangan desimal 1353 dalam bentuk biner adalah 10101010001.

2.

Konversi bilangan desimal ke oktal, N = 4355:

   Langkah pertama, bagi angka desimal (4355) dengan 8 dan catat sisa pembagian.    Bagi hasil pembagian sebelumnya lagi dengan 8 dan catat sisa pembagiannya.    Ulangi langkah ini hingga hasil bagi menjadi 0.

   Balik urutan sisa-sisanya untuk mendapatkan representasi oktalnya.

Hasil: Bilangan desimal 4355 dalam bentuk oktal adalah 10353.

3. Konversi bilangan biner ke desimal, N = 1000101010: Untuk mengkonversikan bilangan biner ke desimal cukup mudah! Kamu tinggal menjumlahkan nilai setiap bit yang bernilai satu (1), dikalikan dua pangkat posisi bit tersebut dimulai dari kanan.

Jadi, 1 * (2^9) + 0 * (2^8) + 0 * (2^7) + 0 * (2^6) + 1 * (2^5) + 0 * (2^4) + 1 * (2^3) + 0 * (2^2) + 1 * (2^1) + 0 * (2^0)

Hasil: Bilangan biner 1000101010 dalam bentuk desimal adalah 682.

4.   Konversi bilangan oktal ke desimal, N = 6245: Untuk mengkonversikan bilangan oktal ke desimal, kamu cukup mengalikan setiap digit dengan delapan pangkat posisinya dimulai dari kanan.

   Jadi, 6*(8³)+ 2*(8²)+ 4*(8¹)+ 5*(8⁰)

   Hasil: Bilangan oktal 6245 dalam bentuk desimal adalah 3325.

Gitu aja penjelasannya! Kalau ada pertanyaan lain atau butuh bantuan lagi, gue siap sedia kok!

Langkah-langkahnya adalah sebagai berikut:

Mulai dari digit paling kanan (paling rendah), atribusikan nilai dari tiap digit oktal sesuai dengan pangkat 8nya.
