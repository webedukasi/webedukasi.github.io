1. Jika sebuah rangkaian NE555 menggunakan R1 = 15k, R2 = 15k, R3 = 1,5k C = 10uF. Catu daya dipakai tegangan 5 volt, maka 1. Hitung berapa T-on dan T-off!​


Jawaban:

Untuk menghitung T-on (waktu hidup) dan T-off (waktu mati) pada rangkaian NE555, dapat digunakan rumus berikut:

T-on = 0.693 * (R1 + R2) * C

T-off = 0.693 * R2 * C

Dengan menggunakan nilai R1 = 15k Ohm, R2 = 15k Ohm, dan C = 10uF, maka:

T-on = 0.693 * (15000 + 15000) * 10e-6 = 0.693 * 30000 * 10e-6 = 208.7ms

T-off = 0.693 * 15000 * 10e-6 = 104.35ms

Jadi, T-on adalah sekitar 208.7ms dan T-off adalah sekitar 104.35ms.

21. Jika sebuah rangkaian NE555 menggunakan R1 = 15k, R2 = 15k, R3 = 1,5k C = 10uF. Catu daya dipakai tegangan 5 volt, maka 1. Hitung berapa T-on dan T-off ! 2. Hitung arus yang mengalir ke LED saat menyala/on !3. jika terjadi kesalahan pemasangan,tertukar antara R2 dan R3,terjadi perubahan nilai T-on dan T-off hitunglah !4. kemudian karena kesalahan tersebut,nyala LED menjadi lebih redup,jelaskan dengan perhitungan penyebabnya !​


Jawaban:

[1]

T-on = 0.693 * (R1 + R2) * C = 0.693 * (15k + 15k) * 10uF = 0.693 * 30k * 10uF = 20.79ms

T-off = 0.693 * (R2) * C = 0.693 * 15k * 10uF = 10.395ms

[2]

Arus yang mengalir ke LED saat menyala adalah sekitar 5mA karena NE555 mengatur arus pada beban sehingga arus ke LED selalu stabil.

[3]

Jika R2 dan R3 tertukar:

T-on = 0.693 * (R1 + R3) * C = 0.693 * (15k + 1.5k) * 10uF = 1.959ms

T-off = 0.693 * (R3) * C = 0.693 * 1.5k * 10uF = 0.648ms

[4]

Jika LED menjadi lebih redup, hal ini mungkin disebabkan oleh beberapa hal seperti kapasitas kondensator yang tidak sesuai, beban yang tidak sesuai, atau voltase yang rendah. Salah satu hal yang dapat menyebabkan LED redup adalah kapasitas kondensator yang lebih kecil dari yang diperlukan. Kondensator ini menentukan periode off dalam siklus sinyal. Jika kapasitas kondensator lebih kecil, periode off akan lebih pendek sehingga arus yang diterima LED akan lebih rendah.

Jawaban:

The original number of $5 coins is 6.The original number of $2 coins is 4.

Penjelasan dengan langkah-langkah:

Number of $5 coins = x

Number of $2 coins = y

Total value is $38:

5x + 2y = 38 ... (1)

If two $5 coins used, then the number of $5 and $2 coins will be the same:

y = x - 2 ... (2)

Substitute the value of y to equation (1):

5x + 2y = 38

5x + 2(x-2) = 38

5x + 2x - 4 = 38

7x - 4 = 38

7x = 38 + 4

7x = 42

x = 42/7

x = 6

y = x-2

y = 6-2

y = 4

Therefore:

The original number of $5 coins is 6.The original number of $2 coins is 4.

Discovered:

Variable x = number of $2 coins

Variable y = number of $2 coins

Linear equation of two variables (1) = ... (1)

Asked:

 Original number of coins = ?

  y = x - 2 ... (2)

 1.1. Search value of variable x

 1.2. Search value of variable y

  [tex]y = x - 2\\y = 6 - 2\\y = 4\\\\ or\\\\5x + 2y = 38\\(5 \times 6) + 2y = 38\\30 + 2y = 38\\2y = 38 - 30\\y = 4[/tex]

38. Nick, Linda and Cindy have 16 780 coins altogether. Linda has 2860 fewer coins than Nick. Cindy has three times as many coins as Nick. How many coins does Cindy have?​


Let's represent the number of coins each person has with variables:

Let N be the number of coins Nick has.

L be the number of coins Linda has.

C be the number of coins Cindy has.

We are given three pieces of information:

1. Nick, Linda, and Cindy have 16,780 coins altogether, which can be expressed as:

N + L + C = 16,780

2. Linda has 2,860 fewer coins than Nick, which can be expressed as:

L = N - 2,860

3. Cindy has three times as many coins as Nick, which can be expressed as:

C = 3N

Now, we can set up a system of equations using these relationships:

1. N + L + C = 16,780

2. L = N - 2,860

3. C = 3N

We can use substitution to solve this system. First, we'll substitute L and C from equations 2 and 3 into equation 1:

N + (N - 2,860) + 3N = 16,780

Now, combine like terms:

5N - 2,860 = 16,780

Add 2,860 to both sides:

5N = 16,780 + 2,860

5N = 19,640

Now, divide by 5:

N = 19,640 / 5

N = 3,928

So, Nick has 3,928 coins. Now, we can find how many coins Cindy has:

C = 3N

C = 3 * 3,928

C = 11,784

Cindy has 11,784 coins.

Jawaban:

T-on = 0.693 * (R1 + R2) * C = 0.693 * (15k + 15k) * 10uF = 0.693 * 30k * 10uF = 20.79ms
