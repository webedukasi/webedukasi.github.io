Jawaban:

Aluc@rd = 10²

=10² × 2

Dit:

Aluc@rd + H@yabusa ?

Jawab:

Aluc@rd + H@yabusa

=10² + 10² × 2

=(10 × 10) + (10 × 10) × 2

=100 + 100 × 2

=100 + 200

=300

14. kelebihan dan kekurangan RD RAM dan SD RAM


Jawaban:

RDRAM lebih baik daripada SDRAM

Penjelasan:

SDRAM Memiliki kemampuan setingkat di atas EDORAM

Slot memory untuk SDRAM adalah 168 Pin

Bentuk SDRAM adalah Dual Inline Memory Modul (DIMM)

Memiliki kecepatan di atas 100 Mhz, sedangkan

Memiliki kecepatan sangat tinggi hingga 800 Mhz

Slot memory untuk RDRAM adalah 184 Pin

Bentuk RDRAM adalah Rate Inline Memory Modul (RIMM)

Pertama kali digunakan untuk computer dengan processor Pentium 4

Jadi kesimpulan saya RDRAM lebih enak daripada SDRAM
