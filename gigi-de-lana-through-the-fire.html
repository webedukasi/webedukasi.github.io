28. ( + ) the ratings extinguished the fire ( - )( ? )​


Jawaban:( + ) the ratings extinguished the fire ( - ) the ratings did not extinguish the fire ( ? ) Did the ratings extinguish the fire?

PEMBAHASAN

Kalimat Positive (+) pada soal menggunakan Verb-2 “extinguished” setelah subject “the ratings”

maka kalimat tersebut Past Tense.

PAST TENSE adalah tenses untuk menyatakan Kegiatan yg terjadi di masa lalu. Rumusnya:

(+) S + V2 + Complement (-) S + did not + V1 + complement(?) Did + S + V1 + complement?
