14. Bantu+terjemahanya!!​


Jawaban dan Penjelasan:

よる てれび を みます  = yoru terebi o mimasu = malam hari nonton tvひる ごはん を たべます= hiru gohan o tabemasu = siang hari makan nasiあさ せんたく を します = asa sentaku o shimasu= pagi hari mencuci pakaianよる にほん-ご を べんきょうします= yoru nihon-go o benkyoushimasu = malam hari belajar bahasa jepangあさ かお を あります = asa kao o arimasu = pagi hari mencuci muka

pola kalimatnya pasti waktu + Kegiatan Sehari-Hari (kosakata ket kerja)

KB (benda) を KK (bentuk masu) します

waktu

あさ (asa) = pagi ひる (hiru) = siang  よる (yoru) = malamゆうがと (yuugato) = sore

Kegiatan Sehari-Hari (kosakata ket kerja)

かおを  あらいます (kao o araimasu) = mencuci muka はを  みがいます (ha o agaimasu) = menyikat gigi みずを  あびます (mizu o abimasu) = mandi ごはんを  たべます (gohan o tabemasu) = makan nasi こうちゃを  のみます (koucha o nomimasu) = minum teh しんぶんを  よみます (shinbun o yomimasu) = membaca koran おんがくを  ききます (ongaku o kikimasu) = mendengarkan musik てがみを  かきます (tegami o kakimasu) = menulis surat テレビを  みます (terebi o mimasu) = menonton TV しゅくだいを  します (shukudai o shimasu) = mengerjakan PR おいのりを  します (oinori o shimasu) = berdoa, ibadah せんたくします (sentaku shimasu) = mencuci (pakaian)

26. Minta Bantuannya+Terjemahanya!​


Nomor 2

a. Hiru ( Shinbun o yomimasu )

   a : 昼は新聞を読みますか？

        [ Hiru wa shinbun wo yomimasu ka ]

　b : はい、読みます。

         [ Hai, yomimasu ]

b. Rina, asa ( Souji o shimasu )

    a : リナさん、朝は部屋掃除をしますか？

          [ Rina-san, asa wa heya wo souji wo shimasu ka }

    b : はい、掃除します。

          [ Hai, souji shimasu ]

c. Yoru ( Ongaku o kikimasu )

   a : 夜はいつも音楽を聴きますか？

        [ Yoru wa itsumo ongaku wo kikimasu ka ]

   b : いいえ、音楽を聴きません。

        [ iie, ongaku wo kikimasen ]

d.  Asa ( koucha o nomimasu )

    a : 朝は紅茶を飲みますか？

         [ Asa wa koucha wo nomimasu ka ]

    b : いいえ、コヒーを飲みます。

         [ iie , kohii wo nomimasu ]

Translate :

a. Hiru ( Shinbun o yomimasu )

   a : Apakah anda membaca koran di siang hari?

   b : Iya, saya membacanya.

b. Rina, asa ( Souji o shimasu )

   a : Apakah Rina membersihkan kamarnya pagi ini?

   b : Iya, saya memnersihkannya.

c. Yoru ( Ongaku o kikimasu )

   a : Apakah anda selalu mendengarkan musik di malam hari?

   b : Tidak, saya tidak mendengarkan musik di malam hari.

d . Asa ( koucha o nomimasu )

    a : Apakah anda minum teh di pagi hari

    b : Tidak, saya minum kopi.

Nomor 3

a. シャワー浴びますそれから新聞を読みます。

   [ Shawa wo abimasu sorekara shinbun wo yomimasu ]

b. 宿題をしますそれからテレビを見ます。

   [ Shukudai wo shimasu sorekara terebi wo mimasu ]

c. 顔を洗いますそれから紅茶を飲みます。

  [ Kao wo araimasu sorekara koucha wo nomimasu ]

d. 歯を磨きますそれからご飯を食べます。

  [ Ha wo migakimasu sorekara gohan wo tabemasu ]

Translate :

a. Mandi setelah itu  membaca koran.

b. Mengerjakan tugas setelah itu menonton TV.

c. Membersihkan wajah setelah itu meminum teh.

d. Menyikat gigi setelah itu sarapan.

Nomor 4

私はいつも5時におきます。それからシャワーを浴びて、歯を磨いて、ご飯を食べます。学校に行く前に本を準備すること、後は学校に行きます。
