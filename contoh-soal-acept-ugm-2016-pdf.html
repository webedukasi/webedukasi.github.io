Soal ini lebih mudah diselesaikan dengan mensubstitusikan pilihan jawaban ke soal dan mengeliminasi jawaban yang salah:

Ambil x = 2, maka diperoleh:

3√(2 - 2)/ 2 - 1 < 2 ⇒ 0 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 2 salah sehingga pilihan jawaban A, B, E salah.

Kemudian ambil x = 0, maka diperoleh:

3√(2 - 0)/ 0 - 1 < 2 ⇒ 3√2/-1 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 0 salah sehingga pilihan jawaban D salah.
Jadi, jawaban yang benar adalah C.

Larutan NaOH 0.05 M dgn vol. = 2 L

Vol. NaOH 8% yg dibutuhkan dan jumlah air yg ditambahkan ???

Pembahasan:

Mr NaOH = 40

1. Hitung mol NaOH dlm 2L 0.05 M

NaOH: 2 L × 0.05 mol/L = 0.1 mol

2. Jumlah NaOH ini (pd no. 1) berasal dari x mL NaOH 8% (%berat/vol) dan penambahan y mL air, sehingga

(x mL × 1.25 g/mL × 8%)/40 g/mol = 0.1 mol

didapatkan x = 40 mL dlm 2000 mL air

Jadi air yg ditambahkan:
2000 - 40 = 1960 mL

3. 3x²+x-3 = 0 => akar : α dan β

α+β = -b/a

      = -1/3

αβ = c/a

     = (-3)/3 = -1

Akar persamaan yang baru :

c = 2+(1/(α+1)) , d = 2+(1/(β+1))

rumus PK baru :

x² - (c+d)x + c.d = 0

c+d = 2+(1/(α+1)) + 2+(1/(β+1))

     = 4 + ( (β+1)+(α+1) )/( (α+1)(β+1) )

     = 4 + (2+(α+β) )/( αβ+1+(α+β) ) <= α+β = -1/3 , αβ = -1

    = 4 + (2 - 1/3 )/( -1+1 - 1/3 )

    = 4 + (5/3)/(-1/3)

   = 4 - 5

c+d = -1

cd = (2+(1/(α+1)) ).(2+(1/(β+1)) )

     = 4 + 2.(1/(β+1)) + 2.(1/(α+1)) + (1/(α+1)).(1/(β+1))

     = 4 + 2.( (1/(β+1)) + (1/(α+1)) ) + 1/( (α+1).(β+1) )

     = 4 + 2.( ( (α+1)+(β+1) )/( (α+1).(β+1) ) ) + 1/( (α+1).(1/(β+1) ) <= (α+1).(β+1) = -1/3 , (α+1)+(β+1) = 5/3

    = 4 + 2.(5/3)/(-1/3) + 1/(-1/3)

    = 4 + 2.(-5) - 3

    = 4 - 10 - 3

c.d = -11

Persamaannya adalah :

x² - (-1)x + (-11) = 0

x² + x - 11 = 0 (A)

4. x² - (3p-2)x + 2p+8 = 0  => akarnya x₁ dan x₂

x₁ + x₂ = - (-(3p-2))/1

x₁ + x₂ = 3p - 2

x₁.x₂ = (2p+8)/1 = 2p+8

x₁,p,x₂ membentuk barisan geometri

r = p/x₁ = x₂/p

    p² = x₁.x₂

     p² = 2p+8

     p²-2p-8 = 0

  (p-4)(p+2) = 0

p = 4 atau -2

di soal dibilang bahwa p positif, maka :

p = 4

x₁ + x₂ = 3p - 2 <= p = 4

x₁ + x₂ = 3.4 - 2

x₁+x₂ = 10

x₁+x₂+p = 10 + 4 = 14 (E)
