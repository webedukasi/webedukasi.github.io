8. 1. Misalkan f(x) suatu fungsi yang kontinu pada domain R, jika aeR dan f(x) diferensiabel pada a. Buktikan bahwa f kontinu pada a 2. Berikan 2 contoh fungsi yang kontinu tetapi tidak diferensiabel. Jelaskanlah kebenaran contoh yang Anda ambil.​


Jawaban:

1. Untuk membuktikan bahwa f kontinu pada a, kita perlu menunjukkan bahwa batasan dari f(x) saat x mendekati a adalah f(a).

Karena f(x) diferensial pada a, kita dapat menggunakan definisi diferensial untuk menunjukkan bahwa batasan f(x) saat x mendekati a adalah f(a). Definisi diferensial adalah:

f'(a) = lim (x -> a) [f(x) - f(a)] / (x - a)

Karena f(x) kontinu pada domain R, maka kita dapat membagi kedua sisi persamaan ini dengan (x - a) dan mengambil batas saat x mendekati a:

lim (x -> a) [f(x) - f(a)] / (x - a) = f'(a)

Karena f'(a) ada dan f(x) kontinu pada domain R, maka batasan dari f(x) saat x mendekati a adalah f(a). Oleh karena itu, f kontinu pada a.

2. Contoh pertama: Fungsi Heaviside

Fungsi Heaviside, yang biasanya dilambangkan sebagai H(x), didefinisikan sebagai:

H(x) =

0, x < 0

1, x >= 0

Fungsi ini kontinu di semua titik kecuali pada x = 0, di mana terjadi "loncatan" dari 0 menjadi 1. Tidak ada turunan yang dapat didefinisikan pada x = 0, sehingga fungsi Heaviside tidak diferensial di titik itu.

Contoh kedua: Fungsi mutlak

Fungsi mutlak, yang biasanya dilambangkan sebagai |x|, didefinisikan sebagai:

|x| =

x, x >= 0

-x, x < 0

Fungsi ini kontinu di semua titik kecuali pada x = 0, di mana terjadi "sudut tajam" di grafiknya. Turunan tidak terdefinisi di titik itu, sehingga fungsi mutlak tidak diferensial di x = 0.

Dalam kedua contoh ini, fungsi-fungsi tersebut memiliki diskontinuitas pada titik tertentu yang menghentikan diferensialitas mereka di titik tersebut.

23. dalam tes kenaikan pangkat suatu perusahaan diberikan 3 nomor soal yang masing masing jawabannya benar atau salah. jika X menyatakan jawaban yang benar dari 3 nomor soal tersebut, buatlah distribusi probabilitas variabel acak X


Distribusi Probabilitas adalah suatu distribusi yang mengambarkan peluang dari sekumpulan varian sebagai pengganti frekuensinya

Distribusi Probabilitas adalah perhitungan matematika yang menggabungkan Statistika dan Peluang / Probabilitas

Distribusi Probabilitas Binomial atau Distribusi Bernoulli adalah suatu distribusi teoritis yang menggunakan variabel random diskrit yang terdiri dari dua kejadian yang berkomplemen, contohnya : benar - salah, ya - tidak

Ciri-ciri Distribusi Binomial adalah :

Setiap percobaan hanya memiliki dua peristiwa, seperti benar - salah, ya - tidakProbabilitas suatu peristiwa adalah tetap, tidak berubah untuk setiap percobaan. Percobaannya bersifat independen, atau dengan pengembalian

Rumus Distribusi Binomial :

[tex]P(X=x) = C_x^n.P^x.Q^{n-x}[/tex]

Dimana :

[tex]C_x^n = \frac{n!}{x!(n-x)!}[/tex]

P(X=x) = peluang kejadian yang diharapkan

n = banyaknya kejadian

40. Sekeping uang logam didesain sedemikian rupa sehingga probabilitas muncul sisi angka dua kali dari probabilitas muncul sisi gambar. Variabel acak x menyatakan banyak kejadian muncul sisi angka. Tentukan distribusi probabilitas, fungsi probabilitas


Sekeping uang logam didesain sedemikian rupa sehingga probabilitas muncul sisi angka dua kali dari probabilitas muncul sisi gambar. Variabel acak x menyatakan banyak kejadian muncul sisi angka dan n adalah banyaknya percobaan.

Maka Distribusi probabilitasnya adalah

P(X = 0) = (1/3)ⁿ

P(X = 1) = n (1/3)ⁿ⁻¹ ( 2/3 )

P(X = 2) = [ n(n-1)/2 ] (1/3)ⁿ⁻² ( 2/3 )²

dan seterusnya....


Fungsi probabilitasnya adalah

P ( X = a ) = ⁿCₐ ( 2/3 )ᵃ ( 1/3 )ⁿ⁻ᵃ


PEMBAHASAN

Peluang suatu kejadian di definisikan sebagai kemungkinan terjadinya suatu kejadian terhadap segala kemungkinan yang ada ( ruang sampel ).

[tex]\textbf{Peluang = banyak kejadian yang di maksud / ruang sampel}[/tex]

Kita misalkan percobaannya adalah melempar sebuah uang koin, maka ruang sampelnya adalah munculnya Gambar(G) atau Angka(A) pada koinnya.  Jadi :

Peluang munculnya Gambar P(G) = 1/2

Peluang munculnya Angka P(A) = 1/2

Sekarang kalau percobaannya adalah melempar 2 buah uang koin , maka ruang sampelnya adalah munculnya Gambar-Gambar (GG), Gambar-Angka (GA) , Angka-Gambar (AG) , dan Angka-Angka(AA). Jadi :

Peluang munculnya 2 Gambar P (2G) = (1/2)² =1/4

Peluang munculnya 1 Gambar P (1G) = 2 x (1/2) x (1/2) = 2/4

Peluang munculnya 0 Gambar P (0G) = (1/2)² =1/4

Sekarang kalau percobaannya adalah melempar 3 buah uang koin , maka dengan cara yang sama akan di dapatkan :

Peluang munculnya 3 Gambar P (3G) = (1/2)³ = 1/8

Peluang munculnya 2Gambar P (1G) = 3 x (1/2)² (1/2) = 3/8

Peluang munculnya 1 Gambar P (0G) = 3 x (1/2)² (1/2) = 3/8

Peluang munculnya 0 Gambar P (0G) = (1/2)³ = 1/8

Distribusi probabilitas seperti ini di sebut distribusi binomial yang memiliki rumus :

P ( X = a ) = ⁿCₐ ( p )ᵃ ( 1 - p )ⁿ⁻ᵃ

Okay lah dengan menggunakan prinsip ini kita akan menyelesaikan soal yang ditanyakan.

Sekeping uang logam didesain sedemikian rupa sehingga probabilitas muncul sisi angka dua kali dari probabilitas muncul sisi gambar , maka bisa di tuliskan sebagai berikut :

P(A) = 2 x P(G)

Sedangkan kita tahu bahwa probabilitas bernilai maksimum 1, maka:

P(A) + P(G) = 1

2 P(G) + P(G) = 1

3 P(G) = 1

P(G) = 1/3

P(A) = 2 x P(G)

P(A) = 2 x 1/3

P(A) = 2/3

Berhubung variabel acak X menyatakan banyak kejadian muncul sisi angka  dan anggap jumlah percobaan lemparan sebanyak n maka bisa di tuliskan distribusi probabilitasnya sebagai berikut ini :

P(X = 0) = (1/3)ⁿ

P(X = 1) = n (1/3)ⁿ⁻¹ ( 2/3 )

P(X = 2) = [ n(n-1)/2 ] (1/3)ⁿ⁻² ( 2/3 )²

dan seterusnya....

Maka bisa di buat fungsi probabilitasnya adalah :

P ( X = a ) = ⁿCₐ ( 2/3 )ᵃ ( 1/3 )ⁿ⁻ᵃ
