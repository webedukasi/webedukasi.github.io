1. Buatlah Text Procedure Dengan Judul How To Print Pdf File On Windows​


Jawaban:

1. Open a file in a Windows application.

2. Choose File > Print.

3. Choose Adobe PDF as the printer in the Print dialog box.

To customize the Adobe PDF printer setting, click the Properties (or Preferences) button. (In some applications, you may need to click Setup in the Print dialog box to open the list of printers, and then click Properties or Preferences.)

4. Click Print. Type a name for your file, and click Save.

2. How to print Double sided


Jawaban:

How to print double sided manually is

1. Look at the properties for your printer, click File, and then choose Print.

2. In the Printer list, select the printer you want to use.

3. Print the first page by typing 1 in the Pages box, then click Print.

4. When the first side is printed, flip the sheet over, then reinsert it into the printer. Print the other side by typing 2 in the pages box.

Penjelasan:

Procedure text di atas adalah bagaimana mencetak bolak-balik secara manual. Jawaban saya di atas karena tidak semua printer dapat digunakan untuk mencetak bolak-balik. Bila diartikan

1. Lihat kelengkapan untuk printer Anda, klik File, lalu pilih Print.

2. Dalam daftar Printer, pilih printer yang ingin Anda gunakan.

3. Cetak halaman pertama dengan mengetik 1 di kotak Pages, lalu klik Print.

4. Ketika sisi pertama telah dicetak, baliklah kertasnya, lalu masukkan kembali ke dalam printer. Cetak sisi lain dengan mengetik 2 di kotak halaman

11. how to turn PDF into word​

 Click on the “Export PDF” tool in the right pane.

 Choose Microsoft Word as your export format, and then choose “Word Document.”

Click “Export.” If your PDF contains scanned text, the Acrobat Word converter will run text recognition automatically.

Save your new Word file:

Name your converted file, choose DOC or DOCX file format, and click the “Save” button. That’s it.

17. Apa yang dimaksud new, open, save, save as, print??


Jawaban:

Itu semua dimaksudkan sebagai submenu dari menu File dalam program pengolahan data (contohnya program pengolah kata, seperti Ms. Word 2003).

Penjelasan:

1. New: untuk membuat dokumen baru.

2. Open: untuk membuka dokumen yang sebelumnya pernah disimpan.

3. Save: untuk menyimpan dokumen.

4. Save as: untuk menyimpan dokumen dengan nama berbeda.

5. Print: untuk melakukan pencetakan dokumen.

21. how to print out your document at your computer


1. Open the document or file you want to print.


2. In the top portion of the window of the program or browser you are using, the File menu is typically located in the upper left-hand corner of the screen.


3. If you have just clicked File, select Print from the drop-down menu.


4. Either option opens a print properties window or automatically starts printing the document or file.


5. If the print properties window is showing, you can specify additional printing options such as how many copies you want or which specific pages you want to print. Once you've selected the options click Ok or Print to start the printing process.

4.how to use flash disk
1. Open My Computer and see which drives are shown. Most computers, for example, have a hard disk such as a C: drive and a few removable storage devices such as a floppy drive, a CD-ROM drive, and perhaps a zip drive. 2. Insert the flash drive into the USB port and watch to see where the USB flash drive appears. Most will appear as removable storage, but some will instead appear as hard drives. Note the name Windows is using to refer to the flash drive ("Removable Disk (G:)," for example). 3. Open My Documents or the location from which you want to transfer files to the flash drive. Select the files or folders you want to save to the flash drive by leftclicking on them. To select more than one, hold down the CTRL key while you click and select all of the files you wish to save. 4. Right-click on the file(s) or folder(s) you selected, then select Send to, then select the name you saw appearing in My Computer for the flash drive ("Removable Disk (G:)," for example). 5. When the copying is finished, do not immediately remove the flash drive from the USB port. Instead, left-click on the Remove Hardware icon located in the System Tray. A window containing a list of the USB devices will appear. begin

For i = 1 To 100

   If i MOD 3 = 0 Then

       Print "A ", i

   Else

       If i MOD 5 = 0 Then

           Print "B ", i

       Else

           If i MOD 15 = 0 Then

               Print "AB ", i

end
