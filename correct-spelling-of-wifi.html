Jawaban:

Pronunciation

Nama kota (cara baca)

Canterbury (Kaegntebri)Greenwich (Grenic // grinic)Leicester (Lester // Leste)Edinburgh (Edinbre)Warwick (Warik)

-Pendahuluan

Pengertian Pronunciation

Pronunciation adalah suatu kata dalam bahasa Inggris, yang jika diartikan berarti 'pengucapan // pelafalan' dari sebuah kata.

Pelafalannya memperhatikan nada, logat // aksen, dan cara membacanya.

Sebagai contoh, Aksen antara logat Amerika dengan logat Inggris berbeda, meskipun mereka memiliki satu bahasa yang sama.

Seperti :

CAR

Pronunciation aksen Inggris huruf r dibaca 'd' dan nadanya pendek, menjadi 'kad'

Pronunciation aksen Amerika huruf r tetap clear atau jelas dibaca huruf r nya tanpa dirubah, tetapi nadanya sedikit panjang, menjadi 'kaar'

Cara Melatih Pronunciation:

Sebelum mencoba untuk pronunciation, untuk membiasakan nya dengan cara listening terlebih dahulu (mendengarkan). kita bisa mendengar nya dari film ataupun lagu (lebih baik lagi karena kita juga bisa sambil bersenang-senang, sehingga kita juga langsung bisa)Sambil mendengar, kita juga bisa sambil menirukan, dengan gerakan lidah pembicaranya. sebagai contoh: huruf r untuk aksen Amerika, gerakan lidah ke atas (mulut membuka), sedikit ditekuk kebelakang. sedangkan huruf r pada akses Inggris, gerakan lidah datar ke depan (mulut rapat // datar)Jangan hanya mendengar saja, kita juga bisa mencoba berbicara sedikit demi sedikit, nantinya akan terbiasa.Biasakan pula untuk membaca literasi berbahasa Inggris.Dan yang paling penting, anda juga harus bisa membaca huruf abjad dalam bahasa Inggris.

Cara Membaca Huruf Abjad Bahasa lnggris

A → eiB → biC → ciD → diE → iF → efJ → jiH → eijI → aiJ → jeK → keiL → elM → emN → enO → ouP → piQ → kyuR → a: (aksen Inggris, r dibaca d 'ad'; aksen Amerika r tetap clear / tetap dibaca 'ar')S → esT → tiU → yuV → viW → dabelyuX → eks (Malaysia); iks (lng & Amerika)Y → wayZ → zet (Malaysia); zi (lng & Amerika)

4. Write out the correct spelling of these place names.

1. /kæntəbrɪ/

2. /grenɪtʃ/

3. /lestə/

4. /edɪnbrə/

5. /wɒrɪk/

Terjemahan :

" Tulis semua nama tempat ini dengan pengejaan yang tepat !"

*Nomor 1

/kæntəbrɪ/ → Canterbury

Pelafalan : Kaegntebri ✓

Merupakan sebuah kota di lnggris

*Nomor 2

/grenɪtʃ/ → Greenwich

Pelafalan : Grenic (utk Inggris) grinic (utk Amerika) ✓

Merupakan sebuah bagian kota di London yg ada di negara lnggris

*Nomor 3

/lestə/ → Leicester

Pelafalan : Leste (utk Inggris) Lester (utk Amerika) ✓

Merupakan sebuah kota di lnggris, dijadikan pula sebagai nama klub sepakbola, dan universitas 'leicester university'

*Nomor 4

/edɪnbrə/ → Edinburgh

Pelafalan : Edinbre ✓

Merupakan lbu kota negara Skotlandia

Contoh lainnya:

Luxemburg → Luksembre

*Nomor 5

/wɒrɪk/ → Warwick

Pelafalan : Warik ✓

Marupakan sebuah kota di negara bagian 'WarwickShire'.

Penjelasan Tambahan:

Nada 'ɪ' diucapkan dengan pendek (tidak panjang)Nada 'ə' dibaca seperti huruf e biasa, tetapi sedikit samar dibaca huruf a Nada 'ɒ' dibaca pendek, dan dibaca seperti huruf a biasaNada 'ʃ' pendek, dibaca seperti huruf c tetapi sedikit samar dibaca jNada 'æ' penuh tekanan dibaca 'aeg'

Kesimpulan

Jadi, Jawaban yang tepat berdasarkan soal tersebut adalah:

Canterbury (Kaegntebri)Greenwich (Grinic / Grenic)Leicester (Lester / Leste)Edinburgh (Edinbre)Warwick (Warik)
