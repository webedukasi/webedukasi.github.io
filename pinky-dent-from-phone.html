27. Pinky menabung uang sebesar Rp1.800.000,00 di Koperasi "USAHA SEJAHTERA". Jumlah tabungan Pinky setelah 6 bulan menjadi Rp2.091.600,00. Maka besar bunga tabungan per bulan di koperasi tersebut adalah ….


Jawab:

Bunga = p% x b/12 x tabungan awal

2.091.600 - 1.800.000 = p% x 6/12 x 1.800.000

291.600 = p/100 x 1/2 x 1.800.000

291.600 = p x 9.000

291.600/9.000 = p

32,4% = p

bunga per tahun = 32,4 %

bunga per bulan = 32,4 % : 12 = 2,7 %

jadi, bunga per bulannya adalah 2,7%
