5. suku ke-4 dan suku ke-19 suatu barisan aritmatika berturut" 110 dan 150. suku ke tiga puluh aritmatika adalah???


u4 = a + 3b = 110
u19 = a + 18b = 150 _
               -15b = -40
                   b = 8/3
a + 3b = 110
a + 8 = 110
     a = 102
u30 = a + 29b
       = 102 + 29(8/3)
       = 102 + 95/3 = 102 + 31 2/3
                           = 133  2/3
 
Suku ke - 4 suatu barisan aritmatika adalah 110, maka :
karena [tex]U_n=a+(n-1)b[/tex], jadi :
[tex]U_4=110[/tex]
[tex]a+(4-1)b=110[/tex]
[tex]a+3b=110[/tex]...(1)
[tex]U_19=150[/tex]
[tex]a+(19-1)b=150[/tex]
[tex]a+18b=150[/tex]...(2)
kurangkan persamaan (2) dengan persamaan (1), sehingga didapatkan :
a + 18b - (a + 3b) = 150 - 110
a - a + 18b - 3b = 40
15 b = 40
b = [tex] \frac{40}{15}= \frac{8}{3}[/tex]
substitusi nilai b tersebut ke persamaan (1) sehingga diperoleh :
[tex]a+3( \frac{8}{3} )=110[/tex]
[tex]a+8=110[/tex]
[tex]a=110-8=102[/tex]
dengan demikian :
[tex]U_30=102+(30-1) \frac{8}{3}=102+ \frac{232}{3}= \frac{306}{3} +\frac{232}{3}=\frac{538}{3}[/tex]
