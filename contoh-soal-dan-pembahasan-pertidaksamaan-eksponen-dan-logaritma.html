  = (½ 3log 6) / {(3log 9 + 3log 2)2 - (3log 2)2}
  = (½ 3log 3.2) / {(3log 32 + 3log 2)2 - (3log 2)2}
  = {½ (3log 3 + 3log 2)} / {(2 3log 3 + 3log 2)2 - (3log 2)2}
  = {½ (1 + 3log 2)} / {(2 + 3log 2)2 - (3log 2)2}
  = {½ (1 + 3log 2)} / {(4 + 4 3log 2 + 3log 22 ) - (3log 2)2}
