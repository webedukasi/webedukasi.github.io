15. Present Perfect Tense. .......you ever....to a theme park alone? A. has, been B. Has, Gone C. Have Gone D. Have, Been


Present Perfect Tense;

have you ever gone to a theme park alone?

Jawabannya C. Have, gone ✔️

PEMBAHASAN Present Perfect Tense

Digunakan untuk menyatakan Kegiatan yang telah selesai namun masih memiliki kaitan dengan masa kini.

Ciri Present Perfect

=> ever.✔️

=> Already

=> Since

=> for

Rumus Present Perfect

(+) s + has/have + V3 + Complement

(-) s + has/have + not + V3 + Complement

(?) Has/have + s + v3 + complement

KESIMPULAN

HAVE you ever GONE to a theme park alone?

=> Subjek “you” maka menggunakan Have

=> Verb-3 “gone”

38. Kristina... To an opera before she lived in jakarta A.was never going B has never gone C.had never gone D.didnt go E. Will have gone. Ini simple past atau apa ya


Pertanyaan:

Kristina... To an opera before she lived in jakarta

Terjemahan:

Kristina ... Ke opera sebelum dia tinggal di jakarta

Jawaban:

A.was never going (tidak pernah pergi)

B.has never gone (belum pernah pergi)

C.had never gone (belum pernah pergi)

D.didnt go (tidak pergi)

E.Will have gone (Akan pergi) ✔

Jadi jawabnya adalah E.Will have gone.
