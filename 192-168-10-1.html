5. Diketahui Alamat IP Kelas C Dengan Alamat :a. 192. 168. 200. 200 /29b. 192. 168. 10. 225. / 30TentukanSubnet Mask :IP Awal :IP Terakhir :Network :Broadcast :Kalo bisa pakai jalan.Jawab salah satu gak papa.​


a. Alamat IP 192.168.200.200/29

Subnet Mask untuk /29 adalah 255.255.255.248 karena ada 5 bit yang digunakan untuk mengidentifikasi jaringan.

Untuk menentukan IP Awal, kita tinggal menambahkan 1 pada alamat IP yang diberikan:

IP Awal = 192.168.200.201

Untuk menentukan IP Terakhir, kita harus menghitung berapa banyak host yang dapat digunakan pada subnet ini. Pada /29, terdapat 8 subnet yang dapat digunakan (2^3 = 8), tetapi 2 IP digunakan untuk Network dan Broadcast. Jadi, hanya ada 6 IP yang dapat digunakan (8 - 2). Oleh karena itu, IP Terakhir akan menjadi alamat sebelum Network pada subnet berikutnya:

IP Terakhir = 192.168.200.206

Network akan sesuai dengan alamat IP awal yang diberikan:

Network = 192.168.200.200

Broadcast akan sesuai dengan alamat IP terakhir yang diberikan:

Broadcast = 192.168.200.207

b. Alamat IP 192.168.10.225/30

Subnet Mask untuk /30 adalah 255.255.255.252 karena ada 2 bit yang digunakan untuk mengidentifikasi jaringan.

IP Awal = 192.168.10.224

IP Terakhir = 192.168.10.227

Network = 192.168.10.224

13. Jika terdapat ebuah cript router abc ip router 192. 168. 100. 0 255. 255. 255. 252 192. 168. 100. 5 jika router adalah router bca maka 192. 168. 100. 0 merupakan.


Jawaban:

Dari informasi yang disampaikan, 192.168.100.0 merupakan alamat IP jaringan dari router BCA. Alamat IP tersebut merupakan alamat IP dari jaringan yang dikontrol oleh router BCA, dan biasanya digunakan sebagai alamat IP dari jaringan internal yang terhubung ke router BCA.

255.255.255.252 merupakan subnet mask yang digunakan untuk menentukan jumlah alamat IP yang tersedia di jaringan tersebut. Subnet mask tersebut menunjukkan bahwa terdapat hanya 4 alamat IP yang tersedia di jaringan tersebut, yaitu 192.168.100.1, 192.168.100.2, 192.168.100.3, dan 192.168.100.4.

192.168.100.5 merupakan alamat IP default dari router BCA. Alamat IP tersebut biasanya digunakan untuk mengakses web-based interface atau interface lainnya pada router BCA.

Dari script yang di sampaikan, 192.168.100.0 merupakan alamat IP subnet mask untuk jaringan yang terhubung ke router abc. Subnet mask ini digunakan untuk menentukan bagian dari alamat IP yang digunakan untuk mengidentifikasi subnet atau jaringan yang terhubung ke router. 255.255.255.252 menunjukkan bahwa hanya terdapat 2 bit yang digunakan untuk menentukan subnet, sehingga hanya terdapat 4 subnet yang mungkin terhubung ke router abc. 192.168.100.5 merupakan alamat IP yang digunakan oleh router abc itu sendiri.

Jika router adalah router bca, maka 192.168.100.0 merupakan alamat IP subnet mask yang sama untuk jaringan yang terhubung ke router bca. Namun, alamat IP yang digunakan oleh router bca sendiri mungkin berbeda dari alamat IP yang digunakan oleh router abc.

31. Ip gateway yang mungkin digunakan jika ip client adalah 192. 168. 20. 254/30 adalah…


Jika IP client adalah 192.168.20.254/30, maka IP gateway yang mungkin digunakan adalah 192.168.20.253.

IP client adalah alamat IP yang digunakan oleh perangkat klien (misalnya komputer atau smartphone) untuk terhubung ke jaringan. Sedangkan IP gateway adalah alamat IP dari perangkat yang bertugas sebagai penghubung antara perangkat klien dengan jaringan luar.

Dalam kasus ini, IP client menggunakan prefix subnet 30, yang berarti hanya terdapat 2 bit yang digunakan untuk mengidentifikasi host. Jadi, mungkin ada hanya 2 perangkat yang terhubung ke jaringan dengan prefix subnet yang sama, yaitu perangkat dengan alamat IP 192.168.20.254 dan perangkat dengan alamat IP 192.168.20.253. Perangkat dengan alamat IP 192.168.20.253 mungkin merupakan IP gateway yang digunakan oleh perangkat klien.

Pembahasan

Ip address adalah bilangan biner untuk mengidentifikasikan komputer dan perangkat lain pada TCP / IP Network. IP Address adalah sebuah alamat pada komputer agar komputer bisa saling terhubung dengan komputer lain, IP Address terdiri dari 4 Blok, setiap Blok di isi oleh angka 0 - 255. Tujuan utama dari IP Adress yaitu memberikan suatu identitas kepada host agar dikenali oleh jaringan. sehingga server pusat tetap dapat memantau penggunaan dan user dari pemilik IP, sehingga ketika ada penyalahgunaan maka dapat mudah untuk dideteksi dan di ambil tindakan.

IP Adress memiliki fungsi sebagai alat untuk mengidentifikasi host atau antarmuka jaringan. alamat IP dianalogikan sebagai alamat rumat agar paket data yang dikirim dapat sampai tujuan (perangkat) yang tepat. Sehingga dapat disimpulkan bahwa IP Address berfungsi sebagai alamat pengiriman data untuk memastikan setiap perangkat dapat menjalin komunikasi. Perangkat seperti komputer dan router mempunyai IP Address bisa dikenali satu sama lain. Dengan alamat IP Address ini masing-masing perangkat dapat berkomunikasi, mengambil data, dan mengirimkannya.
