Contoh Soal Luas Permukaan Prisma

Alas sebuah prisma dengan bentuk segitiga sama kaki beserta panjang sisi-sisinya, 6 cm, 6 cm, dan 4 cm. Jika tinggi prisma tersebut adalah 9 cm, maka hitunglah luas permukaan prisma tersebut!

Penyelesaian Soal

Hal yang terlebih dahulu kita lakukan adalah, kita harus mencari tinggi segitiga alasnya.

t = √62 – 22

= √36 – 4

= √32

= 4√2 cm

= 5.66 cm

Luas permukaan prisma

= 2 x luas alas + (keliling alas x tinggi prisma)

= (2 x 1/2 x 4 x 5.66) + [(6 + 6 + 4) x 9] = 22.63 + 144

= 166.63 cm2

2.

Rumus Luas Permukaan Limas Segi Empat

Untuk mencari permukaan limas bisa memakai persamaan dibawah ini.

L = Luas alas + luas selimut

Jika alas limas berbentuk persegi, maka persamaan menjadi

L = ( s x s ) + ( 4 x 1/2 x s x t. selimut )

Jika alas memiliki bentuk persegi panjang maka persamaan yaitu

L = ( p x l ) + ( 2 x 1/2 x p x t.selimut ) + ( 2 x 1/2 x l x t.selimut )

Contoh Soal

Diketahui

panjang sisi alas adalah 20 cm dan tinggi limas adalah 25 cm.

Tentukanlah luas permukaan limas segiempat tersebut

Jawab :

Luas alas adalah s x s

= 20 x 20 = 400 cm²;

Luas sisi tegak = 4 x (½ x 20 x 25)

= 4 x 10 x 25

= 40 x 25

= 1.000 cm²

Luas limas segiempat

= 400 cm² + 1.000 cm ²

= 1.400 cm²

3.

Rumus:

Keliling (K) K = 4 x s

Luas (L) L = s x s

Contoh Soal

Ada sebuah balok berbentuk persegi yang memiliki sisi 22 cm . cari dan hitinglah keliling balok tersebut !

Penyelesaian :

Diketahui : S = 22 cm

ditanya : K = …?

Jawab :

K = 4 x S

K = 4 x 22 cm

K = 88 cm

Jadi, keliling balok tersebut adalah = 88 cm
