28. Ditentukan fungsi h: x --› ax - x², dengan a bilangan bulat. a. Tentukan nilai h(-2a), h(a), h(2a), dan h(-a).b. Jika diketahui h(2a) = h(0), tentukan nilai a.


kelas : VIII SMP
mapel : matematika
kategori : fungsi
kata kunci : tentukan nilai

Pembahasan :

h(x) = ax - x^2

a) h(-2a) = a(-2a) - (-2a)^2 
               = -2a^2 - 4a^2
               = -6a^2
    h(a) = a(a) - (a)^2
           = a^2 - a^2
           = 0
    h(2a) = a(2a) - (2a)^2
             = 2a^2 - 4a^2
             = -2a^2
    h(-a) = a(-a) - (-a)^2
            = -a^2 - a^2
