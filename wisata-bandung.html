jumlah siswa ke Yogyakarta (nB) = 16

ke keduanya n(AnB) = 6

belum n(AuB)= 4

jumlah siswa n(S)

= nA + nB - nAnB + nAuB

= 24 + 16 - 6 + 4

= 44 - 6

= 38 siswa

27. Sebuah kelas yang terdiri dari 40 anak, diperoleh data bahwa 24 anak pernah berwisata ke kota bandung dan 16 anak pernah berwisata ke kota surabaya. jika 5 anak belum pernah berwisata ke kedua kota tersebut, banyaknya anak yang pernah berwisata ke kedua kota tersebut adalah ... anak.​


Materi : Himpunan

S = himpunan semesta ( 40 )

a = himpunan kota bandung ( 24 )

b = himpunan kota surabaya ( 16 )

n = himpunan kedua kota ( n )

ab^c = tidak kedua kota ( 5 )

\__________________________/

( a - n ) + ( b - n ) + n = S - ab^c

( 24 - n ) + ( 16 - n ) + n = 40 - 5

40 - n = 35

n = 40 - 35

n = 5 anak
