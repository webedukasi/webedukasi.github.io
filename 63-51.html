7. 23³ + 51² - 63⁴ = ???Nt : Tempered glass dikit retak sih :), gnti ntar lah​


Penjelasan dengan langkah-langkah:

Pembahasan

Eksponen atau bilangan berpangkat itu operasi hitung yang berbentuk pangkat perkalian berulang.

Soal Dan Penyelesaian Nya :

= 23³ + 51² - 63⁴

= ( 23 × 23 × 23 ) + 52² - 63⁴

= ( 529 × 23 ) + 52² - 63⁴

= 12.167 + 52² - 63⁴

= 12.167 + ( 52 × 52 ) - 63⁴

= 12.167 + 2.604 - 63⁴

= 12.167 + 2.604 - ( 63 × 63 × 63 × 63 )

= 12.167 + 2.604 - ( 3.969 × 63 × 63 )

= 12.167 + 2.604 - ( 250.047 × 63 )

= 12.167 + 2.604 - 15.752.961

= 14.771 - 15.752.961

= -15.738.190.

Kesimpulan

Jadi , dapat kita lihat hasil dari 23³ + 51² - 63⁴ adalah −15.738.190
