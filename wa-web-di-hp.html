3. bagaimana cara seorang hacker menge hack hp,leptop dan web menggunakan injector


Dengan membuat sebuah jalan pintas layaknya pencuri memasuki rumah yang dinamakan "backdoor". Backdoor adalah script atau kode yang dirancang oleh programmer untuk dijadikan metode agar dapat meng-akses target dan dengan memanfaatkan kelemahan para device (perangkat seperti Smartphone, Laptop, dll) target itu sendiri.

Konsepnya yaitu, backdoor yang berisikan kumpulan kode disimpan menjadi sebuah file, lalu disematkan menjadi satu dalam sebuah aplikasi (software), website, bahkan sebuah gambar maupun video. Setelah itu, ia mengirimkan sebuah umpan ke target, jika berupa software maka, pembuat program ini akan membuat sesuatu agar target memasang program yang dirancang oleh coder. Setelah target meng-install dan menjalankan software tersebut maka akan otomatis mengirimkan sebuah response kepada coder bahwa target sudah masuk ke dalam jaring.

5. Web atau situs web memiliki halaman web yang berisi data dan informasi yang tersimpan di web server dalam bentuk​


﻿﻿Web atau situs web biasanya terdiri dari halaman-halaman web yang berisi data dan informasi dalam bentuk teks, gambar, video, dan elemen-elemen multimedia lainnya. Halaman-halaman web ini disimpan di server web dalam bentuk file HTML, CSS, JavaScript, dan file media lainnya.

﻿File HTML (Hypertext Markup Language) adalah dasar dari semua halaman web dan digunakan untuk membangun struktur halaman web. File CSS (Cascading Style Sheets) digunakan untuk memperindah tampilan halaman web dengan mengatur warna, font, dan layout. Sedangkan file JavaScript digunakan untuk membuat halaman web interaktif dan dinamis.

﻿Selain itu, halaman web juga dapat menyimpan data dan informasi dalam bentuk file media seperti gambar, video, dan audio. File-file media ini disimpan di server web dan dapat diakses oleh pengguna melalui halaman web.

﻿Secara umum, semua data dan informasi yang terdapat di halaman web disimpan di server web dalam bentuk file yang kemudian diakses oleh pengguna melalui browser web.

24. Apakah perbedaan antara web browser,web page,web server


Web browser, web page, dan web server adalah tiga istilah yang terkait dengan penjelajahan halaman di dunia maya yang dilakukan oleh pengguna internet.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan antara web browser, web page, dan web server. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

WEB BROWSER ADALAH PERANGKAT LUNAK YANG DIPAKAI PENGGUNA INTERNET UNTUK BERSELANCAR DI DUNIA MAYA.

WEB PAGE ADALAH HALAMAN DI INTERNET YANG MENYAJIKAN INFORMASI KEPADA PENGGUNA DAN HANYA DAPAT DIBUKA DENGAN WEB BROWSER.

WEB SERVER ADALAH SISTEM ATAU PERANGKAT YANG DIGUNAKAN PIHAK PENYEDIA KONTEN UNTUK MENYAJIKAN ATAU MENAMPILKAN WEB PAGE DI INTERNET SERTA MENGAKOMODIR AKTIVITAS PENGGUNA DI WEB PAGE TERSEBUT.

28. 1. Jelaskan apa tugas profesi dalam pengembangan aplikasi web berikut : a. Web Designer b. Web Programmer c. Web Administrator d. Web Master


Kelas: IX

Mata Pelajaran: Teknologi Informasi

Materi:  World Wide Web        

Kata Kunci:  web, http, internet

 

Tugas profesi dalam pengembangan aplikasi web berikut:


a. Web Designer: merancang susunan dan tampilan suatu situs pemrograman web


b. Web Programmer: membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web


c. Web Administrator: menjaga kelangsungan kerja sistem web sehari-hari


d. Web Master: tugasnya adalah gabungan dari ketiga tugas di atas

 

Jawaban panjang:

 

a. Web Designer:

 

Ketika suatu web akan dibuat, maka disusun terlebih dahulu rancangan bentuk halaman, serta susunan alur halaman dalam bentuk link dari satu halaman ke halaman lain. Jika web menggunakan sistem halaman dinamik, maka database yang digunakan juga perlu dirancang

Merancang susunan dan tampilan suatu situs pemrograman web ini adalah tugas dari web designer.


b. Web Programmer:

 

Ketika sudah ada rancangan bentuk halaman dan alur situs, maka yang membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web adalah Web Programmer. Bahasa pemrograman yang digunakan adalah HTML dengan dapat ditambah dengan Ruby, PHP atau ASP untuk logika halaman, CSS untuk menata tampilan halaman web, SQL untuk menghubungkan halaman web dengan database serta JavaScript untuk tampilan interaktif.


c. Web Administrator:

 

Ketika halaman web sudah jadi maka yang menjaga kelangsungan kerja sistem web sehari-hari adalah Web Administrator. Halaman web memerlukan orang yang rutin mengecek jalanya web apakah ada permasalahan atau tidak.


d. Web Master:

 

Web master tugasnya adalah gabungan dari ketiga tugas di atas. Web master harus memiliki skill yang dibutuhkan untuk melakukan desain, pembuatan dan perawatan situs web.
