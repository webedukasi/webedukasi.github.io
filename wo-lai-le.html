31. Arti dari sebuah kalimat "wo pa le"


我怕了 wǒ pà le
Mean : Saya takut

arti dari sebuah kalimat =

Wǒ pà le = 我怕了 = Saya ketakutan / saya takut.

Berikut merupakan contoh kalimat menggunakan kata saya takut :

1. 晚上我自己在家，我怕了。

2. 我怕了，如果我的妈妈正在生气的。

3. 我怕了，如果我看到一只蟑螂在我的面前。

Terjemahan dari dua buah contoh kalimat diatas adalah :

1. pada malam ini saya sendirian dirumah,saya ketakutan.

2. saya ketakutan , jika ibu saya sedang marah.

3. saya ketakutan ,jika saya kelihatan seekor kecoa di depan saya.

Semoga bermanfaat
