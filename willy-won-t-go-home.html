23. penggunaan because or because of1.randy is happy......he won the competition2.we go home early........a teachers'meeting3.mrs.sonia doesn't teach....she has an appointmentbantuin aku ya kakak pleaseee


Subject : English

Category : Conjunctions

Level : JHS

Code : 5

Keywords : because, because of

--------

Because dan because of merupakan kata sambung yang menyatakan alasan (conjunctions of reason).

Because berarti karena sedangkan because of berarti dikarenakan.

Dalam penggunaannya ada perbedaan dari because dan because of.

Because harus diikuti oleh klausa (subjek + predikat)

sedangkan because of diikuti oleh kata/frasa benda.

Contoh :

The match was cancelled because it was raining.

(it = subjek, was raining = predikat).

The match was cancelled because of the rain.

(the rain = kata benda).

------

Answers :

1. Randy is happy because he won the competition

2. We go home early because of a teachers'meeting.

3. Mrs. Sonia doesn't teach because she has an appointment.
