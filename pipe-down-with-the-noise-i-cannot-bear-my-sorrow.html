19. A plumber had two pipes. The ratioof the length of the longer pipe to the shorter pipe was 9 : 2 when he cut 1.65 meter from the longer pipe the remaining length was 3 times that of the shorter pipe. Find the length of the shorter pipe in metres


x = the longer pipe
y = the shorter pipe
x : y = 9 : 2
2x = 9y
x = 9y/2

x - 1,65 = 3y
x = 3y + 1,65

x1 = x2
9y/2 = 3y + 1,65
9y = 2(3y + 1,65)
9y = 6y + 3,3
9y-6y = 3,3
3y = 3,3
y = 3,3/3 = 1,1 m

So the shorter pipe is 1,1 meterlet
x be the length of the longer pipe and
y be the length of the shorter pipe

the given ratio means
[tex] \frac{x}{y} = \frac{9}{2} [/tex]
so,
[tex]x = \frac{9}{2} y[/tex]

when he cut 1.65 meter from the longer pipe the remaining length was 3 times that of the shorter pipe, it means
[tex]x - 1.65 = 3y \\ x = 3y + 1.65[/tex]

substitute x
[tex]x = x \\ \frac{9}{2} y = 3y + 1.65 [/tex]

multiply by 2 we obtain
[tex]9y = 6y + 3.3 \\ 9y - 6y = 3.3 \\ 3y = 3.3 \\ y = 1.1[/tex]
hence, y = 1.1, so the length of the shorter pipe is 1.1 meter

One night I (3)........ (hear) a strange noise while we (4)...... (put) up the tent. I (5)...... (not know) that a bear (6).... (follow) us. I (7)...... (turn) on the thorch and (8) ....... (see) the bear. It (9)...... (wait) behind a tree. I (10)...... (Shout) for help and the bear (11)...... (run) away.

2. were climbing

3. heard

4. put

5. didn't know

6. followed

Complate the story with the past simple or the past continuous.

1. V1 take = V2 took

2. V1 climb = V2 climbed

Tapi karena ada “while” maka

While we were climbing in the Rockies

3. V1 hear = V2 heard

4. V1 put = V2 put

Tapi karena ada “while” maka

While we were putting up the tent.

5. (-) did not know

6. V1 follow = V2 followed

7. V1 turn = V2 turned

8. V1 see = V2 saw

9. V1 wait = V2 waited

10. V1 shout = V2 shouted

11. V1 run = V2 ran

Atau dalam kurung:

Last year I went camping in Canada, I (1) [took] this photo while we (2) [were climbing] in the Rockies. One night I (3) [heard] a strange noise while we (4)[we were putting] up the tent. I (5) [did not know] that a bear (6) [followed] us. I (7) [turned] on the thorch and (8) [saw] the bear. It (9) [waited] behind a tree. I (10) [shouted] for help and the bear (11) [ran] away.

Catatan:

= Kata “while” biasanya digunakan dalam kalimat Continues Tense dengan ciri to be (is,am,are,were,was,been) dan kata kerja V+ing.

= Rumus Past Continues Tense:

(+) S + was/were + V+ing + O

(-) S + was/were + not + V+ing O

(?) was/were + S + V+ing + O?

= Rumus Past Tense:

(+) S + V2 + O

(-) S + did not + V1 + O

(?) did + S + V1 + O?
