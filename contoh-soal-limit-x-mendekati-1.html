22. 1. Tentukan limit x mendekati 0 x - sin x / x pangkat 3 2. Tentukan limit x mendekati 0 1-cosx / x 3. tentukan limit x mendekati π 1+cosx / sin2x


Maaf sebelumnya kalau boleh tahu, udah diajarkan turunan? Kalau sudah, turunan adalah cara mengerjakannya.

1) lim x→0 dari x - sinx / x³
Turunkan yang atas, turunkan yang bawah.

- Turunan yg atas: 1 - cos x
- Turunan yg bawah: 3x²
Jadinya:
lim x→0 dari 1 - cosx / 3x²
[Jika dimasukkan 0 ke x, hasil masih 0/0. Jadi diturunkan lagi]

- Turunan yg atas: sin x
- Turunan yg bawah: 6x
Jadinya:
lim x→0 dari sinx / 6x
[Jika dimasukkan 0 ke x, hasilnya pun masih 0/0 lagi. Jadi diturunkan lagi]


- Turunan yg atas: cos x
- Turunan yg bawah: 6
Jadinya:
lim x→0 dari cosx / 6
[masukkan 0 ke x]
= cos(0) / 6
= 1/6 ✔

2) lim x→0 dari 1 - cosx / x

-Turunan yg atas = sin x
-Turunan yg bawah = 1

Jadinya:
lim x→0 dari sinx / 1
[Masukkan 0 ke x]
= sin(0) / 1
= 0 ✔
[0 dibagi berapapun hasilnya nol, selama bukan bentuk 0/0]

3) lim x→π dari 1 + cosx / sin2x

-Turunan yg atas: -sin x
-Turunan yg bawah: 2 cos2x

Jadinya:
lim x→0 dari -sinx / 2 cos2x
[masukkan π ke x]
= -sin(π) / 2 cos(2π)
[nilai π = 180°]
= -sin(180) / 2 cos(2.180)
= -sin(180) / 2 cos(360)
= 0 / 2 . 1
= 0 / 2
= 0 ✔

23. 1) limit x mendekati 0 1-cos2x/tan kuadrat 3x 2) limit x mendekati tak hingga (x-√x kuadrat-5x)


1.

lim x->0    1 -cos2x  /  tan²3x
= lim x->0  1 - (1 - 2.sin²2x)  /  tan²3x
= lim x->0  2.sin²2x / tan²3x
= lim x->0  2.3x.3x. sin²2x  /  tan²3x.3x.3x
= lim x->0  2. sin²2x /  3x . 3x
= lim x->0  2. 2x . 2x . sin²2x /  2x. 2x . 3x . 3x
= lim x->0  2 . 2x . 2x  / 3x. 3x
= lim x->0  8x²/9x²
= lim x->0    8/9
= 8/9

2.
lim x-> S    x - √x²-5x
= lim x-> S    (x - √x²-5x)(x + √x²-5x) / (x + √x²-5x)
= lim x-> S    (x² - (x² - 5x)) /  (x + √x²-5x)
= lim x-> S    (5x) / (x + √x²-5x)  ====> sama sama bagi x
= lim x-> S    (5x : x)  /  (x:x  +  √x²-5x  : x)
= lim x-> S    5  /  (1 + √x²/x² - 5x/x²)
= lim x-> S    5/ (1 + √1 - 5/x)
= 5 / (1 + √1-5/S)
= 5 / (1 + √1-0)
= 5 / (1 + 1)
= 5/2
= 2,5

tanda titik  :  maksudnya kali

39. tolong bantu jawab dengan caranya yaa 1.) limit x mendekati tak terhingga x (x-1) tan 1 per X2.) limit x mendekati tak teringga (sin (2 per x) + cos (1 per x))3.) limit x mendekati tak terhingga (5 + sin 3 per x) 4.) limit x mendekati tak terhingga (tan (1 per 2x) per sin (3 per 4x) 5.) limit x mendekati tak terhingga (1-2 sin² (1 per x) per x)6.) limit x mendekati tak terhingga ((4+tan (2 per x)) + (6+tan (3 per x)))


Nilai limit x mendekati tak terhingga x (x – 1) tan 1 per x adalah ∞.

Rumus limit trigonometri

[tex] \lim \limits_{{x}{\rightarrow}{0}} \frac{sin \: ax}{bx} = \lim \limits_{{x}{\rightarrow}{0}} \frac{ax}{ sin \: bx} = \frac{a}{b} [/tex] [tex] \lim \limits_{{x}{\rightarrow}{0}} \frac{tan \: ax}{bx} = \lim \limits_{{x}{\rightarrow}{0}} \frac{ax}{ tan \: bx} = \frac{a}{b} [/tex] [tex] \lim \limits_{{x}{\rightarrow}{0}} \frac{sin \: ax}{sin \: bx} = \lim \limits_{{x}{\rightarrow}{0}} \frac{tan \: ax}{tan \: bx} = \frac{a}{b} [/tex]  [tex] \lim \limits_{{x}{\rightarrow}{0}} \frac{sin \: ax}{tan \: bx} = \lim \limits_{{x}{\rightarrow}{0}} \frac{tan \: ax}{sin \: bx} = \frac{a}{b} [/tex]  

Definisi limit untu x mendekati tak hingga adalah

[tex] \lim \limits_{{x}{\rightarrow}{\infty}} \frac{1}{x} [/tex] = 0

Pembahasan

Misal

[tex]\frac{1}{x} = p [/tex]

Jika x ⇒ ∞, maka [tex]\frac{1}{x} = p [/tex] ⇒ 0

1) [tex] \lim \limits_{{x}{\rightarrow}{\infty}} x (x - 1) \: tan \: \frac{1}{x} [/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} \frac{(x - 1)}{ \frac{1}{x}} \: . \: tan \: \frac{1}{x}[/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} (x - 1) \: . \: \frac{tan \: \frac{1}{x}}{ \frac{1}{x}}[/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} (x - 1) \: . \: \lim \limits_{{x}{\rightarrow}{\infty}} \frac{tan \: \frac{1}{x}}{ \frac{1}{x}} [/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} (x - 1) \: . \: \lim \limits_{{p}{\rightarrow}{0}} \frac{tan \: p}{p}[/tex]

= (∞ – 1) . 1

= ∞

2) [tex] \lim \limits_{{x}{\rightarrow}{\infty}} (sin \: \frac{2}{x} \: + \: cos \: \frac{1}{x}) [/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} (sin \: 2(\frac{1}{x}) \: + \: cos \: \frac{1}{x}) [/tex]

[tex]= \lim \limits_{{p}{\rightarrow}{0}} (sin \: 2p \: + \: cos \: p) [/tex]

= sin 2(0) + cos 0

= sin 0 + cos 0

= 0 + 1

= 1

3) [tex] \lim \limits_{{x}{\rightarrow}{\infty}} (5 \: + \: sin \: \frac{3}{x}) [/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} (5 \: + \: sin \: 3(\frac{1}{x})) [/tex]

[tex]= \lim \limits_{{p}{\rightarrow}{0}} (5 \: + \: sin \: 3p) [/tex]

= 5 + sin 3(0)

= 5 + sin 0

= 5 + 0

= 5

4) [tex] \lim \limits_{{x}{\rightarrow}{\infty}} \frac{ tan \: \frac{1}{2x}}{sin \: \frac{3}{4x}}[/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} \frac{tan \: \frac{1}{2}(\frac{1}{x})}{sin \: \frac{3}{4}(\frac{1}{x})}[/tex]

[tex]= \lim \limits_{{p}{\rightarrow}{0}} \frac{tan \: \frac{1}{2}p}{sin \: \frac{3}{4}p}[/tex]

[tex]= \frac{\frac{1}{2}}{\frac{3}{4}}[/tex]

[tex]= \frac{1}{2} \times \frac{4}{3}[/tex]

[tex]= \frac{2}{3}[/tex]

5) [tex] \lim \limits_{{x}{\rightarrow}{\infty}} \frac{1 \: - \: 2 \: sin^{2} \: \frac{1}{x}}{x}[/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} \frac{cos \: 2 (\frac{1}{x})}{x}[/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} \frac{1}{x} \: . \: cos \: 2 (\frac{1}{x})[/tex]

[tex]= \lim \limits_{{p}{\rightarrow}{0}} p \: . \: cos \: 2p[/tex]

= 0 . cos 2(0)

= 0 . cos 0

= 0 . 1

= 0

6) [tex] \lim \limits_{{x}{\rightarrow}{\infty}} (4 \: + \: tan \: \frac{2}{x}) \: + \: (6 \: + \: tan \: \frac{3}{x}) [/tex]

[tex]= \lim \limits_{{x}{\rightarrow}{\infty}} (4 \: + \: tan \: 2(\frac{1}{x})) \: + \: (6 \: + \: tan \: 3(\frac{1}{x})) [/tex]

[tex]= \lim \limits_{{p}{\rightarrow}{0}} (4 \: + \: tan \: 2p) \: + \: (6 \: + \: tan \: 3p) [/tex]

= (4 + tan 2(0)) + (6 + tan 3(0))

= (4 + tan 0) + (6 + tan 0)

= (4 + 0) + (6 + 0)

= 4 + 6

= 10
