Perulangan (Lopping)

1. Perulangan while

While merupakan instruksi perulangan yang mirip dengan perulangan for. Bentuk perulangan while dikendalikan oleh syarat tertentu , yaitu perulangan akan terus dilaksanakan selama syarat tersebut terpenuhi.

Bentuk umum pernyataan While adalah:

while(syarat)

{

Pernyataan/perintah;

}

Berikut contoh program sederhana menampilkan bilangan ganjil antar 1-10 dengan menggunakan While:

//Program Sederhana Dengan While

#include <stdio.h>

#include <conio.h>

#include <iostream.h>

main()

{

int nilai=1;

clrscr();

while(nilai<=10)

{

cout<<nilai<<" ";

nilai+=2;

}

getch();

}

Hasil outputnya nanti seperti ini

1 3 5 7 9

2. Pernyataan Do....While...

- Pada perulangan Do-While, tubuh loop berupa pernyataan.

- Pada pernyataan Do, mula-mula pernyataan dijalankan.

- Selanjutnya, kondisi diuji. Seandainya kondisi bernilai benar, maka pernyataan dijalankan lagi. Kemudian kondisi diperiksa kembali dan seterusnya.

- Kalau kondisii bernilai salah, maka pernyataan tidak dijalankan lagi

- Pada dasarnya struktur perulangan Do...While sama saja dengan struktur while, hanya saja pada proses perulangan dengan while, seleksi berada di while yang letaknya berada di atas sementara pada perulangan Do...While seleksi berada dibawah batas perulangan. Jadi dengan menggunakan Do...While sekurang-kurangnya akan terjadi satu kali perulangan.

Bentuk umum Do-While

do

{

Perintah/Pernyataan;

}

While(syarat);

Berikut contoh program sederhana menampilkan bilangan ganjil antara 1-10 dengan Do While :

//Program Sederhana Dengan Perintah Do While

#include <stdio.h>

#include <conio.h>

#include <iostream.h>

main()

{

int nilai=1;

clrscr();

do

{

cout<<nilai<<" ";

nilai+=2;

}

while(nilai<=10);

getch();

}

13. Apabila kita ingin melakukan sebuah perulangan(looping) maka fungsi yang harus digunakan adalah...​


Jawaban singkat:

Loop while dan Loop for

Penjelasan:

Pada bahasa pemograman Java, jika ingin melakukan looping bisa menggunakan 4 cara sesuai kondisi yang ada:

1. Looping while  

Sintaks :

while (condition) {  

   //argument

}

Jika ingin menggunakan while, maka harus membuat sebuah variabel counter agar condition bisa terpenuhi sehingga looping punya titik berhenti. Misalnya sebuah variabel i yang awalnya bernilai 0, lalu looping dengan kondisi i < 5, seperti:

int i = 0;

while (i < 5) {  

   //argument

   i++;

}

Nilai i awalnya 0, dengan increment i++ maka sampai i mencapai nilai 5 baru looping akan berhenti.

2. Looping do while

Sintaks:

do {

  //argument

} while (condition)

Konsepnya sama seperti whlie, tetapi yang membedakan, argumen/perintah dijalankan terlebih dahulu baru kondisi dicek. Pada looping while, program akan melakukan pengecekan terlebih dahulu apakah memenuhi kondisi, jika memenuhi maka jalankan argument.

contoh:

int i = 0;

do {

  //argument

  i++;

} while (i < 5)  

3. Looping for

Sintaks:

for (initialization; condition; update) {

   // argument

}

Looping for adalah salah satu cara yang umum digunakan untuk melakukan looping. Strukturnya terdiri dari tiga bagian: initialization, condition, dan update.

- initialization: Pada bagian ini, biasanya digunakan untuk menginisialisasi variabel penghitung atau variabel lain yang digunakan dalam loop.

- condition: Merupakan kondisi yang akan diperiksa pada setiap iterasi. Jika kondisi bernilai true, maka loop akan terus berlanjut; jika bernilai false, loop akan berhenti.

- update: Pada bagian ini, biasanya digunakan untuk mengubah nilai variabel penghitung atau variabel lain yang akan mempengaruhi kondisi loop.

Contoh penggunaan looping for:

for (int i = 0; i < 5; i++) {

   // argument

}

Pada contoh di atas, loop akan dijalankan selama i kurang dari 5, dan setiap iterasi, nilai i akan bertambah satu (dikarenakan i++ di bagian update).

4. Looping foreach

Sintaks:

for (Type var : iterable) {

   // argument

}

Looping foreach digunakan untuk looping elemen-elemen dari suatu koleksi objek (seperti array atau koleksi lainnya) secara lebih mudah dibandingkan dengan loop for konvensional.

- Type: Tipe data dari elemen-elemen dalam koleksi.

- var: Nama variabel yang akan digunakan untuk menyimpan setiap elemen dalam iterasi.

- iterable: Objek koleksi yang akan diiterasi.

Contoh penggunaan looping foreach untuk mengiterasi array:

int[] numbers = {1, 2, 3, 4, 5};

for (int num : numbers) {

   // argument

}

Pada contoh di atas, setiap elemen dalam array numbers akan disimpan dalam variabel num secara berurutan pada setiap iterasi, sehingga program dapat melakukan operasi pada setiap elemen tersebut.

Pembahasan

Pengertian Algoritma

Algoritma adalah susunan logis dan sistematis dari langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan. Misalnya adalah langkah – langkah untuk memasak mie instan, langkah – langkah yang dapat kita temukan di balik bungkus mie instan tersebut juga dapat kita sebut sebagai sebuah algoritma karena merupakan langkah-langkah untuk menyelesaikan permasalahan yaitu memasak mi instan menjadi matang.

Semakin kompleks suatu algoritma, semakin banyak pula statement di dalamnya dan tidak selamanya berurutan. Statement-statement ini perlu diseleksi dan disesuaikan dengan permasalahan dan prosedur menyelesaikan suatu masalah. Proses penyeleksian statement inilah yang disebut dengan percabangan.

Pengertian perulangan dalam program

Perulangan merupakan salah satu komponen yang sangat penting dalam pemrograman. Perulangan memungkinkan suatu program untuk melakukan proses berulang-ulang sesuai dengan kehendak programmer. Dalam bahasa C, terdapat fungsi untuk melakukan perulangan, yaitu for, while, dan do…while.

For

For merupakan salah satu bentuk perulangan, yang memiliki ciri jumlah perulangan yang telah tetap. Syntax dari fungsi for adalah :

for(inisialisasi ; kondisi ; increment/decrement)

{

Statement…     -> Statement2 yang akan diulang

}

While

While merupakan bentuk perulangan yang memiliki ciri jumlah perulangan yang belum tetap. Yang perlu diperhatikan dalam while adalah teman-teman harus menetapkan dengan pasti kapan perulangan tersebut akan berhenti. Karena jika tidak, maka akan terjadi perulangan tak terbatas yang menyebabkan program menjadi crash.

Syntax:

while(kondisi)

{

Statement…     -> Statement2 yang akan diulang

}

Nested Loop

Bentuk perulangan sendiri dapat dibuat menjadi nested (bersarang). Jadi, di dalam perulangan ada perulangan lagi. Perlu diingat bahwa perulangan yang paling dalam akan menyelesaikan perulangannya dahulu sebelum perulangan di luarnya melanjutkan perulangan.


Flowchart untuk semua perulangan adalah sama, karena prinsip perulangan adalah mengulangi program di dalamnya selama statement di dalam perulangan tersebut bernilai benar.

Simbol-simbol dalam flowchart

-Terminator/Simbol Titik Terminal menunjukkan permulaan (start) atau akhir (stop) dari suatu proses.

-Flow Line / garis alir adalah simbol yang digunakan untuk menghubungkan antara simbol yang satu dengan simbol yang lain (connecting line). Simbol ini juga berfungsi untuk menunjukkan arah aliran program.

-Processing Symbol / Simbol Proses digunakan untuk menunjukkan kegiatan yang dilakukan oleh komputer. Pada bidang industri (proses produksi barang), simbol ini menggambarkan kegiatan inspeksi atau yang biasa dikenal dengan simbol inspeksi

-Decision Symbol / Simbol Keputusan merupakan simbol yang digunakan untuk memilih proses atau keputusan berdasarkan kondisi yang ada. Simbol ini biasanya ditemui pada flowchart program.

-Input-Output / Simbol Keluar-Masuk menunjukkan proses input-output yang terjadi tanpa bergantung dari jenis peralatannya.

-Connector (On-page)  

Simbol ini fungsinya adalah untuk menyederhanakan hubungan antar simbol yang letaknya berjauhan atau rumit bila dihubungkan dengan garis dalam satu halaman

-Connector (Off-page)  

Sama seperti on-page connector, hanya saya simbol ini digunakan untuk menghubungkan simbol dalam halaman berbeda. label dari simbol ini dapat menggunakan huruf atau angka

-Preparation Symbol / Simbol Persiapan merupakan simbol yang digunakan untuk mempersiapkan penyimpanan di dalam storage.

public class main {

       int temp = 1;

       for (int i = 1; i <= 15; i++) {

           System.out.print(i + " ");

           if(i == ((temp + 1) * temp) / 2){

           }

       }

   }

}

Pembahasan

Untuk membuat output seperti pada soal, perlu diketahui dulu cara kerja dari program tersebut. Perhatikan bahwa bilangan terakhir pada tiap baris memiliki pola sebagai berikut

1 = 1

3 = 1 + 2

6 = 1 + 2 + 3

10 = 1 + 2 + 3 + 4

15 = 1 + 2 + 3 + 4 + 5

atau dengan kata lain, bilangan terakhir pada baris ke-n adalah

1 + 2 + 3 + ... + n = ((n + 1) x n) / 2

Sehingga alur program diatas dapat ditulis sebagai berikut :

1. Program dimulai

2. Buat variabel temp untuk menyimpan nomor baris
