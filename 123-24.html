1. 123² + 24³ - 22² = ?​


Eksponen

Eksponen adalah operasi hitung matematika berbasis bilangan berpangkat. Bilangan berpangkat atau bisa disebut eksponen atau perpangkatan adalah bilangan yang dikalikan secara berulang ulang dalam bilangan yang sama.

Cara Penyelesaian :

123² + 24³ - 22²

=> (123 × 123) + (24 × 24 × 24) - (22 × 22)

=> 15.129 + (576 × 24) - 484

=> 15.129 + 13.824 - 484

=> 28.953 - 484

=> 28.469

~ Bilangan BerpangkatPenyelesaian :123² = 123 x 123 123 x 123 = 15.129

24³ = 24 x 24 x 2424 x 24 = 576576 x 24 = 13.824

22² = 22 x 2222 x 22 = 484=======================123² + 24³ - 22²= ( 15.129 + 13.824 ) - 484= 28.953 - 484= 28.469=======================✨Kesimpulan✨123² + 24³ - 22² = 28.469Penjelasan :
