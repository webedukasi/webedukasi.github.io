y = [(sin (x³ + 7))⁵]⁹

25. Turunan aturan rantai dari Cos^3(x^2/1-x)


[tex]\boxed{\frac{dy}{dx} = \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: { \sin}^{3} (\frac{ {x}^{2} }{1 - x}) - \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: \sin(\frac{ {x}^{2} }{1 - x})} \\ \\ \text{atau} \\ \\ \boxed{\frac{dy}{dx} = \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: { \cos }^{2} (\frac{ {x}^{2} }{1 - x}) \: \sin(\frac{ {x}^{2} }{1 - x})} \\ \\[/tex]

PEMBAHASAN

Turunan dari suatu fungsi bisa ditentukan dengan aturan rantai. Aturan rantai dinotasikan dalam bentuk aljabar adalah sebagai berikut :

[tex]\text{Misal fungsi} \: \: y = f(x) \: \: \text{maka} \: \: \: \boxed{\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dv} \cdot \frac{dv}{dx}} \\ \\[/tex]

DIKETAHUI :

[tex]y = { \cos }^{3} \left ( \frac{ {x}^{2} }{1 - x} \right ) \\ \\ [/tex]

DITANYA :

[tex]\frac{dy}{dx} \: \: \: (\text{menggunakan aturan rantai}) \\ \\ [/tex]

JAWAB :

[tex]y = { \cos }^{3} \left ( \frac{ {x}^{2} }{1 - x} \right ) \\ \\ \text{Misal fungsi} \: \: y = {u}^{3} \: \longrightarrow \: \frac{dy}{du} = 3 {u}^{2} \\ \\ u = \cos(v) \: \longrightarrow \: \frac{du}{dv} = - \sin(v) \\ \\ v = \frac{ {x}^{2} }{1 - x} \: \longrightarrow \: \frac{dv}{dx} = \frac{x(2 - x)}{ {(1 - x)}^{2} } \\ \\ \frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dv} \cdot \frac{dv}{dx} \\ \\ \frac{dy}{dx} = 3 {u}^{2} \cdot (- \sin(v)) \cdot \frac{x(2 - x)}{ {(1 - x)}^{2} } \\ \\ \frac{dy}{dx} = 3 { \cos }^{2}(v) \cdot (- \sin(v)) \cdot \frac{x(2 - x)}{ {(1 - x)}^{2} } \\ \\ \frac{dy}{dx} = 3 (1 - { \sin}^{2} (v)) \cdot (- \sin(v)) \cdot \frac{x(2 - x)}{ {(1 - x)}^{2} } \\ \\ \frac{dy}{dx} = 3 ({ \sin}^{2} (v) - 1) \cdot ( \sin(v)) \cdot \frac{x(2 - x)}{ {(1 - x)}^{2} } \\ \\ \frac{dy}{dx} = 3 ({ \sin}^{2} (\frac{ {x}^{2} }{1 - x}) - 1) \cdot \sin(\frac{ {x}^{2} }{1 - x}) \cdot \frac{x(2 - x)}{ {(1 - x)}^{2} } \\ \\ \frac{dy}{dx} = \frac{3x(2 - x)}{ {(1 - x)}^{2} } \cdot ({ \sin}^{2} (\frac{ {x}^{2} }{1 - x}) - 1) \cdot \sin(\frac{ {x}^{2} }{1 - x}) \\ \\ \boxed{\frac{dy}{dx} = \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: { \sin}^{3} (\frac{ {x}^{2} }{1 - x}) - \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: \sin(\frac{ {x}^{2} }{1 - x})} \\ \\ \text{atau} \\ \\ \boxed{\frac{dy}{dx} = \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: { \cos }^{2} (\frac{ {x}^{2} }{1 - x}) \: \sin(\frac{ {x}^{2} }{1 - x})} \\ \\[/tex]

KESIMPULAN :

[tex]\boxed{\frac{dy}{dx} = \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: { \sin}^{3} (\frac{ {x}^{2} }{1 - x}) - \frac{3x(2 - x)}{ {(1 - x)}^{2} } \: \sin(\frac{ {x}^{2} }{1 - x})} \\ \\[/tex]

33. Y = 1/(x + 3) ³ turunan aturan rantainya adalah ​


Jawab 1:

Untuk mencari turunan dari fungsi Y = 1/(x + 3)³ menggunakan aturan rantai, kita perlu mengikuti langkah-langkah berikut:

  1.  Identifikasi fungsi dalam bentuk f(x) = u³, di mana u adalah fungsi dalam bentuk u = g(x).

   Dalam kasus ini, f(x) = 1/u³ dan u = x + 3.

  2.  Hitung turunan pertama dari fungsi u terhadap x, yaitu g'(x).

   Dalam kasus ini, g'(x) = 1.

   3. Hitung turunan pertama dari fungsi f terhadap u, yaitu f'(u).

   Dalam kasus ini, f'(u) = -3/u⁴.

   4. Kombinasikan hasil langkah 2 dan 3 menggunakan aturan rantai:

   (f o g)'(x) = f'(g(x)) * g'(x).

   Dalam kasus ini, (f o g)'(x) = f'(u) * g'(x) = -3/(x + 3)⁴ * 1.

Jadi, turunan dari fungsi Y = 1/(x + 3)³ menggunakan aturan rantai adalah -3/(x + 3)⁴.

Cara 2:

Untuk mencari turunan aturan rantai dari fungsi Y = 1/(x + 3) ³, kita perlu menggunakan aturan rantai untuk turunan fungsi komposisi.

Langkah-langkahnya adalah sebagai berikut:

   1. Misalkan kita memiliki fungsi f(x) = u³, dengan u = 1/(x + 3).

   2. Turunan pertama dari f(x) terhadap u adalah df/du = 3u².

  3.  Selanjutnya, kita perlu mengalikan turunan pertama f(x) terhadap u dengan turunan u terhadap x.

   4. Turunan u terhadap x dapat ditemukan dengan menggunakan aturan rantai, yaitu du/dx = -1/(x + 3)².

  5.  Maka, turunan f(x) terhadap x adalah df/dx = (df/du) * (du/dx) = 3u² * (-1/(x + 3)²).

Substitusi u = 1/(x + 3) ke dalam turunan df/dx di atas akan memberikan turunan akhir dari fungsi Y = 1/(x + 3) ³:

df/dx = 3 * (1/(x + 3))² * (-1/(x + 3)²) = -3/(x + 3)⁴

Jadi, turunan aturan rantai dari fungsi Y = 1/(x + 3) ³ adalah -3/(x + 3)⁴.
