2. ina kunu membeli sebuah mobil pajero keluaran edisi terbaru.mobil pajero milik ina kunu kemudian di sewa agar menghasilkan uang untuk meningkat pendapat keluarga.mobil pajero ina kunu termaksud jenis barang


Barang yang dimanfaatkan untuk mendapatkan hasil atau return atas sesuatu disebut barang produksi.

Penjelasan:

Mobil pajero dalam kasus "Ina Kunu membeli sebuah mobil pajero keluaran edisi terbaru.mobil pajero milik Ina Kunu kemudian di sewa agar menghasilkan uang untuk meningkat pendapat keluarga." merupakan salah satu jenis barang produksi. Meskipun menghasilkan pengembalian atau return dimasa depan, sulit untuk mengategorikan barang ini sebagai barang produktif karena pengertian dari barang produktif sendiri adalah barang yang mempunyai nilai pertumbuhan yang mana barang ini memiliki kecenderungan untuk naik di masa depan dalam jangka panjang. Contohnya adalah emas dan juga properti. Sulit juga mengategorikan barang ini sebagai barang konsumtif karena digunakan untuk menghasilkan return yaitu berupa uang.

Disisi lain, barang dikategorikan menjadi 2 jenis berdasarkan atas tujuan penggunaannya, yaitu barang konsumsi dan barang produksi.

Barang produksi ialah barang yang dimanfaatkan untuk dengan tujuan untuk menghasilkan barang lainnya.

Maka berdasarkan atas tujuan pengunaannya, yang mana mobil pajero tersebut digunakan untuk menghasilkan uang, maka dikategorikan sebagai barang produksi.

2. Aina taskunu = Dimana kamu tinggal

3. Kaifa haluk = Bagaimana kabarmu

4. Maa unwanu baytuka = Apa alamat rumahmu

Nomor 1:

Askunu = اَسْكُنُ = Saya tinggalFii = فِيْ = DiJakarta = جَاكَرْتَا = Jakarta

Aina = اَيْنَ = DimanaTaskunu = تَسْكُنُ = Kamu tinggal

Nomor 3:

Kaifa = كَيْفَ = BagaimanaHaluka = حَلُكَ = Kabarmu (laki-laki) Haluki = حَلُكِ = Kabarmu (perempuan)

Nomor 4:

Maa = مَا = ApaUnwanu = عُنْوَانُ = AlamatBaytun = بَيْتٌ = RumahBaytuka = بَيْتُكَ = Rumahmu (laki-laki)
