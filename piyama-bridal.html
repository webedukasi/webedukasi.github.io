10. Harga 2 daster dan 1 piyama adalah Rp140.000,00. Harga 3 daster dan 2 piyama Rp235.000,00. Harga 4 daster dan 5 piyama adalaha. Rp320.000,00b. Rp450.000,00c. Rp430.000,00d. Rp520.000,00​


Jawab:

C. Rp 430.000

Penjelasan dengan langkah-langkah:

Harga 2 daster dan 1 piyama adalah Rp140.000

Harga 3 daster dan 2 piyama Rp235.000

Ditanya :  Harga 4 daster dan 5 piyama ?

Daster = x

Piyama = y

2x + y = Rp140.000 ...... Persamaan 1

3x + 2y = Rp235.000 ....... persamaan 2

Pertama eliminasi x dengan cara memperkali x agar menjadi 0

KPK 2 dan 3 = 6

2x + y = Rp140.000  x 3

3x + 2y = Rp235.000 x 2

6x + 3y = 420,000

6x + 4y = 470,000

Kurangi menjadi

-y = - 50.000

y = 50.000

Masukkan y ke persamaan 1

2x + y = Rp140.000

2x + 50.000 = 140.000

2x  = 140.000 - 50.000

2x = 90.000

x = 45.000

4x + 5y

= 4(45.000) + 5(50.000)

= 180.000 + 250.000

= Rp 430.000

Penjelasan dengan langkah-langkah:

misal : daster = x dan piyama = y, maka dengan metode subtitusi diperoleh:

2x + y = 140.000 => y = 140.000 - 2x

subtitusi y ke persamaan:

3x + 2y = 235.000

3x + 2(140.000-2x) = 235.000

3x + 280.000 -4x = 235.000

3x - 4x = 235.000 - 280.000

-x = -45.000

x = 45.000 (harga 1 daster)

selanjutnya, subtitusi x = 45.000 ke persamaan:

y = 140.000 - 2x

y = 140.000 -2(45.000)

y = 140.000 - 90.000

y = 50.000 (harga 1 piyama)

jadi harga 4 daster dan 5 piyama adalah:

4(45.000) + 5(50.000) = 180.000 + 250.000

= 430.000

Semoga membantu

30. tentukan harga perunit jika diketahui satu kodi piyama anak adalah RP 870.000​


1 kodi = 20 unit/barang

870.000 = 20 unit piyama

1 kodi piyama = 870.000 ÷ 20 = 43.500

Jadi, harga satu unit/barang piyama adalah 43.500

Dik:

Harga piyama = Rp870.000,00 / kodi = Rp87.000,00 / 20 unit

Dit: Harga piyama/unit?

Jawab =

Harga piyama=

= 870.000/20 unit = n/1

= 20.n = 870.000.1

= 20n = 870.000

= n = 870.000/20 = 43.500

= Jadi, harga piyama per unit adalah Rp43.500,00
