3. apa yang menarik dengan web design?​


Web design adalah bidang yang menarik dan penting dalam pengembangan website. Berikut adalah beberapa hal yang membuat web design menarik:

1. Kreativitas: Web design memberikan kesempatan untuk mengekspresikan kreativitas dalam menghadirkan konten dan merancang tampilan yang menarik. Desainer web dapat menciptakan tata letak yang unik, memilih kombinasi warna yang menarik, dan menggunakan elemen desain seperti tipografi yang menarik.

2. Pengalaman Pengguna: Desain web yang baik mempertimbangkan pengalaman pengguna dengan fokus pada kegunaan dan kemudahan navigasi. Memikirkan bagaimana pengguna akan berinteraksi dengan elemen di dalam website dapat meningkatkan pengalaman pengguna dan mempengaruhi cara pengguna berinteraksi dengan konten.

3. Kesesuaian Brand: Desain web juga memainkan peran penting dalam mempresentasikan identitas merek atau brand. Desain yang konsisten dengan merek dapat membuat website terlihat profesional, memberikan kesan yang positif, dan memperkuat citra merek yang diinginkan.

4. Responsif dan Mobile-Friendly: Dalam era mobile, desain web responsif dan ramah mobile sangat penting. Desain web yang responsif memastikan bahwa website terlihat dan berfungsi dengan baik di berbagai perangkat, termasuk ponsel pintar dan tablet.

5. Pengoptimalan SEO: Desain web juga harus mempertimbangkan faktor-faktor SEO (Search Engine Optimization) untuk membantu website mendapatkan peringkat yang lebih baik di mesin pencari. Pemilihan judul yang relevan, struktur URL yang baik, penggunaan tag heading yang tepat, dan pengoptimalan gambar adalah beberapa aspek yang perlu diperhatikan dalam desain web.

6. Inovasi Teknologi: Web design terus berkembang seiring dengan perubahan teknologi dan tren desain. Desainer web harus tetap up-to-date dengan perkembangan terbaru dalam desain web, seperti penggunaan animasi, grafik vektor, video latar belakang, dan banyak lagi.

Kombinasi semua aspek ini menjadikan web design menarik dan penting dalam menciptakan pengalaman pengguna yang baik dan membuat website Anda menonjol di antara pesaing.

Beda dan contoh dari web site dan web page ?

Perbedaan utama antara kedua adalah isi kontennya. Jika Web Page hanya menyediakan informasi tentang suatu topik, Website menawarkan banyak informasi tentang topik (Website ini difokuskan pada topik-nya yang biasanya terjadi).

• Penjelasan:

Web Page
Sebuah halaman web atau web page adalah sebuah dokumen atau informasi sumber daya yang cocok untuk World Wide Web dan dapat diakses melalui web browser dan ditampilkan pada monitor atau perangkat mobile.
Contoh: https://en.wikipedia.org/wiki/Earth

Website
Sebuah situs web (sering pula disingkat menjadi situs saja; web site, site) adalah sebutan bagi sekelompok halaman web (web page), yang umumnya merupakan bagian dari suatu nama domain (domain name) atau subdomain di World Wide Web (WWW) di Internet maupun. WWW terdiri dari seluruh situs web yang tersedia kepada publik. 
Contoh: https://www.wikipedia.org

23. Dimana tempat belajar design UI dan UX ? web atau ig terserah


Jawaban:

1. Gymnasium

Gymnasium menawarkan kursus online gratis yang telah dirancang secara khusus dalam mengajarkan keterampilan bagi pekerja kreatif, seperti UI/UX Designer.

Setiap materi disusun berdasarkan kebutuhan profesional yang dibutuhkan saat ini.

2. UIGradients

Dalam situs ini, kita bisa mendapatkan kombinasi warna gradasi yang cocok untuk interface kita.

Misalnya Grade Gray, Piggy Pink, Evening Sunshine, Cool Sky dan lain-lain.

Situs UI Gradients digunakan oleh desainer untuk memaduserasikan warna yang sesuai.

UI Gradients menyediakan template kombinasi warna gradasi. Kalian bisa memilih warna dasarnya terlebih dahulu.

3. Colors Eva Design

Dalam situs ini kita disuguhkan dengan kombinasi warna yang keren.

Situs Colors Eva Design membantu UI Designer untuk membuat design system dalam rancangan atau mockup proyek mereka.

Biasanya mereka sendiri akan menentukan warna dasar dan warna pendukungnya dalam bentuk dan medium yang berbeda seperti logo, banner, flyer dan lain-lain.

4. InVision

Apalagi UX Designer pemula, kursus ini bisa dibilang worth.

Sebab InVision menyediakan materi dasar yang berkaitan dengan profesi UX Designer.

Kerennya lagi InVision memberikan e-book gratis yang dikirim ke user setiap minggunya.

5. Hackdesign

Hackdesign adalah kursus online yang diperuntukkan bagi UI ataupun UX Designer untuk belajar lebih dalam soal UI UX.

Disini kalian bisa mendapatkan akses pembelajaran UI/UX Gratis.

Tersedia 50 paket pembelajaran secara online dan update materi setiap minggunya. Cara mendapatkannya gampang sekali.

6. Envato Tuts+

7. Dribble
