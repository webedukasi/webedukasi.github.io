1. Sasa mengelola sebuah Cafe bernama cafesa. Menu best seller di cafe tersebut adalah frappuccino ala sasa. Sasa menjual frappuccino seharga 30.000 dan biasa terjual sebanyak 100 gelas per minggu. Pada bulan agustus cafesa menurunkan harga frappuccino sebagai event memperingati hari kemerdekaan. Sasa menurunkan harga frappuccino menjadi 25.000 dan ternyata pada minggu ke 1 terjual 200 gelas. Hitunglah fungsi permintaannya!


Jawaban:

Untuk grafik bisa di salin lalu ditempel di matplotlib.pyplot ya

Penjelasan:

Fungsi permintaan adalah hubungan antara harga dan jumlah permintaan. Dalam kasus ini, harga adalah harga frappuccino dan jumlah permintaan adalah jumlah frappuccino yang terjual per minggu.

Berdasarkan informasi yang diberikan, kita dapat membuat persamaan sebagai berikut:

```

Q = a - bP

```

dimana:

* Q adalah jumlah permintaan

* a adalah konstanta

* b adalah koefisien permintaan

* P adalah harga

Kita dapat menentukan nilai a dan b dengan menggunakan data yang diberikan.

Untuk a, kita dapat menggunakan data saat harga frappuccino adalah 30.000. Pada saat itu, jumlah permintaan adalah 100 gelas.

```

100 = a - b(30.000)

```

```

a = 100 + 30.000b

```

Untuk b, kita dapat menggunakan data saat harga frappuccino adalah 25.000. Pada saat itu, jumlah permintaan adalah 200 gelas.

```

200 = a - b(25.000)

```

```

200 = 100 + 25.000b

```

```

25.000b = 100

```

```

b = 0.04

```

Sekarang, kita dapat memasukkan nilai a dan b ke dalam persamaan awal:

```

Q = 100 + 30.000(0.04)

```

```

Q = 100 + 1200

```

```

Q = 1300

```

Jadi, fungsi permintaan frappuccino ala sasa adalah:

```

Q = 1300 - 12P

```

Artinya, jumlah permintaan frappuccino ala sasa akan menurun sebesar 12 gelas setiap kenaikan harga sebesar 1.000 rupiah.

Berikut adalah grafik fungsi permintaan frappuccino ala sasa:

```

import matplotlib.pyplot as plt

x = range(10, 50)

y = 1300 - 12 * x

plt.plot(x, y)

plt.xlabel("Harga (Rp)")

plt.ylabel("Jumlah Permintaan (gelas)")

plt.show()

```

Dari grafik tersebut, kita dapat melihat bahwa jumlah permintaan frappuccino ala sasa akan meningkat seiring dengan penurunan harga.

EXPLANATION

Present Simple adalah verb tense yg digunakan untuk mendeskripsikan :

~ Habit, aktivitas yg berulang

~ Fakta, kebenaran umum

~ Kegiatan yg sudah terjadwalkan dlm waktu dekat

Time signal

~ Never

~ Every Sunday

~ Everyday

~ Always

~ Every week

~ Sometimes

~ Often

~ Once a week

Pola

Verbal sentence [Predikatnya berupa kata kerja]

[+] S+verb1(s/es)+O/C

[- ] S+do/does+not+verb1+O/C

[?] Do/does+S+verb1+O/C ?

Contoh :

[+] He watches a football match every Sunday

[- ] He DOESN'T watch a football match every Sunday

[?] DOES he watch a football match every Sunday ?

Note :

~ Pakai Aux. "do" untuk subyek [I, They, We, You dan kata benda jamak] pd kalimat (-) (?) dan tanpa akhiran -s/es pd kata kerjanya untuk kalimat (+)

~ Pakai Aux "does" untuk subyek [It, She, He, Nama orang] pd kalimat (-) (?) dan menggunakan akhiran -s/es untuk kalimat (+).

~ Khusus untuk kata kerja berakhiran -ch, -ss, -s, -o, -x, -sh menggunakan akhiran -es jika subyeknya [it, she, he, nama orang]
