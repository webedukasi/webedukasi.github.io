2. coba dong kasi contoh soal biloks sekalian penjelasannya​


1. Tentukan bilangan oksidasi S dalam SO3!

Jawab:

Jumlah bilangan oksidasi SO3 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ b.o S + (3 × b.o O) = 0

⇒ b.o S + (3 × -2) = 0

⇒ b.o S + (-6) = 0

⇒ b.o S = 0 + 6

⇒ b.o S = 6

Jadi, bilangan oksidasi S dalam SO3 adalah 6.

2. Tentukan bilangan oksidasi S dalam SO2!

Jawab:

Jumlah bilangan oksidasi SO2 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ b.o S + (2 × b.o O) = 0

⇒ b.o S + (2 × -2) = 0

⇒ b.o S + (-4) = 0

⇒ b.o S = 0 + 4

⇒ b.o S = 4

Jadi, bilangan oksidasi S dalam SO2 adalah 4.

3. Tentukan biloks F dalam FeO!

Jawab:

Jumlah bilangan oksidasi FeO = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ b.o F + (1 × b.o O) = 0

⇒ b.o F + (1 × -2) = 0

⇒ b.o F + (-2) = 0

⇒ b.o F = 0 + 2

⇒ b.o F = 2

Jadi, bilangan oksidasi F adalam FeO adalah 2.

4. Tentukan bilangan oksidasi Fe dalam Fe2O3!

Jawab:

Jumlah bilangan oksidasi Fe2O3 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ (2 × b.o F) + (3 × b.o O) = 0

⇒ (2 × b.o F) + (3 × -2) = 0

⇒ (2 × b.o F) + (-6) = 0

⇒ (2 × b.o F) = 0 + 6

⇒ (2 × b.o F) = 6

⇒ b.o F = 6/2

⇒ b.o F = 3

Jadi, bilangan oksidasi F adalam Fe2O3 adalah 3.

5. Tentukan biloks N dalam NO2!

Jawab:

Jumlah bilangan oksidasi NO2 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ (b.o N) + (2 × b.o O) = 0

⇒ (b.o N) + (2 × -2) = 0

⇒ (b.o N) + (-4) = 0

⇒ b.o N = +4

Jadi, biloks N dalam NO2 = +4.

6. Tentukan bilangan oksidasi Cl adalam ion ClO3−!

Jawab:

Jumlah bilangan oksidasi ClO3− = −1

Bilangan oksidasi O = -2

Maka:

⇒ (b.o Cl) + (3 × b.o O) = −1

⇒ (b.o Cl) + (3 × -2) = −1

⇒ (b.o Cl) + (-6) = −1

⇒ b.o Cl = −1 + 6

⇒ b.o Cl = 5

Jadi, biloks Cl dalam ClO3– = +5.

7. Tentukan bilangan oksidasi N dalam ion NH4+!

Jawab:

Jumlah bilangan oksidasi NH4+ = +1

Jumlah bilangan oksidasi H = +1

Maka:

⇒ (b.o N) + (4 × b.o H) = 1

⇒ (b.o N) + (4 × 1) = 1

⇒ (b.o N) + (4) = 1

⇒ b.o N = 1 – 4

⇒ b.o N = −3

Jadi, biloks N dalam NH4+ = –3.

8. Tentukan biloks S dalam ion SO42–!

Jawab:

Jumlah bilangan oksidasi SO42– = −2

Bilangan oksidasi O = −2

Maka:

⇒ (b.o S) + (4 × b.o O) = −2

⇒ (b.o S) + (4 × −2) = −2

⇒ (b.o S) + (−8) = −2

⇒ b.o S = −2 + 8

⇒ b.o S = 6

Jadi, biloks S dalam SO42– = +6.

9. Tentukan bilangan oksidasi O dalam H2O2!

Jawab:

Jumlah bilangan oksidasi H2O2 = 0

Bilangan oksidasi H = +1

Maka:

⇒ (2 × b.o H) + (2 × b.o O) = 0

⇒ (2 × 1) + (2 × b.o O) = 0

⇒ (2) + (2 × b.o O) = 0

⇒ 2 × b.o O = −2

⇒ b.o O = −2/2

⇒ b.o O = −1

Jadi, bilangan oksidasi O dalam H2O2 adalah -1.

10. Tentukan biloks Mn adalam MnO4−!

Jawab:

Jumlah bilangan oksidasi MnO4− = −1

Bilangan oksidasi O = −2

Maka:

⇒ (b.o Mn) + (4 × b.o O) = −1

⇒ (b.o Mn) + (4 × −2) = −1

⇒ (b.o Mn) + (−8) = −1

⇒ b.o Mn = −1 + 8

⇒ b.o Mn = 7

Jadi, biloks Mn dalam ion MnO4− adalah + 7.

5. biloks dari 2nah2po3 ?biloks dari 2ph3 ?​


Jawaban:

Biloks dari 2NaH2PO3 (natrium hipofosfit) adalah:

- Biloks natrium (Na) adalah +1 karena ia adalah logam alkali.

- Biloks hidrogen (H) adalah +1 karena ia umumnya memiliki bilangan oksidasi positif ketika terikat dengan nonlogam.

- Biloks fosfor (P) adalah +1 karena ada dua hidrogen yang terikat ke fosfor dan masing-masing hidrogen memiliki bilangan oksidasi +1. Namun, ada dua atom oksigen di sebelah fosfor, yang masing-masing memiliki bilangan oksidasi -2. Karena total muatan dari senyawa harus netral, maka bilangan oksidasi dari fosfor harus +1 untuk menetralkan total bilangan oksidasi hidrogen dan oksigen. Oleh karena itu, bilangan oksidasi fosfor dalam 2NaH2PO3 adalah +1.

Jadi, biloks dari 2NaH2PO3 adalah: Na(+1) + 2H(+1) + P(+1) + 3O(-2) = 0.

Sedangkan, biloks dari 2PH3 (difosfina) adalah:

Biloks fosfor (P) adalah -3 karena ia terikat dengan tiga atom hidrogen (H), dan masing-masing hidrogen memiliki bilangan oksidasi +1. Oleh karena itu, untuk menetralkan total bilangan oksidasi hidrogen dan fosfor harus -3.

Jadi, biloks dari 2PH3 adalah: 2P(-3) + 6H(+1) = 0.

1. Unsur Bebas ⇒ Biloks = 0

a. Logam A ⇒ Biloks = + sesuai golongannya

Contoh :

Gol. IIA = Be, Mg, Ca, Sr, Ba, Ra ⇒ Biloks = +2

Gol. IIIA = B, Al, Ga, In, TI ⇒ Biloks = +3

c. Atom F ⇒ Biloks = -1

Contoh : HF , F2

d. Atom O

Contoh : H2OTetapi ada pengecualian, Atom O bisa memiliki bilangan oksidasi -1 (Biloks O = -1) apabila berada pada senyawa peroksida. Senyawa peroksida adalah senyawa yang kelebihan atom O. biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0

40. soal materi tentang biloks


Soal No. 1
Tentukan bilangan oksidasi unsur S dalam senyawa H2SO4

Pembahasan
H2SO4 jika diuraikan sesuai ion-ion pembentuknya adalah:
H2SO4 → 2H+ + SO42−
Yang ditanyakan adalah biloks S, sehingga cukup melihat pada ion SO42− saja. Jumlah biloks pada ion ini harus − 2. Misal biloks S adalah x dan diingat biloks O disini adalah −2 sehingga:
x + 4(−2) = −2
x = −2 + 8 = +6
Jadi biloks S pada senyawa di atas adalah +6

Soal No. 2
Tentukan bilangan oksidasi unsur Cr dalam senyawa K2Cr2O7

Pembahasan
K2Cr2O7 terdiri atas ion K+ dan Cr2O72−.

Lihat Cr2O72− saja, misal biloks Cr adalah x:
2(x) + 7(−2 ) = −2
2x = 12
x = + 6

Biloks Cr pada K2Cr2O7 dengan demikian adalah +6


Soal No. 3
Diketahui persamaan reaksi redoks sebagai berikut:



Perubahan bilangan oksidasi Mn pada reaksi redoks tersebut adalah...

D. +2 menjadi +4
E. +4 menjadi +2

x + 2(−2) = 0
x = + 4

x = +3

Perubahan biloks dari +4 menjadi +3
