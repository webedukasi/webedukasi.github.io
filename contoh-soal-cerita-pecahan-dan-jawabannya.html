2. contoh soal dan jawaban cerita pecahan,desimal, dan persen


Kelas : VI
Pelajaran : matematika
Kategori : pecahan
Kata kunci : soal cerita

Pembahasan:
Penghasilan Pak Galih sebesar Rp 1.200.000 per bulan. Pak Galih menyisihkan 2,5% untuk zakat. Sebesar 1/5 dari penghasilannya digunakan untuk diberikan kepada ibunya. Sebesar 0,1 dari penghasilannya dipotong untuk pajak penghasilan. Berapa jumlah sisa uang Pak Galih yang per bulan?

Diketahui:
Penghasilan = 1.200.000
Zakat = 2,5%
Ibu = 1/5
Pajak = 0,1

Cara pertama:
Zakat = 1.200.000 x 2,5/100
= 30.000

Ibu = 1.200.000 x 1/5
= 240.000

Pajak = 1.200.000 x 0,1
= 120.000

Maka, sisa uang Pak Galih adalah
= 1.200.000-30.000-240.000-120.000
= 810.000

Cara kedua:
2,5% dijadikan desimal, yaitu 0,025
1/5 dijadikan desimal, yaitu 0,2
Sedangkan penghasilan Pak Galih per bulan, dianggap sebagai 1

= (1-0,025-0,2-0,1) x 1.200.000
= 0,675 x 1.200.000
= 810.000

Jadi, sisa uang Pak Galih per bulan adalah Rp 810.000.
