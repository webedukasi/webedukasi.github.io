18. pakai caranya soal matematika kelas 4 SD semester 2 tentang Segitiga​


Rumus keliling segitiga

Keliling segitiga =  a + b + c

Jawaban atas pertanyaan dari segitiga A-E

Pertama-tama, anggap

Kolom ke-2 = a Kolom ke-3 = b dan Kolom ke-4 = c

A. Diketahui = a = 16 cm, b = 15 cm, c =17 cm.

Ditanya = keliling =___?

Keliling segitiga = a + b + c

Keliling segitiga = 16 cm + 15 cm +17 cm = 48 cm.

B. Diketahui = b = 42 dm, c = 46 dm dan keliling = 125 dm.

Ditanya = a =___?

Keliling segitiga = a + b + c

125 dm = a + 42 dm + 46 dm

125 dm - 42 dm - 46 dm = a

83 dm - 46 dm = a

a = 37 dm.

C. Diketahui = a = 20 m, c = 25 m, Keliling = 67 m

Ditanya = b =___?

Keliling segitiga = a + b + c

67 m = 20 m + b + 25 m

67 m - 20 m - 25 m = b

47 m - 25 m = b

b = 22 m.

D. Diketahui = a = 23 cm, b = 26 cm, Keliling = 63 cm

Ditanya = c =___?

Keliling segitiga = a + b + c

63 cm = 23 cm + 26 cm + c

63 cm - 23 cm - 26 cm = c

c = 40 cm - 26 cm = 14 cm.

E. Diketahui = a = 29 mm, b = 32 mm, Keliling = 100 mm

Ditanya = c =___?

Keliling segitiga = a + b + c

100 cm = 29 cm + 32 cm + c

c = 100 cm - 29 cm - 32 cm.

c = 71 cm - 32 cm = 39 cm.

28. contoh soal ulangan matematika kelas 4 sd tentang mengubah pecahan


Ada 3 bagian:

a. 6/25 (pecahan biasa)
harus diubah dulu menjadi bilangan senilai penyebut 100
untuk mengubah pembilangnya menjadi 100 maka 25 harus dikalikan dengan 4 begitu juga dengan 6 harus dikalikan dengan 4, hasil akhirnya :
6/25 = (6x4) (25x4)
= 24/100
Jadi 6/25 = 24%

b. 1 ½
untuk pecahan campuran, terlebih dahulu kita mengubah pecahan campuran menjadi pecahan biasa,
1 ½ = (2x1+1) /2
= 3/2
Setelah itu baru kita ubah menjadi persen
3/2 = (3x50) (2x50)
= 150/100
jadi, 1 ½ adalah 150%

c. 0,48
Untuk bentuk desimal, langkah yang dilakukan yaitu mengubah desimal ke dalam bentuk pecahan kemudian mengubah pecahan dengan penyebut 100 dan terakhir mengubah ke dalam bentuk persen,
0,48 = 0 + 40/10 + 8/100
= 0 + 40/100 + 8/100
= 48/100
= 48 %
jadi, 0,48 sama saja dengan 48 %
