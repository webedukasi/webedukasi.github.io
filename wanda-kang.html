9. Sandangan kang kalebu sandangan payigeg wanda​


Sandhangan kang kalebu sandhangan panyigeg wanda, yaiku:

WignyanLayarCecakPangkon

➖➖➖➖➖➖➖➖➖➖➖

Pembahasan

Sandhangan aksara jawa kaperang dadi 3, yaiku: sandhangan swara, sandhangan panyigeg wanda lan sandhangan wyanjana.

● Sandhangan panyigeg wanda

Yaiku sandhangan sing digunakake kanggo nyigeg utawa mateni aksara sing disigeg. Cacahing sandhangan panyigeg wanda ana 4, yaiku:

Wignyan → kanggo sigeg h ( ꦃ ).Layar → kanggo sigeg r ( ꦂ ).Cecak → kanggo sigeg ng ( ꦁ ).Pangkon → kanggo sigeg sakliyane h, r lan ng ( ꧀ ).

Tuladha:

Wignyan → Kolah ( ꦏꦺꦴꦭꦃ ).Layar → Pasar ( ꦥꦱꦂ ).Cecak → Gendhang-gendhing ( ꦒꦼꦤ꧀ꦢꦁꦒꦼꦤ꧀ꦢꦶꦁ ).Pangkon → Nyacat ( ꦚꦕꦠ꧀ ).

27. tembung tembung Kang migunakake sandhangan panyigeg Wanda ​


Tembung Tembung sing nggunakake sandhangan panyigeg wanda :

sawah ➡️ hpasar ➡️ rcacing ➡️ ngkapuk ➡️ k

Pendahuluan :

Sandhangan panyigeg wanda asale saka tembung sigeg sing tegese mati ( huruf mati ).

Sandhangan panyigeg wanda yaiku sandhangan kang digunakake kanggo ngganteni aksara sigeg.

Sandhangan panyigeg wanda ana 4 :

wignyan = paten H ➡️ ꦃlayar = paten R ➡️ ꦂcecak = paten ng ➡️ ꦁpangkon = saliyane h, r lan ng ➡️ ꧀

Pembahasan :

Tembung Tembung sing nggunakake sandhangan panyigeg wanda :

sawah ➡️ hpasar ➡️ rcacing ➡️ ngkapuk ➡️ k

1. sawah = ꦱꦮꦃ ➡️ paten H

2. pasar = ꦥꦱꦂ ➡️ paten R

3. cacing = ꦕꦕꦶꦁ ➡️ paten ng

4. kapuk = ꦏꦥꦸꦏ꧀ ➡️ pangkon paten K
