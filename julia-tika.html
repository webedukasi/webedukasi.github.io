10. Julia mempunyai uang Rp60 ribu lebih banyak dari Cicih. Jika Cicih memberikan Rp20 ribu ke Julia, maka uang Julia akan menjadi tiga kali lipat dari Cicih. Berapa uang mula-mula mereka? Mohon penjelasannya ​


Jawab:

Penjelasan dengan langkah-langkah:

Mari kita selesaikan masalah ini dengan cara mendefinisikan variabel untuk uang Cicih dan uang Julia.

Misalkan:

Uang Cicih = x ribu rupiah

Uang Julia = (x + 60) ribu rupiah (karena Julia memiliki uang Rp60 ribu lebih banyak dari Cicih)

Menurut informasi dari soal, jika Cicih memberikan Rp20 ribu ke Julia, maka uang Julia akan menjadi tiga kali lipat dari Cicih. Oleh karena itu, kita dapat menyusun persamaan berikut:

Uang Julia setelah menerima Rp20 ribu = 3 * (Uang Cicih setelah memberikan Rp20 ribu)

Uang Julia setelah menerima Rp20 ribu = (x + 60) + 20 ribu rupiah = x + 80 ribu rupiah

Uang Cicih setelah memberikan Rp20 ribu = x - 20 ribu rupiah

Maka, persamaannya menjadi:

x + 80 = 3 * (x - 20)

Selanjutnya, kita selesaikan persamaan tersebut:

x + 80 = 3x - 60

x - 3x = -60 - 80

-2x = -140

x = -140 / -2

x = 70

Jadi, uang Cicih awalnya adalah 70 ribu rupiah, dan uang Julia awalnya adalah (70 + 60) = 130 ribu rupiah.

Jawaban:

Misalkan uang mula-mula yang dimiliki Cicih adalah X ribu rupiah. Berdasarkan informasi soal, Julia memiliki uang Rp60 ribu lebih banyak dari Cicih. Jadi, uang yang dimiliki Julia adalah (X + 60) ribu rupiah.

Selanjutnya, jika Cicih memberikan Rp20 ribu ke Julia, maka uang Julia akan menjadi tiga kali lipat dari uang Cicih. Dengan demikian, kita dapat membuat persamaan berikut:

(X + 60 - 20) = 3(X - 20)

Mari kita selesaikan persamaan tersebut:

X + 40 = 3X - 60

Kita pindahkan X ke satu sisi persamaan dan angka konstan ke sisi lainnya:

X - 3X = -60 - 40

-2X = -100

Kemudian kita bagi kedua ruas persamaan dengan -2 untuk mendapatkan nilai X:

X = -100 / -2

X = 50

Jadi, uang mula-mula yang dimiliki Cicih adalah Rp50 ribu. Dengan demikian, Julia memiliki uang (50 + 60) = Rp110 ribu.


11. Uang julia 3/5 kali uang Romi, jika romi memberikana uang Rp. 168.000 kepada julia, maka uang romi menjadi 7/9 uang julia, uang julia mula - mula adalah....


Jawab:

Uang Julia (J) uang Romi (R)

(i) J : R = 3 : 5

R = 5/3 J

(ii) (J+168.000) : (R-168.000) = 7 : 9

maka uang J awal ...?

Penjelasan dengan langkah-langkah:

masukan persaaan (i) ke (ii)

(J + 168.000) : (5/3 J - 168.000) = 7 : 9

9(J+168.000) = 7 (5/3 j - 168.000)

9J + 1.512.000 = 35/3 j - 1.176.000

9J - 35/3 J = - 1.176.000 - 1.512.000

-8/3 J = - 2.688.000

J = 2.688.000 x 3/8

J =  8.064.000/8

J = 1.008.000

uang julia mula-mula Rp.1.008.000

34. Pertanyaan bahasa inggris..mana kalimat yang paling tepat, dan kenapa yang lainnya tidak tepat..?1. This is my friend julia2. This is my friends julia3. This is julia my friend4. This is julia my friends


-sentence structure-

[tex] \\ [/tex]

kalimat yang paling tepat adalah kalimat nomor 1.

[tex] \to [/tex] This is my friend Julia.

[tex] \\ [/tex]

PEMBAHASAN

1. This is my friend Julia (ini adalah temanku Julia).

kalimat ini benar, karena susunan kalimatnya sesuai dan artinya pun tepat.

2. This is my friends Julia (ini adalah teman-temanku Julia).

kalimat ini salah, karena meskipun susunan kalimatnya sesuai, terdapat kata friends yang merujuk ke subjek jamak, sedangkan Julia adalah subjek yang berjumlah satu.

3. This is Julia my friend (ini adalah Julia temanku).

kalimat ini salah, karena meskipun artinya benar, masih kurang tanda koma setelah 'Julia' untuk memisahkan kalimat utama dengan penjelas.

4. This is Julia my friends (ini adalah Julia teman-temanku).

kalimat ini salah, karena selain masih kurang tanda koma setelah 'Julia' untuk memisahkan kalimat utama dengan penjelas, terdapat kata friends yang merujuk ke subjek jamak, sedangkan Julia adalah subjek yang berjumlah satu.

Jadi, kalimat yang benar adalah kalimat pada nomor 1 (This is my friend Julia).

[tex] \\ [/tex]
