2. contoh soal diagram batang untuk kelas 3 SD​


CONTOH SOAL MENGENAI DIAGRAM BATANG

Sebelum kita menuju ke contoh soal tentang diagram batang, kita perlu tahu apa itu diagram batang.

Diagram batang adalah salah satu metode penyajian data statistik berupa batang - batang berbentuk persegi panjang yang menjulang mengikuti frekuensi datanya.

Hal - hal yang harus diperhatikan dalam membuat diagram batang adalah :

1. Diagram batang memiliki dua sumbu, yaitu sumbu tegak dan sumbu mendatar. Sumbu tegak adalah sumbu yang berdiri dan memuat parameter frekuensi masing - masing data yang dimuat. Sedangkan sumbu mendatar adalah sumbu yang tidur dan memuat kategori data yang akan disajikan.

Sumbu tegak dan sumbu mendatar saling tegak lurus serta berpotongan di salah satu ujungnya.

2. Jika kategori datanya memuat nama dengan frekuensi tertentu, maka tidak perlu diurutkan. Misalnya :

Data Pengguna Kendaraan Bermotor.

Pengguna Mobil ada 180 orang,

Pengguna motor ada 200 orang,

dan seterusnya.

Sedangkan bila kategori datanya memuat angka dengan frekuensi tertentu, maka perlu diurutkan. Misalnya :

Data Perolehan Nilai Dari Ujian Matematika

Nilai 30 ada 2 orang siswa,

Nilai 40 ada 6 orang siswa,

Nilai 50 ada 3 orang siswa,

Nilai 60 ada 7 orang siswa,

dan seterusnya.

3. Jarak antar batang harus sama, begitu pula besar batang.

4. Pastikan tinggi rendahnya frekuensi benar - benar sesuai dengan datanya.

5. Buatlah diagram menggunakan penggaris agar lebih rapi serta menghasilkan besar batang dan jarak antar batang yang sama.

Nah, setelah kita mengenal diagram batang, kita menuju ke contoh soalnya dalam pembahasan soal berikut.

PEMBAHASAN :

Diagram garis adalah diagram kebanyakan dipakai untuk menyajikan data yang didapat dari waktu ke waktu secara teratur dengan jarak waktu tertentu. Umumnya diagram garis sering dipakai untuk melihat perkembangan/pertumbuhan dari sesuatu secara berkelanjutan.

Diagram batang biasanya digunakan untuk mengetahui perkembangan nilai dari suatu data dalam jangka waktu tertentu maupun suatu jenis data dengan beberapa klasifikasi.

34. contoh soal sifat asosiatif dan distributif kelas 5 sd mapel matematika


Kelas : V (5 SD)
Materi : bilangan bulat
Kata Kunci : bilangan, bulat, sifat-sifat, asosiatif, distributif

Pembahasan :
1. Penjumlahan
Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

(a + b) + c = a + (b + c)

contoh :

(2 + 6) + 8 = 2 + (6 + 8)

⇔ 8 + 8 = 2 + 14

⇔ 16 = 16


2. Pengurangan

tidak bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c tidak berlaku:

(a – b) – c ≠ a – (b – c).

contoh :

(3 - 5) - 7 ≠ 3 - (5 - 7)

⇔ -2 - 7 ≠ 3 - (-2)

⇔ -9 ≠ 5


3. Perkalian

Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c, berlaku :

(a x b) x c = a x (b x c)

contoh :

(2 x 5) x 6 = 2 x (5 x 6)

⇔ 10 x 6 = 2 x 30

⇔ 60 = 60.


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

a x (b + c) = (a x b) + (a x c)

contoh : 

–5 x [9 + (–1)] = [–5 x 9] + [–5 x (–1)] 

⇔ –5 x 8 = –45 + 5

⇔ –40 = –40


ii. distributif terhadap pengurangan

a x (b – c) = (a x b) – (a x c)

contoh :

7 x [–10 – 12] = [7 x (–10)] – [7 x 12]

⇔ 7 x -22 = -70 - 84

⇔ -154 = -154.


4. Pembagian

Tidak bersifat asosiatif, sehingga Untuk sembarang bilangan bulat a, b, dan c berlaku:

(a : b) : c ≠ a : (b : c)

contoh :

(8 : 4) : 2 ≠ 8 : (4 : 2)

⇔ 2 : 2 ≠ 8 : 2

⇔ 1 ≠ 4


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

(a + b) : c = (a : c) + (b : c)

contoh : 

(12 + 6) : 2 = (12 : 2) + (6 : 2)

⇔ 18 : 2 = 6 + 3

⇔ 9 = 9


ii. distributif terhadap pengurangan

(a – b) : c = (a : c) – (b : c)

contoh :

(21 – 9) : 3 = (21 : 3) – (9 : 3)

⇔ 12 : 3 = 7 - 3

⇔ 4 = 4

 

Semangat!

         Gerakan lokomotor adalah gerakan yang dilakukan oleh tubuh sehingga membuat tubuh berpindah tempat. Seperti yang terlah disebutkan diatas, gerakan-gerakan diatas adalah gerakan yang membuat tubuh bergerak berpindah tempat. Gerak lokomotor termasuk ke dalam salah satu gerakan tubuh manusia. Gerakan dasar tubuh manusia dibagi menjadi 3 macam, yaitu gerakan lokomotor, gerakan nonlokomotor, dan gerakan manipulatif.

         Berikut pembahasannya untuk masing-masing gerakan :

Gerakan Lokomotor

Gerakan nonlokomotor ialah, aktivitas yang dilakukan tubuh sehingga tubuh bergerak dan berpindah tempat. Contoh gerakan lokomotor antara lain :

BerjalanBerlariMelompatMeloncatMemanjat

     2. Gerakan Nonlokomotor

Gerakan nonlokomotor adalah gerakan yang dilakukan tubuh sehingga tubuh bergerak namun tidak berpindah tempat/tetap di tempat. Contoh gerakan nonlokomotor antara di antaranya :

JongkokMembungkukMenolehMengayun

     3. Gerakan Manipulatif

Gerakan manipulatif yaitu, gerakan yang dilakukan oleh manusia dengan melibatkan/menggunakan alat tertentu. Contohnya :

MelemparMenendangMenerima/Menangkap
