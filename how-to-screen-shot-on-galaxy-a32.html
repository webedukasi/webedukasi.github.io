[tex] |3x - 7| = |2x + 15| \\ { |3x - 7| }^{2} = { |2x + 15| }^{2} \\ {(3x - 7)}^{2} = {(2x + 15)}^{2} \\ {(3x - 7)}^{2} - {(2x + 15)}^{2} = 0 \\ (3x - 7 + 2x + 15)(3x - 7 - 2x - 15) = 0 \\ (5x + 8)(x - 22) = 0 \\ x = - \frac{8}{5} \: atau \: x = 22[/tex]

Penjelasan dengan langkah-langkah:

|3x - 7| = |2x + 15|

= 3x - 2x = 15 + 7

x = 22

|3x + 7| = |2x + 15|

= -3x + 2x = 15 - 7

-5x = 8

x = -8/5
