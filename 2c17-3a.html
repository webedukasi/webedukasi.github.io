7. a+2b+c=6. 3a-b+2c=17. 2a+b-c=9. Butuh cepat kk


Diketahui a + 2b + c = 6, 3a - b + 2c = 17, dan 2a + b - c = 9. Maka nilai dari a, b, dan c adalah 5, 0, dan 1.

Pendahuluan

Persamaan linear tiga variabel adalah suatu persamaan yang di dalamnya terdapat 3 nilai yang belum diketahui. Persamaan linear tiga variabel hanya dapat diselesaikan jika diketahui minimal 3 persamaan. Persamaan linear tiga variabel memiliki derajat tertinggi 1 pada setiap variabelnya.

Ada beberapa metode yang dapat digunakan untuk menyelesaikan suatu persamaan linear tuga variabel, yaitu:

Subtitusi: metode dengan memasukkan suatu persamaan ke dalam persamaan yang lain sehingga ditemukan satu persamaan baru. Eliminasi: metode dengan mengeliminasi salah satu variabel dari dua persamaan sehingga ditemukan suatu persamaan baru.

Dari penjelasan tersebut, mari kita selesaikan permasalahan di atas!

Pembahasan

Diketahui:

Persamaan 1: a + 2b + c = 6 Persamaan 2: 3a - b + 2c = 17 Persamaan 3: 2a + b - c = 9

Ditanyakan:

Nilai a, b, dan c.

Jawab:

1. Eliminasikan c pada persamaan 1 dan 3.

a + 2b + c = 6

2a + b - c = 9

-------------------- +

3a + 3b = 15

a + b = 5  

a = 5 - b ... (4)

2. Eliminasikan c pada persamaan 1 dan 2.

3a - b + 2c = 17

2a + 4b + 2c = 12

------------------------- -

a - 5b = 5 ... (5)

3. Subtitusikan persamaan 4 dan 5.

[tex]a - 5b = 5 \\ (5 - b) - 5b = 5 \\ 5 - 6b = 5 \\ -6b = 0 \\ \boxed{b = 0} [/tex]

4. Subtitusikan nilai b ke persamaan 4.

[tex] a = 5 - b \\ a = 5 - 0 \\ \boxed{a = 5}[/tex]

5. Subtitusikan nilai a dan b ke persamaan 1.

[tex] a + 2b + c = 6 \\ 5 + 2(0) + c = 6 \\ \boxed{c = 1} [/tex]

Jadi, nilai a, b, dan c adalah 5, 0, dan 1.

Dari persamaan pertama, kita dapatkan:
2((2b - 3c + 17) / 3) - b + 2c = -17
4b - 6c + 34 - 3b + 6c = -51

Dari persamaan ketiga, kita dapatkan:

-2b - 3c = -29
Substitusikan nilai b yang sudah kita dapatkan:
-2(-17) - 3c = -29
34 - 3c = -29
c = (34 + 29) / 3
c = 21/3
Jadi, HP dari SPLTV tersebut adalah:
HP = { (a, b, c) | a = (2b - 3c + 17) / 3, b = -17, c = 7 }
