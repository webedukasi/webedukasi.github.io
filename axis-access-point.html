8. WiFi card,access point,dan access controller merupakan 3 perangkat yang terdapat pada sebuah...​


1. Access Point (Titik Akses) adalah perangkat yang menghubungkan teknologi Wireless LAN dengan Ethernet yang terdapat di komputer. Titik akses memiliki kemampuan untuk melayan pengguna sebanyak 128 orang. Luas daerah yang dapat di jangkauoleh sebuah titik akses mencapai 25-100 meter.

2. Access Controller (Pengendali Akses) adalah perangkat yang berfungsi sebagai alat autentifikasi, mengecek apakah seorang pengguna merupakan orang yang mempunyai hak atau izin untuk melakukan akses.

3. Internet Link adalah perangkat yang menghubungkan lokasi hotspot dengan internet. Internet Link mempunyai kemampuan koneksi internet sampai kecepatan 512 kbps. Kemampuan koneksi ini digunakan untuk melayani seluruh pengguna dalam satu lokasi

20. Jelaskan beberapa mode pada access point


Jelaskan beberapa mode pada access pointJawabanPendahuluan

Sebelum membahas lebih lanjut mengenai mode pada access point, mari kita bahas terlebih dahulu apa itu access point. Access point adalah salah satu perangkat untuk jaringan komputer wireless atau tanpa kabel.

Pembahasan

Beberapa macam mode pada access point adalah sebagai berikut

1. Bridge Mode, mode ini digunakan untuk menyalurkan kembali suatu jaringan, namun dengan nama dan password yang berbeda.

2. Repeater mode, seperti dengan namanya, mode ini berfungsi untuk mengulang jaringan yang ada dengan tujuan agar sinyal dari jaringan yang diulang dapat lebih kuat dan mampu diterima dengan baik oleh klien.

3. Wireless Router Mode, pada mode ini access point berfungsi untuk membagikan internet dari suatu ISP, mode ini yang biasa dipakai oleh ISP Telkom dan lainnya.

Kesimpulan

Demikian pembahasan mengenai beberapa macam mode pada acces point, mulai dari Wireless Router Mode, Repeater Mode dan Bridge Mode. Access point sendiri merupakan salah satu perangkat pada jaringan komputer wireless. Untuk mempelajari materi lebih lanjut mengenai jaringan komputer wireless dapat dibaca pada link berikut

22. Bagaimana cara access point menciptakan jaringan WLAN


Penggunaan acces point pada jaringan WLAN adalah dengan melakukan pengaturan terlebih dahulu pada sebuah acces point. Untuk melakukan pengaturan, kita bisa menggunakan langkah sebagai berikut

Masuk ke control panelBerikan IP pada perangkat laptop kitaAkses IP default pada access point kita, biasanya tertulis pada bagian bawah acces pointLakukan pengaturan lebih lanjut pada acces point

Setelah access point memiliki alamat IP, kita bisa memberikan alamat IP pada perangkat-perangkat laptop atau komputer kita. Apabila IP bersifat static, maka kita harus memberikan IP satu per satu. Namun kita bisa melakukan pengaturan dengan merubahnya menjadi IP dynamic, perangkat akan mendapatkan IP secara otomatis dari perangkat access point.

Pembahasan

Setelah access point sudah terhubung dengan perangkat kita, kita bisa melakukan test dengan cara melakukan ping. Beberapa kemudahan setelah berhasil membuat jaringan WLAN adalah

Berbagi dataBerbagi printerBerbagi koneksi internetPelajari lebih lanjut

Demikian pembahasan mengenai cara access point menciptakan jaringan WLAN, access point berfungsi sebagai penghubung perangkat yang satu dengan perangkat lainnya. Untuk mempelajari lebih lanjut mengenai jaringan WLAN dapat dibaca pada link berikut
