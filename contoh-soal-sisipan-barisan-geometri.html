PEMBAHASAN :

Barisan geometri atau baris ukur merupakan barisan yang memenuhi sifat hasil bagi sebuah suku dengan suku sebelumnya yang berurutan, dan hal tersebut bernilai konstan karena memiliki rasio yang sama.

Di dalam soal disebutkan : Di antara bilangan 2 dan 1458 disisipkan 5 bilangan sehingga bilangan - bilangan semula dengan bilangan-bilangan yang disisipkan membentuk barisan geometri. Ini berarti secara otomatis bilangan 2 berfungsi sebagai suku awal (U1 atau a) dan 1458 berfungsi sebagai U7. Jika suku ke - n (Un) suatu barisan geometri ditentukan dengan rumus Un = a.rⁿ-¹ , maka dari nilai U7 dan U1 tersebut kita dapat menentukan rasionya.

U1 = a = 2


U7 = a.r^6

1458 = 2.r^6

r^6 = 1458 ÷ 2 = 729

r^6 = 3^6

r = 3


Setelah memiliki a dan r, kita dapat menentukan nilai suku - suku sisipannya.

U1 = a = 2

U2 = a.r = 2.3 = 6

U3 = a.r² = 2.3² = 18

U4 = a.r³ = 2.3³ = 54

U5 = a.r⁴ = 2.3⁴ = 162

U6 = a.r^5 = 2.3^5 = 486

U7 = a.r^6 = 2.3^6 = 1458

39. Diantara 3 dan 48 disisipkan 3 buah bilangan (suku) sehingga membentuk barisan geometri.jumlah deret geometri sesudah disisipkan itu adalah


Diantara 3 dan 48 disisipkan 3 buah bilangan (suku) sehingga membentuk barisan geometri.jumlah deret geometri sesudah disisipkan itu adalah 93.

Pendahuluan

Barisan geometri adalah barisan bilangan yg tiap sukunya diperoleh dari suku sebelumnya dg mengalikan atau membagi dg suatu bilangan tetap.

Rumus suku ke-n → [tex]\boxed {Un = a~r^{n-1}}[/tex]

Deret geometri adalah jumlah suku-suku yg ditunjuk oleh barisan geometri.

Jumlah n suku pertama → [tex]\boxed {S_{n} =\frac{a~( r^{n} - 1)}{r - 1}}[/tex]

Sisipan

Rasio baru → [tex]\boxed {r' = \sqrt[k+1]{\frac{y}{x} } }[/tex]

Pembahasan

Diketahui :

a = 3Un = 48k = 3

Ditanya :

Jumlah deret geometri sesudah disisipkan ?

Penyelesaian :

Menetukan rasio baru

Banyak suku yg dibentuk setelah disisipkan

n = k + 2

n = 3 + 2

n = 5

Un = a rⁿ⁻¹

48 = 3.r⁵⁻¹  

[tex]\displaystyle \frac{48}{3}[/tex] = r⁴  

16 = r⁴

2⁴ = r⁴

r = 2

Menentukan jumlah deret geometri sesudah disisipkan

Sn =  [tex]\displaystyle \frac{a(r^{n}-1)}{r-1}[/tex]

S₅ =  [tex]\displaystyle \frac{3(2^{5}-1)}{2-1}[/tex]

    = 3 (2⁵ - 1)

    = 3 × 31

    = 93

Jadi jumlah deret geometri sesudah disisipkan itu adalah 93.
