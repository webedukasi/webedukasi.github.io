19. kerjakan soal berikut? [2sinx dx. ​


Jawaban:

Integral dari 2sinx dx dapat dihitung sebagai berikut:

∫2sinx dx = -2cosx + C

dengan C adalah konstanta integrasi.

Penyelesaian:

Kita ketahui bahwa turunan dari cosx adalah -sinx, sehingga turunan dari -cosx adalah sinx. Dengan demikian, untuk mencari integral dari 2sinx dx, kita dapat menggunakan aturan dasar integral trigonometri, yaitu:

∫ sinx dx = -cosx + C

Karena integral 2sinx dx sama dengan 2 kali integral sinx dx, maka kita dapat menuliskannya sebagai:

∫2sinx dx = 2(∫ sinx dx)

∫2sinx dx = 2(-cosx + C)

∫2sinx dx = -2cosx + C'

dengan C' adalah konstanta integrasi baru.

Jadi, jawaban dari integral ∫2sinx dx adalah -2cosx + C'.

40. Soal Matematika Materi : Integral Trigonometri_______________________[tex]f'(x) = \frac{sin2x}{ 1 - 2sinx} [/tex]f(x) = ?​


Jawab:

[tex]\displaystyle =\frac{1-2\sin x-\ln|1-2\sin x|}{2}+C[/tex]

Penjelasan dengan langkah-langkah:

Ini hanya integral substitusi

[tex]\displaystyle \int \frac{\sin 2x}{1-2\sin x}~dx=\int \frac{2\sin x\cos x}{1-2\sin x}~dx\\u=\sin x\\du=\cos x~dx\\=2\int \frac{u\cancel{\cos x}}{1-2u}~\frac{du}{\cancel{\cos x}}\\=2\int \frac{u}{1-2u}~du\\v=1-2u\rightarrow u=\frac{1-v}{2}\\dv=-2~du\\=2\int \frac{\frac{1-v}{2}}{v}~\frac{dv}{-2}\\=-\frac{1}{2}\int \frac{1-v}{v}~dv\\=\frac{1}{2}\int \frac{v-1}{v}~dv\\=\frac{1}{2}\int \left ( 1-\frac{1}{v} \right )dv\\=\frac{1}{2}(v-\ln|v|)+C\\=\frac{1-2u-\ln|1-2u|}{2}+C\\[/tex]

[tex]\displaystyle =\frac{1-2\sin x-\ln|1-2\sin x|}{2}+C[/tex]

Diketahui : Turunan fungsi trigonometri

Ditanya : Fungsi awal

JAWAB :

Bentuk bisa diubah menjadi :

f(x) =

[tex]∫ \frac{sin2x}{1 - 2sinx} \: dx[/tex]

[tex]∫ \frac{2sinxcosx}{1 - 2sinx} \: dx[/tex]

Misalkan, 1 - 2sinx = t

Maka,

[tex]sinx = \frac{1 - t}{2} [/tex]

Apabila diturunkan, dx(1 - 2sinx) = dt(t)

dx(-2cosx) = dt

[tex]dx = - \frac{dt}{2cosx} [/tex]

[tex]∫ \frac{2( \frac{1 - t}{2}) cosx}{t} \times - \frac{dt}{2cosx} [/tex]

[tex] - ∫ \frac{1 - t } t \times \frac{dt}{2} [/tex]

[tex] - \frac{1}{2} ∫ \frac{1 - t } t dt[/tex]

[tex] - \frac{1}{2} ∫ \frac{1}{t} - 1 \: dt[/tex]

[tex] - \frac{1}{2} (ln |t| - t) + c[/tex]

[tex] = \frac{t - ln |t|}{2} + c[/tex]

[tex] = \frac{1 - 2sinx - ln |1 - 2sinx| }{2} + c[/tex]
