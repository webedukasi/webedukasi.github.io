10. Resep kue mengharuskan suhu oven 180 derajat Celsius. Jika oven yang digunakan hanya memiliki skala Fahrenheit, berapakah suhu yang harus diatur pada oven tersebut?


Jawab:

Penjelasan dengan langkah-langkah:

Konversi suhu dari Celsius (°C) ke Fahrenheit (°F) :

°F = (9/5 × t°C) + 32

Maka :

(9/5 × 180°C) + 32

(9 × 36) + 32

324 + 32 = 356°F.

Jadi, suhu yang harus diatur pada oven tersebut adalah 356°F.

14. Diketahui suhu dalam oven adalah 80 o Reamur. Bila dinyatakan dalam Celcius, berapakah suhu dalam oven tersebut?​


Jawaban:

100°c

Penjelasan:

Untuk mengkonversi suhu dari Reamur ke Celcius, kita dapat menggunakan rumus berikut:

Suhu dalam Celcius = (Suhu dalam Reamur × 5/4)

Dalam kasus ini, suhu dalam oven adalah 80 Reamur. Mari kita gunakan rumus tersebut untuk menghitung suhu dalam Celcius:

Suhu dalam Celcius = (80 × 5/4) = 100°C

Jadi, suhu dalam oven tersebut adalah 100 derajat Celsius.

21. Suatu industri rumah tangga memproduksi roti. Untuk membakar roti industri tersebut memiliki 3 buah oven. Oven A, oven B, dan oven C. Peluang roti akan bantet jika di panggang di oven A sebesar 3%, di oven B sebesar 10%, dan dioven C sebesar 5%. Peluang menggunakan oven A 2 kali lebih besar dari oven B, dan 3 kali lebih besar dari oven C. Tentukan peluang roti yang diproduksi tidak bantet ?


Diketahui :

1). Peluang roti bantet di oven a adalah 3%

2). Peluang roti bantet di oven b adalah 10%

3). Peluang roti bantet di oven c adalah 5%

4). Peluang menggunakan oven a 2x lipat oven B

5). Peluang menggunakan oven a 3x lipat oven C

Ditanya :

Peluang roti yang diproduksi tidak bantet

Perencanaan rumus :

1). Menghitung peluang roti dimasukkan ke oven a, b, dan c

2). Menghitung kemungkinan roti dimasukkan kedalam oven a dan tidak bantet, dimasukkan kedalam oven b dan tidak bantet, dan dimasukkan kedalam oven c dan tidak bantet

3). Ditambahkan

Penyelesaian rumus :

1). Menghitung peluang roti dimasukkan ke oven a, b, dan c

pertama, kita buat perbandingannya oven a, b, dan c berturut-turut

6 : 3 : 2

Lalu, ditambahkan

6 + 3 + 2 = 11

Sehingga, peluang:

Masuk ke oven a = 6/11

Masuk ke oven b = 3/11

Masuk ke oven c = 2/11

2). Menghitung kemungkinan roti dimasukkan kedalam oven a dan tidak bantet, dimasukkan kedalam oven b dan tidak bantet, dan dimasukkan kedalam oven c dan tidak bantet

Roti a dan tidak bantet :

(100 - 3)% x 6/11

= 97% x 6/11

= 97/100 x 6/11

= 582/1100

= 291/550

roti b dan tidak bantet :

(100 - 10)% x 3/11

= 90% x 3/11

= 90/100 x 3/11

= 9/10 x 3/11

= 27/110

roti c dan tidak bantet :

(100 - 5)% x 2/11

= 95% x 2/11

= 95/100 x 2/11

= 19/20 x 2/11

= 38/220

= 19/110

3). ditambahkan

291/550 + 27/110 + 19/110

= 291/550 + 135/550 + 95/550

= 521/550

Kesimpulan :

Jadi, peluang roti tidak bantet adalah 521/550

38. Jika suhu oven 550°F berapakah suhu oven jika diubah ke °C


Mapel : Fisika { IPA }
Kelas : 7 SMP/ VII SMP
Bab : Konversi Suhu 

Pembahasan/Penjelasan :
Rumus Mencari F -> C 
Yaitu :
Suhu Celcius :
°C = { Suhu Fahrenheit - 32 }. 5/9
°C = { 550 - 32 }. 5/9
°C = 518 . 5/9 
°C = 287,777
Jadi, 550°F = 287,777°C
