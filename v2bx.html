33. Limit X-2 √x+₂-V2X X-2​


Jawaban:

Untuk menyelesaikan batas fungsi yang diberikan, kita dapat mengganti nilai x dengan nilai batasnya, yaitu mendekati -2. Mari kita substitusikan nilai batasnya:

Lakukan substitusi X = -2 + h, dengan h mendekati nol saat X mendekati -2.

Sehingga fungsi menjadi:

lim h→0 [(h - 2)√(-2 + h + 2) - √2(-2 + h)] / (h - 2)

= lim h→0 [h√h - √2(h - 2)] / (h - 2)

Kita dapat menerapkan aturan L’Hopital untuk menyelesaikan batas ini. Aturan L’Hopital memungkinkan kita untuk mengambil turunan dari pembilang dan penyebut batas fungsi tersebut terlebih dahulu. Mari kita lakukan turunan pada pembilang dan penyebut:

Bilangan:

d/dh [h√h - √2(h - 2)] = √h + (1/2)h^(-1/2) - √2

= √h + (√2/2)h^(-1/2) - √2

= √h - (√2/2)√(1/h) - √2

Penyebut:

d/dh [h - 2] = 1

Kita dapat mengambil batas turunan ini saat h mendekati 0:

lim h→0 [√h - (√2/2)√(1/h) - √2 ] / 1

= √0 - (√2/2)√(1/0) - √2

Sebagai akibatnya, batas di atas tidak terdefinisi secara matematika. Oleh karena itu, batas dari fungsi tersebut di x mendekati -2 tidak ada.
