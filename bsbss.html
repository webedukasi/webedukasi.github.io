|2 - x| = |2x + 5|

(2 - x + (2x + 5)).(2 - x - (2x + 5)) = 0

(x + 7).(-3x - 3) = 0

Maka,

x + 7 = 0

x = -7

Atau

-3x - 3 = 0

-3x = 3

x = -3/3

x = -1

Karena penyelesaiannya dalam bentuk p dan q dan diketahui p > q, maka p = -1 dan q = -7, sehingga:

2p - q = 2.(-1) - (-7) = -2 + 7 = 5
