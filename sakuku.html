16. uang saku rina 5/8 dari uang saku desi.jika selisih uang saku mereka rp 6.000,00 berapa jumlah uang saku mereka


Uang saku rina adalah ⁵/₈ dari uang saku desi. Jika selisih uang saku mereka Rp. 6.000. Maka jumlah uang saku mereka adalah Rp. 66.000.

Penyelesaian Soal :

Misalkan Uang saku Rina = x

               Uang saku Desi = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Uang saku rina adalah ⁵/₈ dari uang saku desi" yaitu :

x = ⁵/₈y   ...(Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Selisih uang saku mereka Rp. 6.000" yaitu :

y - x = 6.000   ...(Persamaan 2)

LANGKAH KETIGA (III)

Subtitusikan nilai x pada persamaan 2 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

y - x = 6.000

y - ⁵/₈y = 6.000

³/₈y = 6.000

3y = 6.000 × 8

3y = 108.000

y = [tex]\frac{108.000}{3}[/tex]

y = 36.000

LANGKAH KEEMPAT (IV)

Subtitusikan nilai y pada persamaan 2 sehingga diperoleh nilai x dengan menggunakan cara sebagai berikut :

y - x = 6.000

36.000 - x = 6.000

-x = 6.000 - 36.000

-x = -30.000

x = 30.000

LANGKAH KELIMA (V)

Hitung jumlah uang saku rina dan desi dengan menggunakan cara sebagai berikut :

x + y = Rp. 30.000 + Rp. 36.000

        = Rp. 66.000

∴ Kesimpulan jumlah uang saku rina dan desi adalah sebesar Rp. 66.000.
