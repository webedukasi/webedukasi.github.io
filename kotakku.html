6. Jumlah kotak C dan kotak D adalah 84. Jika Perbandingan kotak C : kotak D adalah 3 : 4 ,Maka banyak kotak C dan kotak D masing – masing adalah….​


Jawab:

Banyak kotak C dan kotak D masing–masing adalah:

36 kotak C, dan 48 kotak D.

Penjelasan dengan langkah-langkah:

Jumlah kotak C dan kotak D adalah 84.

Perbandingan kotak C : kotak D adalah 3 : 4.

Jika perbandingan tersebut dinyatakan dalam c : d, maka banyak kotak C dapat dihitung seperti berikut ini.

[tex]\large\text{$\begin{aligned}&n_C=\frac{c}{c+d}\times n_{C+D}\\&\quad\,=\tfrac{3}{7}\times84\\&\quad\,=3\times12\\&\quad\,=\bf36\ kotak\end{aligned}$}[/tex]

Sedangkan banyak kotak D bisa diperoleh dari mengurangkan banyak kotak C dari jumlah banyak kotak C dan kotak D, sehingga:

Banyak kotak D = 84 – 36 = 48 kotak

Atau dengan cara seperti perhitungan banyak kotak C di atas.

[tex]\large\text{$\begin{aligned}&n_D=\frac{d}{c+d}\times n_{C+D}\\&\quad\,=\tfrac{4}{7}\times84\\&\quad\,=4\times12\\&\quad\,=\bf48\ kotak\end{aligned}$}[/tex]

∴  Dengan demikian, banyak kotak C dan kotak D masing–masing adalah 36 kotak C dan 48 kotak D.

9. 1. sebagian dari 11 kotak berukuran besar memuat 8 kotak ukuran sedang. Sebagian dari kotak-kotak berukuran sedang ini memuat 8 kotak berukuran kecil. Dari semua kotak yang ada 102 kotak di antaranya ternyata kosong. Berapa jumlah keseluruhan kotak yang ada?


Jika dari semua kotak yang ada, 102 kotak di antaranya ternyata kosong. maka jumlah keseluruhan kotak adalah 115 kotak.

Pembahasan

Menyelesaikan Soal Cerita Yang Berhubungan Persamaan Bentuk Aljabar.

Yang dimaksud dengan persamaan bentuk aljabar adalah sebuah bentuk persamaan yang terdiri dari bentuk-bentuk aljabar. Sebuah bentuk aljabar biasanya memuat unsur-unsur, antara lain :

Variabel, yaitu bentuk bilangan yang dinyatakan dengan huruf dan nilainya bisa berubah tergantung koefisien dan bilangan yang diwakilinya. Koefisien, yaitu bilangan yang memiliki variabel. Konstanta, yaitu bilangan tetap yang tidak memiliki variabel

Pada soal di atas, diketahui :

Sebagian dari 11 kotak berukuran besar memuat 8 kotak ukuran sedang.

Sebagian dari kotak-kotak berukuran sedang ini memuat 8 kotak berukuran kecil.

Dari semua kotak yang ada, 102 kotak di antaranya ternyata kosong.

Ditanya :

Jumlah keseluruhan kotak.

Kita mulai menyelesaikan soal ini dengan membuat bentuk persamaan aljabar berdasarkan soal di atas.

Misalkan banyak kotak besar yang memuat 8 kotak sedang adalah a, maka banyak kotak sedang yang dimuat ke dalam kotak besar adalah 8a. Dengan demikian banyak kotak besar yang kosong adalah 11 - a.

Kemudian misalkan banyak kotak sedang yang memuat kotak kecil adalah b, maka banyak kotak kecil yang dimuat ke dalam kotak sedang adalah 8b. Dengan demikian banyak kotak sedang yang kosong adalah 8a - b.

Adapun kotak kecil semuanya kosong.

Dari seluruh kotak kosong yang berjumlah 102, dapat kita buat persamaan :

102 = (11 - a) + (8a - b) + 8b

Sekarang kita selesaikan persamaan di atas dengan cara :

1) Menjumlahkan suku sejenis.

[tex]102 = 11 - a + 8a - b + 8b \\

102 = 11 + 7a + 7b[/tex]

2) Memindahkan konstanta ke sebelah kiri dan tandanya diubah.

[tex]102 - 11 = 7a + 7b [/tex]

3) Menghitung selisih.

[tex]91 = 7a + 7b \\ [/tex]

4) Membagi kedua ruas dengan 7

[tex]91 : 7 = 7a : 7 + 7b : 7 \\

13 = a + b[/tex]

5) Memindahkan posisi variabel ke sebelah kanan dan konstanta ke sebelah kiri

[tex]a + b = 13 \\ [/tex]

6) Menghitung keseluruhan kotak

Misalkan jumlah seluruh kotak adalah n, maka :

[tex]n = 11 + 8a + 8b \\

n = 11 + 8(a + b)[/tex]

Masukkan nilai a + b yang sudah dicari di atas ke dalam persamaan.

[tex]n = 11 + 8(13) \\

n = 11 + 104 \\

n = 115[/tex]

Jadi jumlah seluruh kotak adalah 115 kotak.

sb = 3. sk

sb = 3. 5 cm

→ Jumlah kotak kecil yang dapat dimasukkan

n = Vb / Vk

n = sb³ / sk³

n = (sb / sk)³

n = (15 cm / 5 cm)³

n = 3³

40. Jika ada 4 kotak,kotak pertama ditambah kotak kedua = 8, Kotak pertama tambah kotaK ketiga = 13, kotak kedua tambah kotak ke 4 = 8,dan kota ketiga dikurang kota ke 4 = 6.maka berapakah angka yang ada pada masing-masing keempat kotak tersebut??


Mapel : Matematika
Kelas : VIII SMP
Bab : Aljabar

Pembahasan :
Misal kotak 1 = A
Kotak 2 = B
Kotak 3 = C
Kotak 4 = D

A + B = 8 ——» A = 8 - B
A + C = 13 ——» A = 13 - C
B + D = 8 ——» D = 8 - B
C - D = 6 ——» D = C - 6

Buat Persamaan...
A = A
8 - B = 13 - C
C - B = 5

D = D
8 - B = C - 6
B + C = 14

Eliminasi...
B + C = 14
C - B = 5
—————— (+)
2C = 19
C = 19/2

Substitusi...
D = C - 6
D = 19/2 - 6
D = (19 - 12)/2
D = 7/2

Substitusi...
B + D = 8
B = 8 - D
B = 8 - (7/2)
B = (16 - 7)/2
B = 9/2

Substitusi...
A = 8 - B
A = 8 - (9/2)
A = (16 - 9)/2
A = 7/2

Maka...
Kotak Pertama = 7/2
Kotak Kedua = 9/2
Kotak Ketiga = 19/2
Kotak Keempat = 7/2
