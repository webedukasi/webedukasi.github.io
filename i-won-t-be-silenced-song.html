33. if i won the competition. lili woul be envious me. ubahlah menjadi conditional type 3​


║CONDITIONAL SENTENCE-MIXED TIME║

Khusus untuk kalimat no.1 terdapat konteks yang missing, maka, saya buat 2 kondisi fakta, baik kondisi fakta dalam conditional type 3 dan mixed conditional.

1. Fact { "The weather is bad. I didn't go to Purwokerto yesterday "} => penerapan mixed conditional.

Fact {" I didn't go to Purwokerto because the weather was bad yesterday" } => penerapan conditional tipe ke 3

2. Conditional sentence => If my servant weren't sick, I wouldn't do the house hold by myself.

Pembahasan Conditional tipe ke 3

Pernyataan no.1 merupakan kalimat conditional/pengandaian. Maka, yang harus dicari adalah kondisi faktanya.

Seharusnya kalimat conditional nya tertulis sedemikian untuk type 3 :

"If the weather had been good yesterday I would have gone to Purwokerto."

Rumus utama Conditional tipe ke 3 yaitu If+past perfect, past future perfect.

Maka, bentuk kondisi fakta dalam conditional type ke 3 menggunakan pernyataan berstruktur simple past tense karena conditional tipe ke 3 merujuk pada kejadian yang tidak terwujud di waktu lampau=> "I didn't go to Purwokerto because the weather was bad yesterday."

Pembahasan mixed conditional ( kalimat conditional campuran )

Seharusnya keterangan waktunya "yesterday", tertulis dan diletakkan dalam klausa utamanya.

"If the weather was good, I would have gone to Purwokerto yesterday."

Pernyataan diatas merupakan bentuk mixed conditional (kalimat conditional campuran), dimana bentuk verb (kata kerja) dalam -if clause "was" dan bentuk verb dalam main clause "would have gone" adalah berbeda tenses. Maka, kondisi fakta nya akan berbentuk pernyataan campuran tenses => Present & Past.

Maka, untuk faktanya (fact) jadi tidak terstruktur , dimana kalimat pertama menggunakan simple present dan yang kedua menggunakan simple past => The weather is not good, I didn't go to Purwokerto yesterday.

Untuk mengetahui bahwa kondisi faktanya dan bentuk conditional nya campuran adalah dengan berpicu pada pola utama dari ke 3 bentuk -if conditional sentence.

2. My servant is sick now so i do the house hold by myself.

Pernyataan no.2 merupakan penjabaran dari kondisi faktanya. Maka, yang dicari adalah bentuk kalimat -if conditional nya dan ini tidak termasuk kategori mixed conditional. Ini terlihat jelas bahwa keseluruhan struktur pernyataan kondisi faktanya terstruktur dengan menggunakan pola simple present tense. Cermati bentuk kata kerja dalam -if clause nya "is" dan main clause "do."

Dikarenakan fakta nya berbentuk pernyataan yang menggunakan struktur simple present. Maka, kalimat conditional yang digunakan berbentuk simple past. Ini merupakan penerapan conditional type ke 2, dimana kondisi fakta yang terjadi saat ini, berbanding terbalik dari apa yang diucapkan.

-IF Conditional sentence structure consists -If clause and result clause ( main clause ).

→ 3 main types of Conditional Sentences←

TYPE 1 CONDITIONAL

The Type 1 Conditional is used to describe the truth in the present/future ( The planning (prediction) will be probably happened ). In fact, the speaker will actualize his/her dream now or soon .

Pattern :

If +S+ verb (s/es) + O, S+will+verb1+O/C

Example :

If I study hard, I will pass the exam easily.

Analysis

→ It uses simple future form in its main clause / result clause to describe the possible condition and state the prediction will be true.

TYPE 2 CONDITIONAL

The Type 2 Conditional type is used to describe the untruth in the present / future. The speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future.

Pattern:

If+S+verb2+O/C , S+would+verb1+O/C

Example :

If I were a president, I would build the largest library in the Southeast Asia.

Analysis :

→ It uses the simple past form to reflect the untruth condition in the present and the speaker's intention will not come true because it is contrary to the real condition for now or maybe future. Meaning : I can't build the largest library because I'm not a president now.

TYPE 3 CONDITIONAL

The Type 3 Conditional Is used to describe the untruth in the past. It is contrary to the fact in the past and It reflects a regret for something that the speaker couldn't afford to do in the past.

Pattern :

If+S+Had+Verb 3+O , S+Would+have+Verb 3+O

Example :

If I had known my friend's house address correctly, I wouldn't have been lost.

Meaning : I didn't know my friend's house address so I was lost.
