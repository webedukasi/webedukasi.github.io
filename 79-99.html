4. Denyut Nadi per menit 60-69 70-79 80-89 90-99 100-109 Frekuensi 2,8,5,4​


Jawaban:

Untuk menghitung jumlah data yang ada, kita dapat menjumlahkan frekuensi dari setiap kategori:

Jumlah data = 2 + 8 + 5 + 4 = 19

Untuk menghitung mean atau rata-rata, kita dapat menggunakan formula:

mean = (jumlah nilai x frekuensi) / jumlah data

Sebagai contoh, untuk kategori 60-69, jumlah nilainya adalah 65 ((60+69)/2), dan frekuensinya adalah 2. Oleh karena itu, kita dapat menghitung rata-rata untuk setiap kategori seperti ini:

rata-rata (mean) untuk kategori 60-69 = (65 x 2) / 19 = 6,84

rata-rata (mean) untuk kategori 70-79 = (75 x 8) / 19 = 31,58

rata-rata (mean) untuk kategori 80-89 = (85 x 5) / 19 = 22,37

rata-rata (mean) untuk kategori 90-99 = (95 x 4) / 19 = 20

rata-rata (mean) untuk kategori 100-109 = (105 x 0) / 19 = 0

Jadi, rata-rata denyut nadi per menit berdasarkan data yang diberikan adalah 6,84 untuk kategori 60-69, 31,58 untuk kategori 70-79, 22,37 untuk kategori 80-89, 20 untuk kategori 90-99, dan 0 untuk kategori 100-109.

Operasi hitung bilangan ada 4 yaitu:

PerkalianPembagianPenjumlahanPengurangan

Rumus ke 4 operasi di atas adalah:

1. Perkalian

Rumus:

- a × b = - a (misalnya)

- a × - b = a (misalnya)

a × - b = - b (misalnya)

2. Pembagian

Rumus:

- a ÷ b = - a (misalnya)

- a ÷ - b = - b (misalnya)

a ÷ - b = - a (misalnya)

3. Penjumlahan

Rumus:

- a + b = a - b

a + - b = a - b

- a + - b = - a - b = a + b

4. Pengurangan

Rumus:

- a - b = a + b

a - - b = a + b

- a - - b = - a + b = a - b
