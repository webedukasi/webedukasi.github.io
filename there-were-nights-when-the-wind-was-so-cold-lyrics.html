4. (Angin sangat kencang sehingga jembatan rubuh)

If the wind wasn't so strong, the bridge wouldn't collapse.

(Jika angin tidak bertiup sangat kencang, jembatan tidak akan rubuh.)

5. (Aku tidak dapat menelepon Sally karena aku telah kehilangan nomer teleponnya)

If I hadn't lost Sally's number, I would have called her.

(Jika aku tidak telah kehilangan nomer telepon Sally, aku pasti sudah meneleponnya.)

Pembahasan:

4. If the wind wasn't so strong, the bridge wouldn't collapse. = Conditional Sentence type 2, karena sangat kecil kemungkinan terjadi. Angin bertiup sangat kencang atau tidak bukanlah kuasa manusia, sehingga kita hanya dapat berangan-angan jika angin bertiup tidak sangat kencang.

5. If I hadn't lost Sally's number, I would have called her. = Conditional Sentence type 3, karena berlawanan dengan yang sudah terlanjur terjadi yaitu aku telah kehilangan nomer telepon milik Sally, sehingga aku tidak dapat meneleponnya.

28. This sailboat (will) (would) go faster if there were more wind


jawabannya :
would, karena kalimat tersebut berbentuk lampau dengan ditandai to be-nya were.

MAPEL : BAHASA INGGRIS

MATERI : CONDITIONAL SENTENCE

KELAS : 9

KODE SOAL : 9.5

KODE KATEGORISASI : 9.5.3


SOAL DAN PEMBAHASAN

=========================

This sailboat (will) (would) go faster if there were more wind.


The answer is :

This sailboat (would) go faster if there were more wind.


Kalimat di atas merupakan kalimat conditional sentence type 2, hal ini bisa di lihat dari kalimat pengandaiannya menggunakan to be past yaitu "were" di kalimat "if there were more wind", karena nya menggunakan WOULD + V1.

Rumus :

if + Simple Past , Conditional I (= would + Infinitive)
