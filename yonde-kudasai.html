1. arti dari Yonde Kudasai, Itte Kudasai, dan Mite Kudasai??


よんで ください Yonde Kudasai.

Bacalah! / Ayo Baca!

いって ください Itte Kudasai.

Pergilah! / Silahkan pergi!

みて ください Mite Kudasai.

Lihatlah! / Ayo Lihat!

Pembahasan

KaIimat Perintah daIam Bahasa Jepang terdiri dari beberapa kriteria sesuai penempatannya. Ada yang bentuknya sopan dan ada juga yang bentuknya kasar. Untuk bentuk sopan biasanya menggunakan bentuk ください (Kudasai) & ちょうだい (Choudai). Untuk kata yang tergoIong kasar biasanya menggunakan bentuk せ (se), ろ (ro), dan え (e).

Penggunaan Bentuk ください

Secara umum kata tersebut memiliki arti ToIong, Mohon, dan sering digunakan untuk menggantikan akhiran -lah sebagai bentuk perintah serta akhiran ini bersifat sopan. Sebelum menambahkan akhiran ini kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh:

ToIong Bantu dia:

Membantu dalam Bahasa Jepang 助けつ (Tasuketsu). diubah ke bentuk -te menjadi 助けて (Tasukete).Dia dalam bahasa jepang 彼 (Kare) + partikel を (o).

Sehingga polanya:

S + Kata Kerja bentuk -te + Kudasai.

= Kare o + Tasukete + Kudasai.

= 彼を助けてください

Penggunaan bentuk ちょうだい

Secara umum kata tersebut memiliki fungsi yang sama dengan kudasai cuma lebih ke bentuk permohonan. Cara menggunakan pola Choudai juga sama dengan kudasai yaitu kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh:

Mohon Kerjakan PR-ku:

Kerjakan dalam Bahasa Jepang します (Shimasu). diubah ke bentuk te menjadi して (Sh|te).PR dalam Bahasa Jepang 宿題 (Shukudai) ditambah partikel を (o).

Sehingga polanya:

O + Kata kerja bentuk -te + Coudai.

= Shukudai o + sh|te + Coudai.

= 宿題をしてちょうだい

Penggunaan bentuk せ (se), ろ (ro) dan え (e).

Penggunaan bentuk -se dan -e dilakukan pada Kata Kerja 1 sedangkan -ro pada kata kerja 2. Akhiran ini berfungsi sebagai akhiran kalimat perintah yang sifatnya marah atau membentak. Untuk membuat pola kalimatnya kata kerja masu harus diubah ke bentuk se, ro, atau e. Contoh:

Pergilah!!

Pergi dalam Bahasa Jepang 行く (Iku). Nah, pergi termasuk Kata Kerja 1 maka harus diubah ke bentuk e atau se. Maka, ketika diubah menjadi 行け！(Ise!)

5. perbedaan bentuk "te kudasai" dan "o kkerja kudasai" ?


Perbedaan bentuk 'te kudasai' dan 'o∼kudasai' adalah bentuk 'o∼kudasai' merupakan bentuk permintaan atau permohonan yang lebih sopan daripada bentuk 'te kudasai'. Kedua bentuk tersebut sama-sama sopan, namun bentuk 'o∼kudasai' lebih sopan lagi.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Untuk membuat kalimat permintaan, permohonan, atau perintah secara halus dalam Bahasa Jepang, bisa menggunakan bentuk 'te kudasai' dan 'o∼kudasai' ini. Untuk memahami lebih lanjut mengenai kedua bentuk ini bisa disimak pembahasan di bawah.

てください (Te kudasai)

Untuk membuat kata kerja bentuk 'te kudasai', cara nya dengan mengubah kata kerja nya ke bentuk Te terlebih dahulu, kemudian ditambahkan kata Kudasai yang berarti "Mohon", "Tolong", dsb. Untuk cara mengubah kata kerja ke bentuk Te bisa dilihat di link yang saya berikan di bawah nanti.

Pola : KK(Bentuk Te) + Kudasai

Contoh kalimat :

Suwatte kudasai = Silahkan dudukYamete kudasai = Mohon hentikanYurushιte kudasai = Tolong maafkan

お～ください (O∼kudasai)

Untuk membuat kata kerja bentuk 'o∼kudasai', cara nya hanya dengan meletakkan kata kerja bentuk stem (bentuk masu tanpa menggunakan masu nya) di antara kata O dan Kudasai. Alasan kenapa bentuk ini lebih sopan dari bentuk 'te kudasai' adalah karena menggunakan honorifik (bahasa penghormatan), yaitu O di awal kata.

Pola : O + KK(Bentuk stem) + Kudasai

Contoh kalimat :

Osuwari kudasai = Silahkan dudukOyame kudasai = Mohon hentikanOyurushi kudasai = Tolong maafkan

10. 1. jelaskan kata "kudasai"NO NGASAL ✅NO COPAS ✅note: entah kenapa saya tidak paham dengan kata "kudasai"​


Kudasai merupakan suatu ungkapan permohonan. Kalau dalam bahasa Indonesia, kata kudasai artinya seperti "tolong", "mohon", dsb. Kata ini juga bisa digunakan sebagai bentuk perintah yang sopan.

Contoh kalimat menggunakan kudasai ↴

Yamete kudasai = Mohon hentikanOshiete kudasai = Tolong ajarkanShizuka ni shlte kudasai = Mohon tenang/diamKeeki o hitotsu kudasai = Tolong kuenya satuRingo o ikko kudasai = Tolong apelnya satu

Selain kudasai, ada kata lain yang memiliki arti dan penggunaan yang sama. Kata tersebut adalah kure. Perbedaannya adalah kata kure lebih kasar dari kudasai dan biasa digunakan laki-laki.

Jawaban:

Saya jawab pakai romaji saja ya, biar enak dibacanya..

-----------------------------------------------------------------------------------------------------------

[Perubahan K. Kerja bentuk ます menjadi K.Kerja bentuk て]

Dibagi menjadi beberapa kelompok yaitu:

1. Berakhiran huruf berikut sebelum Masu い(I), ち(Chi), り(Ri)

Maka ketiga huruf tersebut harus diganti dengan って(tte) dan ます(Masu) dihilangkan.

Contoh:

Kaimasu (Katte) Membeli

Tachimasu (Tatte) Berdiri

Machimasu (Matte) Menunggu

Hairimasu (Haitte) Masuk

-----------------------------------------------------------------------------------------------------------

2. Berakhiran huruf berikut sebelum Masu み(Mi), び(Bi), に(Ni)

Maka ketiga huruf tersebut harus diganti dengan んで(nde) dan ます(Masu) dihilangkan.

Contoh:

Yomimasu (Yonde) Membaca

Yobimasu (Yonde) Memanggil

Shinimasu (Shinde) Meninggal

*Bentuk て dari yomimasu dan yobimasu memang sama, tetapi tidak akan membuat salah paham karena kanjinya beda dan apa yang berkaitan dengan kata kerja tsb pasti beda.

-----------------------------------------------------------------------------------------------------------

3. Berakhiran dengan huruf し(Shi)

Maka cukup mengganti Masu dengan て(Te)

Contoh:

Hanashimasu (Hanashite) Berbicara

Kashimasu (Kashite) Meminjamkan

-----------------------------------------------------------------------------------------------------------

4. Kata kerja yang hurufnya bervokal え(E) sebelum Masu

Maka cukup mengganti Masu dengan て(Te)

Contoh:

Oshiemasu (Oshiete) Memberi tahu/Mengajari

Akemasu (Akete) Membuka

Tsukemasu (Tsukete) Menyalakan

Kaemasu (Kaete) Menukar

-----------------------------------------------------------------------------------------------------------

5. Berakhiran き(Ki) dan ぎ(Gi)

Pembahasan

Kalimat Perintah dalam Bahasa Jepang terdiri dari beberapa kriteria sesuai penempatannya. Ada yang bentuknya sopan ada juga yang bentuknya kasar. Untuk bentuk sopan biasanya menggunakan akhiran ください (Kudasai) dan ちょうだい (Choudai). Untuk bentuk yang tergolong kasar biasanya menggunakan akhiran せ (se), ろ (ro), dan え (e).

Penggunaan akhiran ください

Secara umum kata tersebut memiliki arti Tolong, Mohon, dan sering digunakan untuk menggantikan akhiran -lah sebagai bentuk perintah serta akhiran ini bersifat sopan. Sebelum menambahkan akhiran ini kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh :

Tolong Tunggu Sebentar

Tunggu dalam Bahasa Jepang 待つ (Matsu) diubah ke bentuk -te menjadi 待って (Matte)Sebentar dalam Bahasa Jepang ちょっと (Chotto)

Sehingga polanya :

Kata Kerja bentuk -te + Kudasai

= Chotto Matte + Kudasai

= ちょっと 待って ください

Penggunaan akhiran ちょうだい

Secara umum kata tersebut memiliki fungsi yang sama dengan kudasai namun lebih ke bentuk permohonan.

. Cara menggunakan pola Choudai juga sama dengan kudasai yaitu kata kerja kerja masu (ます) harus diubah ke bentuk te (て). Contoh :

Mohon beri anak saya makan

Beri makan dalam Bahasa Jepang 養っ (Yashinasu) diubah ke bentuk -te menjadi 養って(Yashinate)Anak saya dalam Bahasa Jepang 私の子供 (Watashi no kodomo) ditambah partikel を (o)

Sehingga polanya :

Kata kerja bentuk -te + Coudai

= Watashi no kodomo o yashinate + Choudai

= 私の子供を養って ちょうだい

Penggunaan akhiran せ (se), ろ (ro) dan え (e)

Penggunaan akhiran -se dan -e dilakukan pada Kata Kerja 1 sedangkan -ro pada kata kerja 2. Akhiran ini berfungsi sebagai akhiran kalimat perintah yang sifatnya marah atau membentak. Untuk membuat pola kalimatnya kata kerja masu harus diubah ke bentuk se, ro, atau e. Contoh :

Bicaralah!!

Berbicara dalam Bahasa Jepang はなす(Hanasu)

Nah, berbicara termasuk Kata Kerja 1 maka harus diubah ke bentuk e atau se. Maka, ketika diubah menjadi はなせ (Hanase)
