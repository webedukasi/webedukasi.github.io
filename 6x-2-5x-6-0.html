Tips!

- Rumus ABC :

[tex] \boxed{x_1,x_2 = \frac{-b \pm\sqrt{b^2 - 4ac} }{2a}} [/tex]

Pertanyaan :

Dengan rumus ABC, cari himpunan penyelesaian dari persamaan kuadrat berikut :

1) x² + 5x + 6 = 0

Bentuk umum persamaan kuadrat : ax² + bx + c = 0, dimana a ≠ 0. Maka :

a = 1

b = 5

c = 6

[tex] x_1,x_2 =\frac{- 5 \pm \sqrt{5^2 - 4(1)(6)}}{2(1)} \\\\ x_1,x_2 =\frac{- 5 \pm\sqrt{1} }{2} \\\\ x_1,x_2 =\frac{- 5\pm 1}{2} \\\\ x_1 = \frac{- 5 + 1}{2} = -\frac{4}{2} = \boxed{- 2} \\\\ x_2 = \frac{- 5 - 1}{2} = -\frac{6}{2} = \boxed{- 3} [/tex]

Diperoleh x = - 2 atau x = - 3

2) x² - 6x - 7 = 0

a = 1
