Jumlah uang Budi dan Agung = ..?

• Uang Budi

7 / 8 × Rp 72.000 = Rp 63.000

• Jumlah uang Budi dan Agung

Rp 72.000 + Rp 63.000 = Rp 135.000

Jadi, jumlah uang Budi dan Agung adalah Rp 135.000

Buatlah algoritma dan program python berikut: aldi mempunyai kelereng 15 lebih banyak dari budi, sedangkan anto mempunyai kelerang 2x jumlah kelereng aldi dan budi. agung memiliki kelerang 5 buah lebih sedikit dari jumlah kelereng aldi, budi dan anto. berapakah jumlah kelereng budi, anto dan agung apabila jumlah kelereng aldi diketahui


Algoritma :

MulaiMasukkan jumlah kelereng AldiCari kelereng Budi, dengan formula "kelereng Aldi - 15"Cari kelereng Anto, dengan formula "2 x (kelereng Budi + kelereng Aldi)Cari kelereng Agung, dengan formula "Kelereng Aldi + Kelereng Budi + Kelereng Anto - 5"Cari jumlah kelereng Budi, Anto dan AgungKeluarkan melalui bentuk printSelesai

Program Python :

aldi = int(input("Kelereng Aldi : "))

budi = aldi - 15

anto = 2 * (aldi + budi)

agung = (aldi + budi + anto) - 5

jumlah = budi + anto + agung

print("Jumlah : " + str(jumlah))

Python merupakan salah satu Bahasa Pemrograman yang cukup populer. Salah satu alasan yang menyebabkan Python menjadi Bahasa populer adalah karena tingkat kesukarannya yang rendah jika dibandingkan bahasa yang lain. Sehingga, Python mudah dipelajari oleh awam.

Pembuatan algoritma

Disini, kita menemui bahwa kita hanya dapat mengetahui jumlah kelereng Aldi dan kita harus mencari jumlah kelereng Budi, Anto, dan Agung. Maka dari itu, kita harus mencari kelereng Budi, Anto, dan Agung satu per satu

Langkah 1 : Mencari jumlah kelereng Budi

"Aldi mempunyai kelereng 15 lebih banyak dari budi"

Aldi = Budi + 15

Budi = Aldi - 15

Langkah 2 : Mencari jumlah kelereng Anto

"Anto mempunyai kelerang 2x jumlah kelereng aldi dan budi"

Anto = 2 x (Aldi + Budi)

Langkah 3 : Mencari jumlah kelereng Agung

"Agung memiliki kelerang 5 buah lebih sedikit dari jumlah kelereng aldi, budi dan anto"

Agung = Aldi + Budi + Anto - 5

Langkah 4 : Mencari jumlahnya

Jumah = Budi + Anto + Agung

Langkah 5 : Gabungkan

Apabila digabungkan, maka akan membentuk algoritma seperti dibawah ini :

Setelah membuat algoritma, barulah kita dapat membuat kode Python. Kode Python berbentuk seperti ini :

aldi = int(input("Kelereng Aldi : "))
