Jawaban:

Pembahasan

Struktur percabangan dari pemrogramman if bersarang atau nested if adalah sebuah bentuk dari struktur pemrograman if yang dianggap paling rumit, hal in idikarenakan merupakan sebuah bentuk dari perluasan dan juga kombinasi yang berasal dari berabgai maccam bentuk struktur if lainnya.Konsep daripada percabangan ini sendiri adalah sebuah bentuk yang terdapat pada struktr if yang berada didalamnya adalah struktur if lainnya. Arti lainnya ke dalam sebuah pernyataan if akan bersarang adalah jiga kondisi if yang dianggap paling luar atau paling atas yang bernilai benar, sehingga kondisi if yang berada didalamnya akan dapat dilihat di bagian cek.

Berikut adalah bentuk umum dari penggunaan daripada struktur daripada if bersarang atau struktur if yang akan berada didalam struktur if lainnya:

if(kondisi1){

if(kondisi1a){

Statemen_jika_kondisi1_dan_1a_terpenuhi;

}

else if(kondisi1b){

Statemen_jika_kondisi1_dan_1b_terpenuhi;

}

}

}

23. apa yang di maksud dengan tingkatan operasi hitung diexcel

1. Mengerjakan yang didalam kurung terlebih dahulu. Jika tanda kurungnya banyak, maka dikerjakan dari tanda kurung yang paling dalam keluar.
2. Mengerjakan perkalian, pangkat, akar dan pembagian terlebih dahulu.
3. Mengerjakan penjumlahan dan penguranagan
Fungsi Microsoft excel dalam operasi hitung bilangan matematika yang dapat dioperasikan antara lain:
- Operator matematika

- Fungsi max
- Fungsi min
- Fungsi count

- Fungsi Average
