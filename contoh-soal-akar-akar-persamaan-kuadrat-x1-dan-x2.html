21. Akar-akar kuadrat dari persamaan kuadrat 4x²-10x+2=0 adalah x1 dan x2. persamaan kuadrat yang akar akarnya x1+2/x2 dan x2+2/x1 adalah....​


Jawaban:

2x² - 25x + 59 = 0

Penjelasan dengan langkah-langkah:

4x² - 10x + 2 = 0

2x² - 5x + 1 =0

x1 + x2 = -b/a = 5/2

x1. x2 = c/a = 1/2

1/x1 + 1/x2 = x1 + x2/x1.x2 = 5

x1² + x2² = (x1 + x2)² - 2x1.x2 = 21/4

Misalkan : A = x1 + 2/x2 ; B = x2 + 2/x2

A + B = ( x1 + 2/x1) + (x2 + 2/x2)

= (x1 + x2) + 2(1/x1 + 1/x2)

= 5/2 + 10

= 25/2

A.B = (x1 + 2/x1) . (x2 + 2/x2)

= x1.x2 + 2(x1/x2 + x2/x1) + 4/x1.x2

= ½ + 2( x1² + x2²/x1.x2) + 4/x1.x2

= ½ + 2(21/2) + 8

= ½ + 21 + 8

= 59/2

Jadi persamaan kuadrat baru yang akar-akarnya A dan B adalah

x² - (A + B)x + A.B = 0

x² - 25x/2 + 59/2 = 0

2x² - 25x + 59 = 0
