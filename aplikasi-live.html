17. 1. Apa yang dimaksud dengan file system, jelaskan dan berikan contohnya 2. apa yang dimaksud atribut file disistem operasi linuk dan windows jelaskan 3. Apa yang di maksud dengan partisi harddisk primary dan active jealaskan 4. Jelaskan perbedeaan partisi harddisk di instalasi linuk dan windows 5. Tuliskan langkah instalasi aplikasi / pakettambahan di system linux 6. Apa yang dimaksud dengan cloud computing jelaskan 7. Apa yang di maksud dengan system operasi live cd dan live usb 8. Berikan contoh system operasi yang bias dijadikan live usb


1. File System adalah struktur logika yang digunakan untuk mengendalikan akses terhadap data yang ada pada disk atau media penyimpanan lainnya.
Contoh:
a. Windows : FAT (File Allocation Table) dan NTFS (New Technology File System)
b. Linux : ext 1/2/3 (Extended)

2. Atribut file adalah sebuah file atau directori yg digunakan sistem operasi untuk memisahkan tipe-tipe file tertentu. untuk windows biasanya Atributnya berupa Archive, Hidden, ReadOnly dan System. Dan untuk linux berupa R=Read, X=Write dan X=Execute.

3.Partisi Primary adalah Partisi yang digunakan sebagai Boot table yang berfungsi sebagai Master pada sebuah sistem operasi agar dapat di-boot pada awal booting.

Partisi Active merupakan partisi yang telah terpasang sistem operasi berbeda didalamnya yang biasanya untuk dual sistem operasi.

4. Partisi Windows : FAT (16 dan 33) dan NTFS
Partisi Linus : Ext 2/3/4 dan Swap

5. apt cache search
Contoh: apt cache search gimp.

sudo apt-get install
misalnya: sudo apt-get install gimp.

6.Cloud Computing adalah media penyimpanan yang berbasis internet. Seperti Google Drive yang storage nya disediakan oleh Google.

7. Live CD adalah sebuah CD yang bisa digunakan untuk menjalankan sistem operasi linux dari CD ROM tanpa harus menginstalnya ke harddisk terlebih dahulu.

Live USB sama hal nya seperti Live CD. Hanya berbeda di media penyimpanan nya yg menggunakan USB atau Flashdisk.

8. GNU/Linux, FreeBSD, OpenBSD, NetBSD, ReactOS, dan BeOS

Maaf gak semua bisa kejawab.
7. Sistem operasi Live CD adalah sistem operasi yang datanya hanya tersimpan pada sesi pertama. Misalnya kita boot Ubuntu live CD, terus kita make Firefoxnya, maka ketika komputernya di shutdown, data riwayat firefox hilang. Jadi ketika boot selanjutnya, data riwayat firefox kita tidak tersimpan dan tidak bisa terbuka.

Sedangkan jika kita memakai Live USB, data yang ada bisa disimpan. Misalnya hari ini boot ubuntu live usb, terus bikin notepad dan disimpan. Terus shutdown komputer. Maka pada saat booting berikutnya, data notepadnya tetap tersimpan. Maaf agak gak jelas.
