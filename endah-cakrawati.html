Dik :

jadi suhu terendah di Paris adalah -4°C.


6. Jumlah uang Endah dan uang Amir adalah Rp99.000,00. Jika uang Endah dan uang Amir berbanding 4: 5, berapa besar uang Endah?​


Jawab:

Penjelasan dengan langkah-langkah:

Misalkan :

Uang Amir = 4x

Uang Endah = 5x

Uang Amir + uang Endah = 4x + 5x

Uang Amir + uang Endah = 9x

99.000 = 9x

x = 99.000/9

x = 11.000

Uang Endah = 5 (11.000)

Uang Endah = 55.000

Jadi, uang Endah adalah Rp.55.000,00.
