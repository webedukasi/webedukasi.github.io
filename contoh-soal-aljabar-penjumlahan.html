Misalkan :

B = Buku

P = Pensil

Maka :

3B + 5P = Rp. 11.000,00

B + 2P = Rp. 4.000,00

B = ???

P = ??

Maka untuk mencari B dan P kita harus ubah B + 2P = Rp. 4.000,00 menjadi B saja, maka :

B + 2P = Rp. 4.000,00

B + 2P - 2P = Rp. 4.000,00 - 2P

B = Rp. 4.000,00 - 2P

Jika B = Rp. 4.000,00 - 2P, maka 3B + 5P = Rp. 11.000,00, menjadi :

3( Rp. 4.000,00 - 2P) + 5P = Rp. 11.000,00

Rp. 12.000,00 - 6P + 5P = Rp. 11.000,00

Rp. 12.000,00 - P = Rp. 11.000,00

Rp. 12.000,00 - P - Rp.12.000,00 = Rp. 11.000,00 - Rp.12.000,00

-p = -Rp.1.000

-p x -1 = -Rp.1.000 x -1

p = Rp.1.000,00

Jadi harga 1 pensil sama dengan Rp.1.000,00

Kemudian kita cari harga sebuah buku dengan cara memasukan p = Rp.1.000,00 kedalam 3B + 5P = Rp. 11.000,00, maka :

3B + 5(Rp.1.000,00) = Rp. 11.000,00

3B + Rp.5.000,00 = Rp. 11.000,00

3B + Rp.5.000,00 - Rp.5.000,00 = Rp. 11.000,00 - Rp.5.000,00

3B = Rp. 6.000,00

3B x 1/3 = Rp. 6.000,00 x 1/3

B = Rp. 2.000,00

Jadi harga 1 buku sama dengan Rp.2.000,00

Maka harga 1 buku dan 1 pensil adalah Rp.2.000,00 dan Rp.1.000,00
