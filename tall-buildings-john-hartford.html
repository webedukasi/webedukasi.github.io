10. There are two tall buildings in the city. The average height of the two buildings is 617 m. Building A is 334 m taller than Building B. What is the height of Building A?


Let's assume the height of Building B is x meters.

According to the given information, the average height of the two buildings is 617 m. Since we have the height of Building B as x, we can set up the following equation:

(x + x + 334) / 2 = 617

Combining like terms and simplifying:

(2x + 334) / 2 = 617
2x + 334 = 2 * 617
2x + 334 = 1234

Now, let's solve for x:

2x = 1234 - 334
2x = 900
x = 900 / 2
x = 450

Therefore, the height of Building B is 450 meters.

To find the height of Building A, we can add 334 meters to the height of Building B:

Height of Building A = Height of Building B + 334
Height of Building A = 450 + 334
Height of Building A = 784

So, the height of Building A is 784 meters.
