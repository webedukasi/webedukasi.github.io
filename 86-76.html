2. 86+(76+67)=(86+76)+67 tolong dong ini rumus nya gimana


Jawab:

Rumus yang digunakan dalam menyederhanakan ekspresi matematika tersebut adalah sifat asosiatif pada operasi penjumlahan. Sifat asosiatif pada penjumlahan menyatakan bahwa urutan pengelompokan dalam penjumlahan tidak akan mempengaruhi hasil akhir.

Dengan menggunakan rumus tersebut, ekspresi matematika di atas dapat disederhanakan sebagai berikut:

1. Langkah pertama: Lakukan operasi penjumlahan dalam tanda kurung pertama (76 + 67) terlebih dahulu.

  86 + (76 + 67) = 86 + 143

2. Langkah kedua: Lakukan operasi penjumlahan di sebelah kiri tanda sama dengan (86 + 76) terlebih dahulu.

  (86 + 76) + 67 = 162 + 67

Setelah itu, kita akan mendapatkan hasil yang sama dari kedua operasi tersebut:

  86 + (76 + 67) = 86 + 143 = 229

  (86 + 76) + 67 = 162 + 67 = 229

Sehingga, ekspresi matematika tersebut dapat disederhanakan menjadi:

  86 + (76 + 67) = (86 + 76) + 67 = 229

26. Tulislah alasan yang digunakan untuk mendapatkan hasilnya a. 86 + (76 +...) = (86+...) + 67 b. + (96 + 27) = (39 + 96) + ... **​


a. 86 + (76 + 67) = (86 + 76) + 67 b. 39 + (96 + 27) = (39 + 96) + 27

Alasan yang digunakan untuk mendapatkan hasil tersebut yaitu menggunakan sifat Asosiatif (pengelompokkan).

Penjelasan dengan langkah-langkah

Diketahui :

a. 86 + (76 + ...) = (86 + ...) + 67

b. ... + (96 + 27) = (39 + 96) + ...

Ditanya :

Menuliskan alasan yang digunakan untuk mendapatkan hasil.

Jawab :

Sifat Asosiatif (pengelompokkan)

(a + b) + c = a + (b + c), dimana a, b, dan c merupakan bilangan bulat.

Untuk mengisi titik-titik kita lihat bilangan yang belum ada.

a. 86 + (76 + 67) = (86 + 76) + 67

b. 39 + (96 + 27) = (39 + 96) + 27
